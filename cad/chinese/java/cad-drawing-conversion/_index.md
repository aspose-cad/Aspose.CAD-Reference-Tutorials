## 常见问题

**问：如何在不丢失图层信息的情况下将 DWG 文件导出为 PDF？**  
答：使用 `PdfOptions` 并设置 `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` 以保留矢量数据，然后调用 `image.save("output.pdf", options);`。

**问：我能一次性批量将多个 DWG 文件处理为 PDF 吗？**  
答：可以——遍历文件集合，使用 `Image.load` 加载每个文件，并使用相同的 PDF 选项进行保存。

**问：是否有办法直接将 DWT 模板转换为 DXF？**  
答：当然。使用 `Image.load` 加载 DWT，然后使用 `image.save("output.dxf");` 将其保存为 DXF。

**问：Aspose.CAD 支持哪些 Java 版本？**  
答：该库兼容 Java 8 及更高版本，包括 Java 11、17 以及后续的 LTS 版本。

**问：导出 PDF 是否需要单独的许可证？**  
答：不需要，Aspose.CAD 许可证涵盖所有输出格式，包括 PDF、光栅图像和 DXF。
