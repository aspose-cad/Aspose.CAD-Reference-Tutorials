## 常见问题

**Q: 如何在不丢失图层信息的情况下将 DWG 文件导出为 PDF？**  
A: 使用 `PdfOptions` 并设置 `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` 以保留矢量数据，然后调用 `image.save("output.pdf", options);`。

**Q: 能否一次性批量处理多个 DWG 文件导出为 PDF？**  
A: 可以——遍历文件集合，使用 `Image.load` 加载每个文件，并使用相同的 PDF 选项进行保存。

**Q: 是否可以直接将 DWT 模板转换为 DXF？**  
A: 完全可以。使用 `Image.load` 加载 DWT，然后使用 `image.save("output.dxf");` 将其保存为 DXF。

**Q: Aspose.CAD 支持哪些 Java 版本？**  
A: 该库兼容 Java 8 及更高版本，包括 Java 11、17 以及后续的 LTS 版本。

**Q: 导出 PDF 是否需要单独的许可证？**  
A: 不需要，Aspose.CAD 许可证覆盖所有输出格式，包括 PDF、光栅图像和 DXF。

**Q: 在转换大型 DWG 文件时，如何提升渲染速度？**  
A: 通过在单独线程中处理每个布局并复用同一个 `PdfOptions` 实例来启用多线程，以减少开销。

**Q: 能否为导出的 PDF 设置自定义页面尺寸？**  
A: 可以，在保存前使用 `PdfOptions.setPageSize(PageSize.A4)`（或其他尺寸）配置 `PdfOptions`。
