**Q: 我可以向已经受密码保护的 DWG 文件添加水印吗？**

**Q: 添加水印会显著增加文件大小吗？**  
**A:** 增长很小，因为水印以轻量级覆盖层的形式存储。

**Q: 能否批量处理多个 CAD 文件以添加相同的水印？**  
**A:** 当然。遍历文件集合，对每个图纸调用相同的水印例程。

**Q: 我可以控制水印的透明度和旋转角度吗？**  
**A:** API 提供属性来设置透明度、旋转角度和位置，让您拥有完整的视觉控制。

**Q: 这些教程支持哪些 Java 版本？**  
**A:** 完全支持 Java 8、11 以及更新的 LTS 版本。
