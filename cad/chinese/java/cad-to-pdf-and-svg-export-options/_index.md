## 常见问题

**Q:** *我能在不耗尽内存的情况下将大型 DWG 文件转换为 PDF/A 吗？*  
A: 可以。使用带有流式选项的 `CadImage` 类，并及时释放图像以保持低内存占用。

**Q:** *是否可以仅导出 CAD 文件中的特定布局为 SVG？*  
A: 当然。按名称加载所需布局，并使用 SVG 选项调用 `Save` 方法。

**Q:** *每种输出格式都需要单独的许可证吗？*  
A: 不需要。单一的 Aspose.CAD for Java 许可证覆盖所有支持的导出格式。

**Q:** *我可以一次性批量处理多个 CAD 文件吗？*  
A: 可以。遍历文件集合，使用 `CadImage.load` 加载每个文件并保存为目标格式。

**Q:** *如果我的 CAD 文件包含光栅图像，导出为 PDF 时会被保留吗？*  
A: 导出为 PDF 时，光栅图像会自动嵌入，保持视觉保真度。
