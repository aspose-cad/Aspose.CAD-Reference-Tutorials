## 常见问题

**Q: 如何在 C# 中打开 DWFX 文件？**  
A: 使用 `Aspose.CAD.Image.Load("file.dwfx")` 加载文件，然后根据需要访问其页面或实体。

**Q: 能否在不先转换的情况下直接用 C# 读取 DWG 文件？**  
A: 可以——Aspose.CAD 能直接加载 DWG 文件，允许您查询图层、实体和元数据。

**Q: 向 DWG 图纸添加文本的最简方法是什么？**  
A: 创建 `CadText` 实体，设置其 `Value` 和位置，然后将其添加到图纸的模型空间集合中。

**Q: 处理大型 DWFX 或 DWG 文件时是否有尺寸限制？**  
A: 库能够高效处理大文件，但您应确保有足够的内存，并考虑对大型光栅图像进行流式处理。

**Q: 每种 CAD 格式都需要单独的许可证吗？**  
A: 不需要——单一的 Aspose.CAD 许可证覆盖所有受支持的格式，包括 DWG、DWFX、DWF、DXF 等。
