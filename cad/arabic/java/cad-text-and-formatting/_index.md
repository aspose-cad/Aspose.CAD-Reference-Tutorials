## ما هو البحث عن النص في dwg؟
يعني البحث عن النص في ملفات DWG تحديد قيم السلاسل داخل كيانات الرسم مثل MText و Text وتعريفات السمات. هذه القدرة أساسية للمهام مثل إنشاء تقارير، أتمتة التعديلات، أو بناء مستودعات CAD قابلة للبحث.

## لماذا تستخدم Aspose.CAD for Java لنص وتنسيق DWG؟
- **Speed:** تعالج الواجهة البرمجية الرسومات الكبيرة بسرعة دون الحاجة إلى تثبيت AutoCAD.  
- **Accuracy:** تتعامل مع جميع كيانات نص DWG، مع الحفاظ على التنسيق والنمط.  
- **Flexibility:** تعمل عبر الأنظمة (Windows، Linux، macOS) وتندمج بسهولة مع تطبيقات Java الحالية.  
- **Extensibility:** تدعم الخطوط المخفية، كيانات MLeader، وتعديل السمات مباشرةً.
