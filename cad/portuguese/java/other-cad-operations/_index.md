## Perguntas Frequentes

**Q: Posso adicionar uma marca d'água a um arquivo DWG que já está protegido por senha?**  
A: Sim. Carregue o DWG com a senha apropriada, aplique a marca d'água usando a API e, em seguida, salve o arquivo.

**Q: Adicionar uma marca d'água afeta significativamente o tamanho do arquivo?**  
A: O aumento é mínimo porque a marca d'água é armazenada como uma sobreposição leve.

**Q: É possível processar em lote vários arquivos CAD para adicionar a mesma marca d'água?**  
A: Absolutamente. Percorra sua coleção de arquivos e invoque a mesma rotina de marca d'água para cada desenho.

**Q: Posso controlar a opacidade e rotação da marca d'água?**  
A: A API fornece propriedades para definir opacidade, ângulo de rotação e posicionamento, oferecendo controle visual total.

**Q: Quais versões do Java são suportadas para esses tutoriais?**  
A: Java 8, 11 e versões LTS mais recentes são totalmente suportadas.
