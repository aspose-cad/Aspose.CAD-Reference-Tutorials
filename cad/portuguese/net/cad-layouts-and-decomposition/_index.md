## Como Converter CAD para PDF Usando Aspose.CAD para .NET
Abaixo está um processo conciso, passo a passo, que você pode seguir em qualquer projeto .NET.

### Passo 1: Instalar o pacote NuGet Aspose.CAD
Abra o gerenciador NuGet do seu projeto e adicione `Aspose.CAD`. Isso inclui todas as assemblies necessárias.

### Passo 2: Carregar seu arquivo CAD
Use a classe `Image` para abrir um DWG, DXF ou outro formato suportado.

### Passo 3: Configurar opções de exportação PDF (opcional)
Você pode especificar o tamanho da página, DPI de rasterização e se deve incluir camadas ocultas.

### Passo 4: Salvar o desenho como PDF
Chame o método `Save` com o objeto `PdfOptions` para gerar um PDF de alta qualidade.
