## Vanliga frågor

**Q: Kan jag lägga till en vattenstämpel i en DWG‑fil som redan är lösenordsskyddad?**  
A: Ja. Läs in DWG‑filen med rätt lösenord, applicera vattenstämpeln via API‑t och spara sedan filen.

**Q: Påverkar tillägg av en vattenstämpel filstorleken avsevärt?**  
A: Ökningen är minimal eftersom vattenstämpeln lagras som ett lättviktigt överlägg.

**Q: Är det möjligt att batch‑processa flera CAD‑filer för att lägga till samma vattenstämpel?**  
A: Absolut. Loop igenom din filsamling och anropa samma vattenstämpel‑rutin för varje ritning.

**Q: Kan jag styra opaciteten och rotationen på vattenstämpeln?**  
A: API‑t erbjuder egenskaper för att sätta opacitet, rotationsvinkel och positionering, vilket ger dig full visuell kontroll.

**Q: Vilka versioner av Java stöds för dessa handledningar?**  
A: Java 8, 11 och nyare LTS‑utgåvor stöds fullt ut.
