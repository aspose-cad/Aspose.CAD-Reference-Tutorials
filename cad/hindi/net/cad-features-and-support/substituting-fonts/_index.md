# Aspose.CAD for .NET में फ़ॉन्ट कैसे बदलें

## परिचय

.NET का उपयोग करके CAD विकास के क्षेत्र में, **फ़ॉन्ट कैसे बदलें** सीखना एक महत्वपूर्ण कौशल है जो आपके ड्रॉइंग की दृश्य गुणवत्ता को नाटकीय रूप से सुधार सकता है। Aspose.CAD for .NET एक सरल API प्रदान करता है जो आपको किसी भी शैली के लिए **प्राथमिक फ़ॉन्ट नाम** सेट करने की अनुमति देता है, जिससे वैश्विक या चयनात्मक फ़ॉन्ट प्रतिस्थापन आसान हो जाता है। इस ट्यूटोरियल में हम पूरी प्रक्रिया को देखेंगे—ड्रॉइंग लोड करने से लेकर फ़ॉन्ट को वैश्विक रूप से या विशिष्ट शैली नाम द्वारा बदलने तक।

## त्वरित उत्तर
- **CAD हेरफेर के लिए मुख्य क्लास कौन सी है?** `Aspose.CAD.Image` (या इसका व्युत्पन्न `CadImage`)।
- **फ़ॉन्ट बदलने वाली प्रॉपर्टी कौन सी है?** `PrimaryFontName` on `CadStyleTableObject`।
- **क्या मैं सभी शैलियों के फ़ॉन्ट एक साथ बदल सकता हूँ?** हाँ, `cadImage.Styles` पर इटररेट करके प्रॉपर्टी सेट करें।
- **क्या उत्पादन के लिए लाइसेंस चाहिए?** गैर‑ट्रायल उपयोग के लिए एक वैध Aspose.CAD लाइसेंस आवश्यक है।
- **समर्थित .NET संस्करण?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7।

## CAD में फ़ॉन्ट प्रतिस्थापन क्या है?
फ़ॉन्ट प्रतिस्थापन का अर्थ है CAD ड्रॉइंग में उपयोग किए गए मूल टाइपफ़ेस को उस सिस्टम पर उपलब्ध किसी अन्य फ़ॉन्ट से बदलना। यह विशेष रूप से तब उपयोगी होता है जब मूल फ़ॉन्ट अनुपलब्ध हो, जब आपको एकसमान कॉर्पोरेट शैली चाहिए, या जब सीमित फ़ॉन्ट सेट वाले प्रिंटिंग डिवाइसों के लिए ड्रॉइंग तैयार कर रहे हों।

## Aspose.CAD के साथ फ़ॉन्ट क्यों बदलें?
- **कोई बाहरी निर्भरताएँ नहीं** – लाइब्रेरी फ़ॉन्ट मैपिंग को आंतरिक रूप से संभालती है।
- **कई फ़ॉर्मैट्स के साथ काम करता है** – DXF, DWG, DGN, आदि।
- **प्रोग्रामेटिक नियंत्रण** – बैच रूपांतरण या CI पाइपलाइन के लिए प्रक्रिया को स्वचालित करें।
- **ड्रॉइंग ज्योमेट्री को संरक्षित रखता है** – केवल टेक्स्ट का दृश्य प्रतिनिधित्व बदलता है।

## आवश्यकताएँ
- .NET प्रोग्रामिंग का बुनियादी ज्ञान।
- Aspose.CAD for .NET स्थापित है। यदि आपने अभी तक स्थापित नहीं किया है, तो इसे [here](https://releases.aspose.com/cad/net/) से डाउनलोड करें।
- प्रयोग के लिए एक CAD ड्रॉइंग फ़ाइल (DXF, DWG, आदि)।

## नेमस्पेस आयात करें
शुरू करने से पहले, आवश्यक नेमस्पेस आयात करें ताकि आप अपने .NET एप्लिकेशन में Aspose.CAD कार्यक्षमताओं तक पहुंच सकें।

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using Aspose.CAD;
using Aspose.CAD.FileFormats.Cad.CadTables;
```

## चरण 1: CAD ड्रॉइंग लोड करें

CAD ड्रॉइंग को `CadImage` के एक इंस्टेंस में लोड करके शुरू करें। सुनिश्चित करें कि आप अपने दस्तावेज़ निर्देशिका का सही पथ प्रदान करें।

```csharp
string MyDir = "Your Document Directory";
string sourceFilePath = MyDir + "conic_pyramid.dxf";

using (CadImage cadImage = (CadImage)Aspose.CAD.Image.Load(sourceFilePath))
{
    // Your code for further actions goes here
}
```

## चरण 2: शैलियों पर इटररेट करें

अगले चरण में, CAD ड्रॉइंग में शैलियों पर `foreach` लूप का उपयोग करके इटररेट करें। इससे आप व्यक्तिगत फ़ॉन्ट शैलियों तक पहुंच और उन्हें संशोधित कर सकते हैं।

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    // Your code for style manipulation goes here
}
```

## फ़ॉन्ट प्रतिस्थापन के लिए प्राथमिक फ़ॉन्ट नाम कैसे सेट करें
प्रत्येक `CadStyleTableObject` पर `PrimaryFontName` प्रॉपर्टी नियंत्रित करती है कि ड्रॉइंग रेंडर होने पर कौन सा फ़ॉन्ट उपयोग किया जाता है। इस प्रॉपर्टी को सेट करके आप मूल फ़ॉन्ट को प्रभावी रूप से बदल देते हैं।

### चरण 3: फ़ॉन्ट्स को वैश्विक रूप से बदलें
**सभी** शैलियों के फ़ॉन्ट को बदलने के लिए, प्रत्येक शैली के लिए `PrimaryFontName` प्रॉपर्टी को इच्छित फ़ॉन्ट नाम पर सेट करें, उदाहरण के लिए, `"Arial"`।

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    style.PrimaryFontName = "Arial";
}
```

### चरण 4: शैली नाम द्वारा फ़ॉन्ट बदलें
यदि आपको केवल किसी विशिष्ट शैली (जैसे, `"Roman"` नाम की शैली) के फ़ॉन्ट को बदलना है, तो लूप के भीतर एक शर्तीय जांच जोड़ें।

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    if (style.StyleName == "Roman")
    {
        style.PrimaryFontName = "Arial";
    }
}
```

## सामान्य समस्याएँ और ट्रबलशूटिंग

| समस्या | कारण | समाधान |
|-------|-------|-----|
| कोड चलाने के बाद फ़ॉन्ट नहीं बदलता | ड्रॉइंग कैश्ड है या रीड‑ओनली मोड में खुला है | परिवर्तन के बाद इमेज को सेव करें (`cadImage.Save(...)`) या सत्यापन के लिए फ़ाइल को पुनः लोड करें। |
| वांछित फ़ॉन्ट सिस्टम पर नहीं मिला | फ़ॉन्ट कोड चलाने वाली मशीन पर स्थापित नहीं है | आवश्यक TrueType/OpenType फ़ॉन्ट स्थापित करें या इसे एप्लिकेशन संसाधनों में एम्बेड करें। |
| टेक्स्ट गड़बड़ दिखता है | गलत एन्कोडिंग या यूनिकोड समर्थन की कमी | ऐसा फ़ॉन्ट उपयोग करें जो आवश्यक कैरेक्टर सेट को सपोर्ट करता हो (जैसे, Arial Unicode MS)। |

## अक्सर पूछे जाने वाले प्रश्न

**प्र:** क्या मैं Aspose.CAD for .NET में फ़ॉन्ट परिवर्तन को वापस कर सकता हूँ?  
**उ:** हाँ, आप मूल CAD ड्रॉइंग को पुनः लोड करके या संशोधन से पहले बैकअप कॉपी रखकर वापस कर सकते हैं।

**प्र:** क्या मैं अन्य फ़ॉन्ट प्रॉपर्टीज़ भी संशोधित कर सकता हूँ?  
**उ:** बिल्कुल। `PrimaryFontName` के अलावा, आप `SecondaryFontName`, `FontFamily`, और अन्य शैली‑संबंधित गुणों के साथ काम कर सकते हैं उन्नत अनुकूलन के लिए।

**प्र:** क्या Aspose.CAD विभिन्न CAD फ़ॉर्मैट्स के साथ संगत है?  
**उ:** हाँ, Aspose.CAD DXF, DWG, DGN, DWF आदि सहित कई फ़ॉर्मैट्स को सपोर्ट करता है, जिससे आपको प्रोजेक्ट्स में लचीलापन मिलता है।

**प्र:** क्या मैं बैच प्रोसेसिंग में फ़ॉन्ट प्रतिस्थापन को स्वचालित कर सकता हूँ?  
**उ:** बिल्कुल। लोडिंग और प्रतिस्थापन लॉजिक को एक लूप में रखें जो CAD फ़ाइलों के फ़ोल्डर पर इटररेट करे, या इसे CI/CD पाइपलाइन में एकीकृत करें।

**प्र:** Aspose.CAD for .NET के लिए अतिरिक्त समर्थन कहाँ मिल सकता है?  
**उ:** अतिरिक्त समर्थन और समुदाय चर्चा के लिए, [Aspose.CAD फोरम](https://forum.aspose.com/c/cad/19) पर जाएँ।
