# C# के साथ DWG लेयर्स निर्यात करें – Aspose.CAD ट्यूटोरियल

## परिचय

इस व्यापक गाइड में आप C# और Aspose.CAD लाइब्रेरी का उपयोग करके **DWG लेयर्स निर्यात करने** का तरीका सीखेंगे। चाहे आपको **DWG इमेज** फ़ॉर्मेट बदलने की आवश्यकता हो, **DWG लेयर विज़िबिलिटी** नियंत्रित करनी हो, या बस **DWG JPEG** फ़ाइलें सहेजनी हों, नीचे दिए गए चरण आपको यह कुशलतापूर्वक करने का सही तरीका दिखाएंगे। ट्यूटोरियल के अंत तक आपके पास एक तैयार‑चलाने योग्य स्निपेट होगा जो एक विशिष्ट लेयर को अलग करता है और उसे उच्च‑गुणवत्ता वाले JPEG के रूप में रेंडर करता है।

## त्वरित उत्तर

- **“export dwg layers” का क्या मतलब है?** इसका अर्थ है DWG फ़ाइल की चयनित लेयर्स को JPEG या PNG जैसे इमेज फ़ॉर्मेट में रास्टराइज़ करना।  
- **इस कार्य के लिए सबसे अच्छी लाइब्रेरी कौन सी है?** .NET के लिए Aspose.CAD बिना AutoCAD की आवश्यकता के पूर्ण‑फ़ीचर समर्थन प्रदान करती है।  
- **क्या मैं एक साथ कई लेयर्स निर्यात कर सकता हूँ?** हाँ – रास्टराइज़ेशन विकल्पों में लेयर नामों की एक एरे पास करें।  
- **उत्पादन उपयोग के लिए क्या मुझे लाइसेंस चाहिए?** एक व्यावसायिक लाइसेंस आवश्यक है; मूल्यांकन के लिए एक मुफ्त ट्रायल उपलब्ध है।  
- **कौन से आउटपुट फ़ॉर्मेट समर्थित हैं?** JPEG, PNG, BMP, TIFF और ImageOptions क्लासेज़ के माध्यम से कई अन्य फ़ॉर्मेट।

## DWG लेयर्स निर्यात क्या है?

DWG लेयर्स निर्यात करने का मतलब है DWG ड्राइंग के भीतर एक या अधिक लेयर्स से संबंधित वेक्टर डेटा को JPEG या PNG जैसे बिटमैप इमेज में रास्टराइज़ करना। यह तब उपयोगी होता है जब आप ड्राइंग के किसी विशिष्ट भाग का दृश्य साझा करना चाहते हैं बिना पूरे CAD फ़ाइल को उजागर किए।

## DWG लेयर विज़िबिलिटी को नियंत्रित क्यों करें?

- प्रस्तुति या दस्तावेज़ीकरण के लिए साफ़ दृश्य संपत्तियां बनाएं।  
- केवल आवश्यक ज्योमेट्री निर्यात करके फ़ाइल आकार घटाएं।  
- गोपनीय लेयर्स को छिपाकर स्वामित्व डिज़ाइन विवरणों की सुरक्षा करें।

## पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि आपके पास है:

- C# प्रोग्रामिंग भाषा का मूल ज्ञान।  
- आपके मशीन पर Visual Studio स्थापित हो।  
- Aspose.CAD for .NET लाइब्रेरी, जिसे आप [Aspose.CAD वेबसाइट](https://releases.aspose.com/cad/net/) से डाउनलोड कर सकते हैं।

## नेमस्पेस आयात करें

शुरू करने के लिए, उन नेमस्पेस को आयात करें जो आपको रास्टराइज़ेशन और इमेज‑एक्सपोर्ट सुविधाओं तक पहुंच प्रदान करते हैं।

```csharp
using Aspose.CAD.ImageOptions;
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text;
```

## चरण 1: DWG फ़ाइल लोड करें

स्रोत DWG (या DWF) फ़ाइल को एक `Image` ऑब्जेक्ट में लोड करें। यह ऑब्जेक्ट मेमोरी में पूरे ड्राइंग का प्रतिनिधित्व करता है।

```csharp
string MyDir = "Your Document Directory";
string sourceFilePath = MyDir + "for_layers_test.dwf";

using (Aspose.CAD.Image image = Aspose.CAD.Image.Load(sourceFilePath))
{
    // Your code for subsequent steps goes here
}
```

*यह क्यों महत्वपूर्ण है:* फ़ाइल को एक बार लोड करने से आप उसी `image` इंस्टेंस को कई लेयर‑विशिष्ट निर्यातों के लिए पुनः उपयोग कर सकते हैं, जिससे प्रदर्शन बेहतर होता है।

## चरण 2: रास्टराइज़ेशन विकल्प कॉन्फ़िगर करें

`CadRasterizationOptions` का एक इंस्टेंस बनाएं ताकि Aspose.CAD को बताया जा सके कि ड्राइंग को कैसे रेंडर करना है। यहाँ हम उच्च रेज़ॉल्यूशन (1600 × 1600) सेट करते हैं ताकि निर्यात किया गया JPEG तेज़ हो।

```csharp
Aspose.CAD.ImageOptions.CadRasterizationOptions rasterizationOptions = new Aspose.CAD.ImageOptions.CadRasterizationOptions();
rasterizationOptions.PageWidth = 1600;
rasterizationOptions.PageHeight = 1600;
```

यदि आवश्यक हो तो आप बैकग्राउंड रंग, लाइन वेट स्केलिंग, या एंटी‑एलियासिंग सेटिंग्स को भी समायोजित कर सकते हैं।

## चरण 3: लेयर्स निर्दिष्ट करें (DWG लेयर विज़िबिलिटी)

वे लेयर्स जोड़ें जिन्हें आप **export dwg layers** करना चाहते हैं। इस उदाहरण में हम केवल “LayerA” निर्यात करते हैं। कई लेयर्स निर्यात करने के लिए, उन्हें एरे में सूचीबद्ध करें।

```csharp
rasterizationOptions.Layers = new string[] { "LayerA" };
```

*प्रो टिप:* ड्राइंग में जैसा दिखता है वैसा ही लेयर नाम उपयोग करें; लेयर नाम केस‑सेंसिटिव होते हैं।

## चरण 4: इमेज एक्सपोर्ट विकल्प कॉन्फ़िगर करें

वह इमेज फ़ॉर्मेट चुनें जिसे आप बनाना चाहते हैं। हम `JpegOptions` का उपयोग करेंगे क्योंकि JPEG गुणवत्ता और फ़ाइल आकार के बीच अच्छा संतुलन प्रदान करता है, जो वेब प्रीव्यू के लिए **save dwg jpeg** फ़ाइलें बनाने के लिए आदर्श है।

```csharp
JpegOptions jpegOptions = new JpegOptions();
jpegOptions.VectorRasterizationOptions = rasterizationOptions;
```

यदि आपको **convert dwg image** को PNG या TIFF में बदलना है, तो `JpegOptions` को संबंधित विकल्प क्लास से बदल दें।

## चरण 5: निर्यात की गई इमेज सहेजें

आउटपुट फ़ाइल पाथ निर्धारित करें और `Save` को कॉल करें। रास्टराइज़ेशन इंजन आपके द्वारा प्रदान की गई लेयर सूची का सम्मान करता है, इसलिए अंतिम JPEG में केवल “LayerA” दिखेगा।

```csharp
MyDir = MyDir + "for_layers_test.jpg";
image.Save(MyDir, jpegOptions);
```

इस लाइन के चलने के बाद, आपको `for_layers_test.jpg` आपके दस्तावेज़ डायरेक्टरी में मिलेगा, जिसमें केवल चयनित लेयर होगी।
