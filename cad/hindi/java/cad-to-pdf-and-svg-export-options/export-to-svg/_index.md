# Aspose.CAD for Java का उपयोग करके CAD को SVG में निर्यात करें

## परिचय

Aspose.CAD for Java की दुनिया में आपका स्वागत है, एक शक्तिशाली लाइब्रेरी जो डेवलपर्स को आसानी से CAD ड्रॉइंग्स को नियंत्रित करने में सक्षम बनाती है। चाहे आप एक अनुभवी डेवलपर हों या CAD क्षेत्र में नए हों, यह व्यापक गाइड आपको **export CAD to SVG** चरण दर चरण समझाएगा, यह दिखाते हुए कि DWG को SVG में कैसे बदलें, SVG रंग मोड कैसे सेट करें, और API को अपने Java प्रोजेक्ट में कैसे एकीकृत करें।

## त्वरित उत्तर
- **export CAD to SVG** का क्या अर्थ है? यह एक CAD ड्रॉइंग (जैसे DWG) को Scalable Vector Graphics फ़ाइल में बदलता है जिसे ब्राउज़र में प्रदर्शित किया जा सकता है।  
- **कौन सी लाइब्रेरी इस रूपांतरण को संभालती है?** Aspose.CAD for Java इस कार्य के लिए एक सरल API प्रदान करती है।  
- **क्या विकास के लिए लाइसेंस की आवश्यकता है?** परीक्षण के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **क्या मैं SVG रंग आउटपुट को नियंत्रित कर सकता हूँ?** हाँ, आप SVG रंग मोड सेट कर सकते हैं (जैसे ग्रेस्केल)।  
- **क्या कोई अतिरिक्त सॉफ़्टवेयर आवश्यक है?** केवल एक Java रनटाइम और Aspose.CAD JAR फ़ाइल की आवश्यकता है।

## पूर्वापेक्षाएँ

ट्यूटोरियल शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- Java विकास पर्यावरण: सुनिश्चित करें कि आपके सिस्टम पर Java स्थापित है।  
- Aspose.CAD लाइब्रेरी: Aspose.CAD for Java लाइब्रेरी को [download link](https://releases.aspose.com/cad/java/) से डाउनलोड और इंस्टॉल करें।  
- डॉक्यूमेंट डायरेक्टरी: अपने CAD ड्रॉइंग्स के लिए एक फ़ोल्डर बनाएं, और उसका पथ नोट करें।

## नेमस्पेस आयात करें

इस चरण में, हम आवश्यक नेमस्पेस आयात करेंगे ताकि हमारी Aspose.CAD यात्रा शुरू हो सके। इन चरणों का पालन करें:

### चरण 1: अपना Java प्रोजेक्ट खोलें
अपनी पसंद के IDE में अपना Java प्रोजेक्ट खोलें।

### चरण 2: Aspose.CAD लाइब्रेरी जोड़ें
Aspose.CAD लाइब्रेरी को अपने प्रोजेक्ट में जोड़ें। आप यह JAR फ़ाइल को प्रोजेक्ट की निर्भरताओं में शामिल करके कर सकते हैं।

### चरण 3: नेमस्पेस आयात करें
अपने Java क्लास में, आवश्यक नेमस्पेस आयात करें:

```java
import com.aspose.cad.Image;
import com.aspose.cad.imageoptions.SvgOptions;
import com.aspose.cad.imageoptions.SvgColorMode;
```

## CAD को SVG में निर्यात करें

अब जब हमने तैयारी कर ली है, चलिए Aspose.CAD for Java का उपयोग करके **export CAD to SVG** की चरण‑दर‑चरण प्रक्रिया में प्रवेश करते हैं।

### चरण 1: रिसोर्स डायरेक्टरी निर्दिष्ट करें
अपने रिसोर्स डायरेक्टरी का पथ निर्धारित करें, जहाँ आपके CAD ड्रॉइंग्स स्थित हैं:

```java
String dataDir = "Your Document Directory" + "DWGDrawings/";
```

### चरण 2: CAD ड्रॉइंग लोड करें
Aspose.CAD लाइब्रेरी का उपयोग करके CAD ड्रॉइंग लोड करें:

```java
Image image = Image.load(dataDir + "meshes.dwg");
```

### चरण 3: SVG निर्यात विकल्प कॉन्फ़िगर करें
आउटपुट को अनुकूलित करने के लिए SVG निर्यात विकल्प कॉन्फ़िगर करें। यहाँ हम **SVG रंग मोड** को ग्रेस्केल सेट करते हैं और एक्सपोर्टर को टेक्स्ट को शैप्स में बदलने के लिए कहते हैं:

```java
SvgOptions options = new SvgOptions();
options.setColorType(SvgColorMode.Grayscale);
options.setTextAsShapes(true);
```

### चरण 4: SVG के रूप में सहेजें
CAD ड्रॉइंग को SVG फ़ाइल के रूप में सहेजें:

```java
image.save(dataDir + "meshes.svg");
```

> **प्रो टिप:** यदि आपको रंगों को बनाए रखते हुए **DWG को SVG में बदलने** की आवश्यकता है, तो `SvgColorMode.Grayscale` को `SvgColorMode.FullColor` में बदलें।

बधाई हो! आपने Aspose.CAD for Java का उपयोग करके सफलतापूर्वक एक CAD ड्रॉइंग को SVG में निर्यात कर लिया है।

## CAD को SVG में निर्यात करने के लिए Aspose.CAD का उपयोग क्यों करें?
- **उच्च सटीकता:** वेक्टर डेटा बरकरार रहता है, जिससे SVG मूल CAD ड्रॉइंग जैसा ही दिखता है।  
- **कोई बाहरी निर्भरताएँ नहीं:** रूपांतरण पूरी तरह से Java में चलता है, जिससे अतिरिक्त टूल्स की आवश्यकता नहीं रहती।  
- **अनुकूलन योग्य आउटपुट:** `setColorType` जैसे विकल्प आपको यह नियंत्रित करने देते हैं कि SVG ग्रेस्केल हो या पूर्ण‑रंग।

## सामान्य समस्याएँ और समाधान
- **फ़ाइल नहीं मिली:** सुनिश्चित करें कि `dataDir` सही फ़ोल्डर की ओर इशारा कर रहा है और DWG फ़ाइल का नाम मेल खाता है।  
- **खाली SVG आउटपुट:** यदि ड्रॉइंग में टेक्स्ट है जिसे शैप्स के रूप में दिखना चाहिए, तो सुनिश्चित करें कि आपने `options.setTextAsShapes(true)` सेट किया है।  
- **असमर्थित CAD फ़ॉर्मेट:** Aspose.CAD DWG, DXF, DWF और कई अन्य फ़ॉर्मेट्स को समर्थन देता है; सटीक सूची के लिए दस्तावेज़ देखें।
