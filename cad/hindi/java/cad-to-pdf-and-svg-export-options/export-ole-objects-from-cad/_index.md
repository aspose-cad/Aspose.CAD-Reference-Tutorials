# Aspose.CAD for Java के साथ CAD को PNG के रूप में सहेजें और OLE ऑब्जेक्ट्स निर्यात करें

## परिचय

कंप्यूटर‑एडेड डिज़ाइन (CAD) की तेज़‑तर्रार दुनिया में, **CAD को PNG के रूप में सहेजना** और एम्बेडेड OLE (ऑब्जेक्ट लिंकिंग और एम्बेडिंग) ऑब्जेक्ट्स को निकालना आपके कार्यप्रवाह को काफी सरल बना सकता है। चाहे आप बैच‑कन्वर्ज़न टूल बना रहे हों, वेब पोर्टल के लिए थंबनेल जेनरेट कर रहे हों, या सिर्फ OLE कंटेंट को आर्काइव करना चाहते हों, Aspose.CAD for Java आपको एक साफ़, प्रोग्रामेटिक तरीका प्रदान करता है। इस ट्यूटोरियल में हम पूरी प्रक्रिया को कवर करेंगे, प्रोजेक्ट सेटअप से लेकर प्रत्येक OLE ऑब्जेक्ट को PNG इमेज के रूप में एक्सपोर्ट करने तक।

## त्वरित उत्तर
- **क्या मैं OLE ऑब्जेक्ट्स को सीधे PNG में एक्सपोर्ट कर सकता हूँ?** हाँ – Aspose.CAD CAD फ़ाइल को लोड करता है और आपको प्रत्येक लेआउट को PNG में रास्टराइज़ करने देता है।  
- **कौन सा Java संस्करण आवश्यक है?** Java 8 या उसके बाद का संस्करण।  
- **क्या इस फीचर के लिए लाइसेंस चाहिए?** मूल्यांकन के लिए फ्री ट्रायल काम करता है; प्रोडक्शन के लिए लाइसेंस आवश्यक है।  
- **क्या वेक्टर डेटा संरक्षित रहेगा?** PNG रास्टराइज़ेशन विज़ुअल फ़िडेलिटी रखता है, लेकिन आउटपुट रास्टर है, वेक्टर नहीं।  
- **क्या बैच प्रोसेसिंग समर्थित है?** बिल्कुल – नीचे दिए गए कोड सैंपल में दिखाए अनुसार फ़ाइलों की एरे पर इटरेट करें।

## पूर्वापेक्षाएँ

शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हों:

- **Java Development Kit (JDK)** – Java 8+ स्थापित और कॉन्फ़िगर किया हुआ।  
- **Aspose.CAD for Java** – नवीनतम लाइब्रेरी [download link](https://releases.aspose.com/cad/java/) से डाउनलोड करें।  
- **CAD स्रोत फ़ाइलें** – कोई भी DWG/DXF/DGN फ़ाइलें जिनमें आप OLE ऑब्जेक्ट्स एक्सपोर्ट करना चाहते हैं।

## नेमस्पेस आयात करें

CAD फ़ाइलों के साथ काम करने के लिए आपको कोर Aspose.CAD क्लासेज़ को इम्पोर्ट करना होगा। इम्पोर्ट ब्लॉक को बिल्कुल जैसा दिखाया गया है वैसा ही रखें – यह ट्यूटोरियल में बाद में उपयोग होने वाली क्लासेज़ प्रदान करता है।

```java
import com.aspose.cad.Image;

import com.aspose.cad.fileformats.cad.CadImage;
import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PngOptions;
```

## CAD को PNG के रूप में सहेजने का तरीका

नीचे एक संक्षिप्त, चरण‑दर‑चरण गाइड है जो **OLE ऑब्जेक्ट्स को एक्सपोर्ट करने और CAD को PNG के रूप में सहेजने** का तरीका दिखाता है। प्रत्येक चरण में एक छोटा विवरण और फिर वह कोड दिया गया है जिसे आप अपने प्रोजेक्ट में कॉपी कर सकते हैं।

### चरण 1: अपना दस्तावेज़ डायरेक्टरी सेट करें

उस फ़ोल्डर को परिभाषित करें जिसमें आपके CAD ड्रॉइंग्स हैं। प्लेसहोल्डर को अपने मशीन पर वास्तविक पाथ से बदलें।

```java
String dataDir = "Your Document Directory" + "DWGDrawings/";
```

### चरण 2: CAD फ़ाइल नाम निर्धारित करें

उन सभी CAD फ़ाइलों की सूची बनाएं जिन्हें आप प्रोसेस करना चाहते हैं। आवश्यकता अनुसार जितने भी एंट्रीज़ जोड़ें।

```java
String[] files = new String[] { "D ZD junior D10m H2m.dwg", "ZD - Senior D6m H2m45.dwg" };
```

### चरण 3: PNG एक्सपोर्ट विकल्प सेट करें

रास्टराइज़ेशन सेटिंग्स को कॉन्फ़िगर करें। यहाँ हम एक विशिष्ट लेआउट (`Layout1`) को टार्गेट कर रहे हैं और डिफ़ॉल्ट PNG विकल्पों का उपयोग कर रहे हैं।

```java
PngOptions pngOptions = new PngOptions();
CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
pngOptions.setVectorRasterizationOptions(rasterizationOptions);
rasterizationOptions.setLayouts(new String[] { "Layout1" });
```

### चरण 4: CAD फ़ाइलों पर इटरेट करें और PNG के रूप में सहेजें

प्रत्येक CAD फ़ाइल को लोड करें, OLE ऑब्जेक्ट्स को रास्टराइज़ करें, और आउटपुट PNG फ़ाइल लिखें। `_out.png` उपसर्ग आपको जनरेटेड इमेजेज़ को पहचानने में मदद करता है।

```java
for(String file : files)
{
    CadImage cadImage = (CadImage)Image.load(dataDir + file);
    cadImage.save(dataDir + file + "_out.png", pngOptions);
}
```

## सामान्य समस्याएँ और समाधान

| समस्या | क्यों होता है | समाधान |
|---------|----------------|-----|
| **`Image.load` throws `FileNotFoundException`** | `dataDir` पाथ गलत है या फ़ाइल नाम में टाइपो है। | डायरेक्टरी स्ट्रिंग को दोबारा जांचें और सुनिश्चित करें कि फ़ाइल नाम बिल्कुल मेल खाता हो, स्पेसेस सहित। |
| **आउटपुट PNG खाली है** | लेआउट नाम स्रोत CAD फ़ाइल में मौजूद नहीं है। | `cadImage.getLayouts()` का उपयोग करके उपलब्ध लेआउट्स की सूची देखें, फिर `rasterizationOptions.setLayouts(...)` को अपडेट करें। |
| **बड़ी CAD फ़ाइलों से OutOfMemoryError** | हाई‑रिज़ॉल्यूशन इमेजेज़ रास्टराइज़ करने से बहुत मेमोरी खपत होती है। | JVM हीप (`-Xmx2g`) बढ़ाएँ या `rasterizationOptions.setResolution(...)` से रिज़ॉल्यूशन कम करें। |
