# CAD को PNG के रूप में सहेजें – Aspose.CAD for Java का उपयोग करके OLE ऑब्जेक्ट्स निर्यात करें

## परिचय

कंप्यूटर-एडेड डिज़ाइन (CAD) की गतिशील दुनिया में, OLE (ऑब्जेक्ट लिंकिंग और एम्बेडिंग) ऑब्जेक्ट्स को कुशलतापूर्वक प्रबंधित और निकालना—और **CAD को PNG के रूप में सहेजना**—रिपोर्टिंग, वेब प्रीव्यू या अभिलेखीय कार्यों जैसे डाउनस्ट्रीम वर्कफ़्लो के लिए महत्वपूर्ण है। Aspose.CAD for Java एक शक्तिशाली, कोड‑फ़र्स्ट समाधान प्रदान करता है जो आपको OLE ऑब्जेक्ट्स को निर्यात करने और CAD ड्रॉइंग्स को उच्च‑गुणवत्ता वाले PNG इमेज में कुछ ही कोड लाइनों में परिवर्तित करने की सुविधा देता है।

## त्वरित उत्तर
- **Aspose.CAD क्या करता है?** यह CAD फ़ाइलें (DWG, DXF, DGN, आदि) को पढ़ता, संशोधित करता और परिवर्तित करता है बिना मूल CAD सॉफ़्टवेयर की आवश्यकता के।  
- **क्या मैं CAD को PNG के रूप में सहेज सकता हूँ?** हाँ—किसी भी लेआउट को रास्टराइज़ करने के लिए `PngOptions` को `CadRasterizationOptions` के साथ उपयोग करें।  
- **OLE ऑब्जेक्ट्स को कैसे निर्यात करें?** `Image.load` के साथ CAD फ़ाइल लोड करें और प्रत्येक OLE‑समाहित लेआउट को PNG के रूप में सहेजें।  
- **क्या मुझे लाइसेंस चाहिए?** एक मुफ्त ट्रायल उपलब्ध है; एक व्यावसायिक लाइसेंस मूल्यांकन सीमाओं को हटा देता है।  
- **कौन सा Java संस्करण आवश्यक है?** Java 8 या उससे ऊपर पूरी तरह समर्थित है।

## क्या है **CAD को PNG के रूप में सहेजना**?
CAD को PNG के रूप में सहेजना का अर्थ है वेक्टर‑आधारित CAD ड्रॉइंग्स को पिक्सेल‑आधारित PNG इमेज में रास्टराइज़ करना। यह रूपांतरण तब उपयोगी होता है जब आपको वेब पेज, मोबाइल ऐप या दस्तावेज़ीकरण के लिए हल्का, सार्वभौमिक रूप से देखी जाने वाली फ़ॉर्मेट की आवश्यकता होती है।

## Aspose.CAD for Java का उपयोग करके **CAD को PNG में परिवर्तित** क्यों करें?
- **CAD इंस्टॉलेशन की आवश्यकता नहीं** – लाइब्रेरी पूरी तरह से Java में काम करती है।  
- **लेआउट की सटीकता को बनाए रखता है** – आप विशिष्ट लेआउट चुन सकते हैं, DPI नियंत्रित कर सकते हैं, और लाइन क्वालिटी को बरकरार रख सकते हैं।  
- **बैच प्रोसेसिंग** – एक सरल लूप के साथ कई फ़ाइलों पर इटररेट करें।  
- **OLE ऑब्जेक्ट्स निर्यात करें** – DWG/DXF फ़ाइलों में एम्बेडेड OLE कंटेंट स्वचालित रूप से PNG आउटपुट में रेंडर हो जाता है।

## पूर्वापेक्षाएँ

ट्यूटोरियल में जाने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ मौजूद हैं:

- **Java पर्यावरण** – सुनिश्चित करें कि आपके मशीन पर Java विकास पर्यावरण स्थापित है।  
- **Aspose.CAD for Java** – Aspose.CAD for Java लाइब्रेरी डाउनलोड और इंस्टॉल करें। आप लाइब्रेरी को [download link](https://releases.aspose.com/cad/java/) पर पा सकते हैं।  
- **CAD फ़ाइलें** – उन CAD फ़ाइलों को तैयार करें जिनमें OLE ऑब्जेक्ट्स हों जिन्हें आप निर्यात करना चाहते हैं।

## Namespaces आयात करें

शुरू करने के लिए, अपने Java प्रोजेक्ट में आवश्यक namespaces आयात करें। ये namespaces Aspose.CAD का उपयोग करके CAD फ़ाइलों के साथ काम करने के लिए आवश्यक क्लासेज़ और कार्यक्षमताएँ प्रदान करते हैं।

```java
import com.aspose.cad.Image;

import com.aspose.cad.fileformats.cad.CadImage;
import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PngOptions;
```

अब, CAD से OLE ऑब्जेक्ट्स को निर्यात करने की प्रक्रिया को कई चरणों में विभाजित करते हैं:

## कैसे **CAD को PNG के रूप में सहेजें** और OLE ऑब्जेक्ट्स निर्यात करें

### चरण 1: अपना दस्तावेज़ निर्देशिका सेट करें

```java
String dataDir = "Your Document Directory" + "DWGDrawings/";
```

`"Your Document Directory"` को उस पथ से बदलें जहाँ आपके CAD फ़ाइलें स्थित हैं।

### चरण 2: CAD फ़ाइल नाम निर्धारित करें

```java
String[] files = new String[] { "D ZD junior D10m H2m.dwg", "ZD - Senior D6m H2m45.dwg" };
```

`files` एरे में उन CAD फ़ाइलों के नाम निर्दिष्ट करें जिन्हें आप प्रोसेस करना चाहते हैं।

### चरण 3: PNG निर्यात विकल्प सेट करें

```java
PngOptions pngOptions = new PngOptions();
CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
pngOptions.setVectorRasterizationOptions(rasterizationOptions);
rasterizationOptions.setLayouts(new String[] { "Layout1" });
```

PNG निर्यात विकल्पों को कॉन्फ़िगर करें, जिसमें वेक्टर रास्टराइज़ेशन और लेआउट सेटिंग्स शामिल हैं। ये विकल्प ही आपको वांछित गुणवत्ता के साथ **CAD को PNG में परिवर्तित** करने में सक्षम बनाते हैं।

### चरण 4: CAD फ़ाइलों के माध्यम से इटररेट करें

```java
for(String file : files)
{
    CadImage cadImage = (CadImage)Image.load(dataDir + file);
    cadImage.save(dataDir + file + "_out.png", pngOptions);
}
```

प्रत्येक निर्दिष्ट CAD फ़ाइल के माध्यम से इटररेट करें, इसे Aspose.CAD का उपयोग करके लोड करें, और OLE ऑब्जेक्ट्स को PNG इमेज के रूप में सहेजें। यह लूप बैच में **DWG को PNG में परिवर्तित** करने का एक सरल तरीका दर्शाता है।

## सामान्य समस्याएँ और समाधान

| समस्या | कारण | समाधान |
|-------|-------|----------|
| **खाली PNG आउटपुट** | लेआउट नाम का मेल नहीं होना | सुनिश्चित करें कि लेआउट नाम (`"Layout1"`) स्रोत DWG में मौजूद है। |
| **OLE ग्राफ़िक्स गायब** | OLE ऑब्जेक्ट्स अलग लेआउट में संग्रहीत हैं | `rasterizationOptions.setLayouts(...)` में सभी संबंधित लेआउट शामिल करें। |
| **बड़ी फ़ाइलों पर मेमोरी समाप्ति त्रुटि** | उच्च DPI सेटिंग्स | `rasterizationOptions.setResolution(...)` के माध्यम से DPI घटाएँ या फ़ाइलों को एक‑एक करके प्रोसेस करें। |
