# Pen Support in Export

## परिचय

CAD रूपांतरण की तेज़ गति वाली दुनिया में, डेवलपर्स को अक्सर **CAD से PDF बनाना** पड़ता है जबकि दृश्य गुणवत्ता को बनाए रखा जाता है। Aspose.CAD for Java इसे सरल बनाता है, पेन कस्टमाइज़ेशन जैसी समृद्ध विकल्प प्रदान करता है जिससे आप निर्यात प्रक्रिया के दौरान लाइन स्टाइल को बारीकी से समायोजित कर सकते हैं। इस गाइड में हम एक पूर्ण, व्यावहारिक उदाहरण के माध्यम से दिखाएंगे कि कैसे **CAD को PDF में निर्यात** किया जाए कस्टम पेन सेटिंग्स के साथ, ताकि आप DXF ड्रॉइंग्स से सीधे परिष्कृत PDFs बना सकें।

## त्वरित उत्तर
- **“CAD से PDF बनाना” का क्या अर्थ है?** एक CAD ड्रॉइंग (जैसे DXF) को PDF दस्तावेज़ में बदलना, जबकि वेक्टर गुणवत्ता को बरकरार रखना।  
- **पेन कस्टमाइज़ेशन कौन सी लाइब्रेरी संभालती है?** Aspose.CAD for Java की `PenOptions` क्लास।  
- **क्या मैं इसे अन्य फ़ॉर्मेट्स के लिए उपयोग कर सकता हूँ?** हाँ – वही पेन सेटिंग्स PNG, BMP, TIFF आदि पर लागू होती हैं।  
- **क्या मुझे लाइसेंस चाहिए?** प्रोडक्शन उपयोग के लिए एक वैध Aspose.CAD लाइसेंस आवश्यक है।  
- **न्यूनतम Java संस्करण क्या है?** Java 8 या उससे ऊपर।

## “CAD से PDF बनाना” क्या है?
CAD से PDF बनाना का मतलब है CAD ड्रॉइंग को PDF फ़ाइल में रास्टराइज़ या वेक्टर‑रेंडर करना। इससे इंजीनियरिंग डिज़ाइनों को आसानी से साझा, प्रिंट और अभिलेखित किया जा सकता है, बिना प्राप्तकर्ता के पास CAD सॉफ़्टवेयर होने की आवश्यकता के।

## CAD को PDF में निर्यात करते समय पेन समर्थन क्यों उपयोग करें?
पेन समर्थन आपको लाइन कैप, जॉइन और मोटाई को नियंत्रित करने की अनुमति देता है, जिससे आप कॉर्पोरेट ब्रांडिंग या तकनीकी ड्रॉइंग मानकों के अनुरूप बना सकते हैं। यह विशेष रूप से तब उपयोगी होता है जब डिफ़ॉल्ट लाइन रेंडरिंग आपके दृश्य आवश्यकताओं को पूरा नहीं करती।

## CAD से PDF कैसे बनाएं – चरण‑दर‑चरण गाइड
नीचे एक व्यावहारिक walkthrough दिया गया है जो पर्यावरण सेटअप से लेकर अंतिम PDF जनरेट करने तक सब कुछ कवर करता है। प्रत्येक चरण का पालन करें, और आपके पास **CAD को PDF में निर्यात** करने के लिए पूर्ण पेन नियंत्रण के साथ एक तैयार‑से‑उपयोग समाधान होगा।

## पूर्वापेक्षाएँ

- **Java Development Environment** – एक कार्यशील JDK (8 या नया) और आपका पसंदीदा IDE या बिल्ड टूल।  
- **Aspose.CAD Library** – आधिकारिक साइट से नवीनतम JAR डाउनलोड करें [here](https://releases.aspose.com/cad/java/)।  
- **एक नमूना DXF फ़ाइल** – इस ट्यूटोरियल के लिए हम `conic_pyramid.dxf` का उपयोग करेंगे।

अब जब हमने मंच तैयार कर लिया है, चलिए कोड में डुबकी लगाते हैं।

## नामस्थान आयात करें

```java
import com.aspose.cad.Image;

import com.aspose.cad.fileformats.cad.CadImage;
import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PdfOptions;
import com.aspose.cad.imageoptions.PenOptions;
import com.aspose.cad.internal.imaging.LineCap;
```

## चरण 1: अपना दस्तावेज़ निर्देशिका निर्धारित करें

```java
String dataDir = "Your Document Directory" + "CADConversion/";
```

> **Pro tip:** `"Your Document Directory"` को उस पूर्ण पथ से बदलें जहाँ आपके DXF फ़ाइलें स्थित हैं।

## चरण 2: CAD फ़ाइल लोड करें

```java
String srcFile = dataDir + "conic_pyramid.dxf";
CadImage cadImage = (CadImage) Image.load(srcFile);
```

`Image.load` मेथड DXF फ़ाइल को पढ़ता है और एक `CadImage` ऑब्जेक्ट बनाता है जिसे हम आगे संशोधित कर सकते हैं।

## चरण 3: रास्टराइज़ेशन विकल्प कॉन्फ़िगर करें

```java
CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
rasterizationOptions.setPageWidth(cadImage.getWidth() * 100);
rasterizationOptions.setPageHeight(cadImage.getHeight() * 100);
```

पृष्ठ आयाम को समायोजित करके आप उत्पन्न PDF की रिज़ॉल्यूशन नियंत्रित कर सकते हैं। 100 से गुणा करने पर प्रिंटिंग के लिए उपयुक्त उच्च‑रिज़ॉल्यूशन आउटपुट मिलता है।

## चरण 4: पेन विकल्प अनुकूलित करें

```java
PenOptions penOts = new PenOptions();
penOts.setStartCap(LineCap.Flat);
penOts.setEndCap(LineCap.Flat);
```

यहाँ हम पेन के दोनों प्रारंभ और अंत कैप को `Flat` सेट करते हैं। आप विभिन्न `LineCap` मानों (जैसे `Round`, `Square`) के साथ प्रयोग करके अलग‑अलग दृश्य प्रभाव प्राप्त कर सकते हैं।

## चरण 5: PDF निर्यात विकल्प कॉन्फ़िगर करें

```java
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.setVectorRasterizationOptions(rasterizationOptions);
```

`PdfOptions` ऑब्जेक्ट रास्टराइज़ेशन सेटिंग्स को PDF निर्यात प्रक्रिया से जोड़ता है।

## चरण 6: निर्यातित PDF सहेजें

```java
cadImage.save((dataDir + "9LHATT-A56_generated.pdf"), pdfOptions);
```

इस लाइन को चलाने से `9LHATT-A56_generated.pdf` नाम की PDF फ़ाइल आपके `dataDir` फ़ोल्डर में लिखी जाएगी, जिसमें आपने परिभाषित कस्टम पेन स्टाइलिंग शामिल होगी।

## सामान्य उपयोग केस

- **तकनीकी दस्तावेज़ीकरण** – PDF मैनुअल में सटीक इंजीनियरिंग ड्रॉइंग्स एम्बेड करें।  
- **स्वचालित रिपोर्टिंग** – वेब सेवाओं में CAD डेटा से तुरंत PDFs जनरेट करें।  
- **गुणवत्ता नियंत्रण** – माप लाइन या टॉलरेंस को हाइलाइट करने के लिए कस्टम लाइन कैप लागू करें।

## समस्या निवारण और टिप्स

- **गलत फ़ाइल पथ** – सुनिश्चित करें कि `dataDir` फ़ाइल सेपरेटर (`/` या `\\`) के साथ समाप्त हो।  
- **लाइसेंस अनुपलब्ध** – वैध लाइसेंस के बिना लाइब्रेरी एवाल्यूएशन मोड में चलती है, जिससे वॉटरमार्क जोड़ सकता है।  
- **अप्रत्याशित लाइन स्टाइल** – `save` कॉल करने से पहले `PenOptions` सेट हैं या नहीं, दोबारा जांचें; अन्यथा डिफ़ॉल्ट उपयोग होते हैं।

## अक्सर पूछे जाने वाले प्रश्न

### Q1: क्या मैं PDF के अलावा अन्य फ़ॉर्मेट्स के लिए पेन विकल्प कस्टमाइज़ कर सकता हूँ?

A1: हाँ, इस ट्यूटोरियल में दिखाए गए पेन कस्टमाइज़ेशन को PDF, PNG, BMP, GIF, JPEG2000, JPEG, PSD, TIFF, और WMF सहित विभिन्न इमेज फ़ॉर्मेट्स पर लागू किया जा सकता है।

### Q2: पेन के विभिन्न प्रारंभ और अंत कैप कैसे सेट करूँ?

A2: `PenOptions` क्लास का उपयोग करके इच्छित प्रारंभ और अंत कैप सेट करें, जिससे लाइनों की उपस्थिति को लचीले ढंग से परिभाषित किया जा सकता है।

### Q3: यदि मैं पेन विकल्प नहीं बताता तो क्या होगा?

A3: यदि पेन विकल्प स्पष्ट रूप से सेट नहीं किए जाते, तो सिस्टम अपने डिफ़ॉल्ट पेन का उपयोग करेगा, जो विभिन्न संदर्भों में अलग‑अलग हो सकते हैं।

### Q4: रास्टराइज़ेशन विकल्पों के लिए कोई विशेष विचार हैं?

A4: निर्यातित इमेज के आयाम को नियंत्रित करने के लिए रास्टराइज़ेशन विकल्पों में पृष्ठ की चौड़ाई और ऊँचाई को समायोजित करें।

### Q5: अतिरिक्त समर्थन या समुदाय चर्चा कहाँ मिल सकती है?

A5: समर्थन और चर्चाओं के लिए Aspose.CAD समुदाय फ़ोरम देखें [here](https://forum.aspose.com/c/cad/19)।
