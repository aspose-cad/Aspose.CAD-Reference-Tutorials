# Aspose.CAD for Java का उपयोग करके DWG को BMP में कैसे कन्वर्ट करें

## परिचय

यदि आप Java से **dwg को bmp में कन्वर्ट** करने का स्पष्ट और भरोसेमंद तरीका खोज रहे हैं, तो आप सही जगह पर आए हैं। Aspose.CAD for Java के साथ आप न केवल ड्राइंग आकारों को ऑटो‑एडजस्ट कर सकते हैं बल्कि **CAD** फ़ाइलों को BMP, PNG, PDF और कई अन्य फ़ॉर्मैट में **एक्सपोर्ट** भी कर सकते हैं। यह ट्यूटोरियल आपको पूरे प्रोसेस के माध्यम से ले जाता है—पर्यावरण सेटअप से लेकर एक BMP इमेज जेनरेट करने तक जो मूल CAD लेआउट को संरक्षित रखता है—और साथ ही दिखाता है कि आप **बैच में CAD** ड्रॉइंग्स को कैसे कन्वर्ट या **CAD लेआउट** को चयनात्मक रूप से कैसे रेंडर कर सकते हैं।

## त्वरित उत्तर
- **“dwg को bmp में कन्वर्ट” का क्या मतलब है?** यह DWG (या अन्य CAD) फ़ाइल को BMP रास्टर इमेज के रूप में रेंडर करने को दर्शाता है।  
- **कौन सी लाइब्रेरी इस कन्वर्ज़न को संभालती है?** Aspose.CAD for Java इस कार्य के लिए एक सरल, शुद्ध‑Java API प्रदान करता है।  
- **क्या मुझे लाइसेंस चाहिए?** परीक्षण के लिए एक टेम्पररी लाइसेंस काम करता है; प्रोडक्शन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **क्या मैं एक साथ कई लेआउट एक्सपोर्ट कर सकता हूँ?** हाँ – `Layouts` प्रॉपर्टी सेट करके आप निर्दिष्ट कर सकते हैं कि कौन से लेआउट रेंडर करने हैं।  
- **क्या BMP ही एकमात्र आउटपुट फ़ॉर्मैट है?** नहीं – आप PNG, JPEG, TIFF, PDF और अधिक में भी एक्सपोर्ट कर सकते हैं।

## Aspose.CAD for Java का उपयोग करके DWG को BMP में कैसे कन्वर्ट करें
इस सेक्शन में हम मुख्य प्रश्न का सीधा उत्तर देते हैं और आगे के चरण‑बद्ध गाइड की तैयारी करते हैं।

### Aspose.CAD के साथ “dwg को bmp में कन्वर्ट” क्या है?
DWG को BMP में कन्वर्ट करना मतलब एक मूल CAD ड्रॉइंग (जैसे DWG फ़ाइल) को बिटमैप इमेज में रास्टराइज़ करना है। Aspose.CAD सभी जटिल कार्य संभालता है: CAD संरचना को पार्स करना, वेक्टर एंटिटीज़ को रेंडर करना, और परिणाम को एक BMP फ़ाइल में लिखना जो मूल लेआउट के आयाम और रंगों से मेल खाती है।

### **DWG को BMP में कन्वर्ट** करने के लिए Aspose.CAD for Java क्यों उपयोग करें?
- **शुद्ध Java इम्प्लीमेंटेशन** – कोई नेटिव DLL या बाहरी टूल्स की आवश्यकता नहीं।  
- **विस्तृत फ़ॉर्मैट सपोर्ट** – DWG, DXF, DGN और कई अन्य CAD फ़ॉर्मैट नेटिव रूप से पढ़े जा सकते हैं।  
- **सूक्ष्म नियंत्रण** – आप विशिष्ट लेआउट चुन सकते हैं, DPI, बैकग्राउंड कलर आदि सेट कर सकते हैं।  
- **स्केलेबल परफ़ॉर्मेंस** – सर्वर या डेस्कटॉप यूटिलिटीज़ में बैच में CAD फ़ाइलों को कन्वर्ट करने के लिए आदर्श।  

## आवश्यकताएँ

शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. **Java Development Kit** – इसे [यहाँ](https://www.java.com/en/download/) से डाउनलोड करें।  
2. **Aspose.CAD for Java लाइब्रेरी** – नवीनतम JAR [यहाँ](https://releases.aspose.com/cad/java/) से प्राप्त करें।  
3. **सैंपल CAD फ़ाइल** – एक DWG फ़ाइल (जैसे `sample.dwg`) को अपने प्रोजेक्ट की `document` डायरेक्टरी में रखें।

## नेमस्पेस इम्पोर्ट करें

अपने Java एप्लिकेशन में Aspose.CAD फ़ंक्शनैलिटी का उपयोग करने के लिए आवश्यक नेमस्पेस शामिल करें। नीचे एक उदाहरण दिया गया है:

```java
import com.aspose.cad.Image;
import com.aspose.cad.ImageOptionsBase;

import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PngOptions;
```

अब, **dwg को bmp में कन्वर्ट** प्रक्रिया को प्रबंधनीय चरणों में विभाजित करते हैं।

## चरण‑दर‑चरण गाइड

### चरण 1: CAD ड्रॉइंग लोड करें

```java
String dataDir = "Your Document Directory" + "CADConversion/";
String sourceFilePath = dataDir + "sample.dwg";
Image objImage = Image.load(sourceFilePath);
```

हम स्रोत DWG फ़ाइल को एक `Image` ऑब्जेक्ट में लोड करते हैं, जो सभी आगे के ऑपरेशन्स का एंट्री पॉइंट है।

### चरण 2: `BmpOptions` बनाएं

```java
BmpOptions bmpOptions = new BmpOptions();
```

`BmpOptions` BMP आउटपुट के लिए विशेष रास्टराइज़ेशन सेटिंग्स रखेगा।

### चरण 3: रास्टराइज़ेशन सेटिंग्स कॉन्फ़िगर करें

```java
CadRasterizationOptions cadRasterizationOptions = new CadRasterizationOptions();
bmpOptions.setVectorRasterizationOptions(cadRasterizationOptions);
```

यहाँ हम रास्टराइज़ेशन विकल्पों को BMP विकल्पों से लिंक करते हैं, जिससे हम नियंत्रित कर सकते हैं कि CAD एंटिटीज़ कैसे रेंडर होंगी।

### चरण 4: एक्सपोर्ट करने के लिए लेआउट(s) सेट करें

```java
cadRasterizationOptions.setLayouts(new String[]{"Model"});
```

`Layouts` प्रॉपर्टी Aspose.CAD को बताती है कि कौन से ड्रॉइंग लेआउट शामिल करने हैं। अधिकांश मामलों में, `"Model"` वह प्राथमिक लेआउट है जिसे आप कन्वर्ट करना चाहते हैं, लेकिन आप **कई CAD लेआउट्स को एक्सपोर्ट** करने के लिए लेआउट नामों की एरे पास कर सकते हैं।

### चरण 5: BMP फ़ॉर्मैट में एक्सपोर्ट करें

```java
String outPath = sourceFilePath + ".bmp";
objImage.save(outPath, bmpOptions);
```

कॉन्फ़िगर किए गए `BmpOptions` के साथ `save` कॉल करने से BMP फ़ाइल डिस्क पर लिखी जाती है। यह **dwg को bmp में कन्वर्ट** वर्कफ़्लो को पूरा करता है।

## बैच में CAD फ़ाइलों को कन्वर्ट करने के अतिरिक्त टिप्स

- **डायरेक्टरी के माध्यम से लूप** करें और प्रत्येक DWG फ़ाइल पर वही चरण लागू करें; इससे **बैच में CAD** कन्वर्ज़न संभव हो जाता है।
