## Respuestas rápidas
- **¿Qué significa “convertir CAD a PDF”?** Transforma dibujos CAD nativos en archivos PDF universalmente visualizables mientras preserva capas y metadatos.  
- **¿Qué biblioteca gestiona la conversión?** Aspose.CAD para .NET ofrece una API limpia sin necesidad de AutoCAD.

## ¿Qué es “convertir CAD a PDF”?
Convertir CAD a PDF significa tomar un archivo DWG, DXF u otro archivo CAD y generar un PDF que conserve la fidelidad visual del dibujo original. Esto es esencial para compartir diseños con partes interesadas que pueden no disponer de software CAD.

## Cómo convertir CAD a PDF usando Aspose.CAD para .NET
A continuación se muestra un proceso conciso, paso a paso, que puedes seguir en cualquier proyecto .NET.

### Paso 1: Instalar el paquete NuGet Aspose.CAD
Abre el administrador de paquetes NuGet de tu proyecto y agrega `Aspose.CAD`. Esto incluye todos los ensamblados requeridos.

### Paso 2: Cargar tu archivo CAD
Utiliza la clase `Image` para abrir un DWG, DXF u otro formato compatible.

### Paso 3: Configurar opciones de exportación PDF (opcional)
Puedes especificar el tamaño de página, la DPI de rasterizado y si se incluyen capas ocultas.

### Paso 4: Guardar el dibujo como PDF
Llama al método `Save` con el objeto `PdfOptions` para producir un PDF de alta calidad.
