- **¿Cuál es el propósito principal de Aspose.CAD para .NET?** Leer, editar y convertir archivos CAD y BIM (incluyendo DWG, DWFX, DWF) de forma programática.  
- **¿Puedo abrir un archivo DWFX directamente en C#?** Sí – Aspose.CAD ofrece una API sencilla para cargar y acceder al contenido DWFX.  
- **¿Necesito una licencia para desarrollo?** Una licencia de evaluación gratuita funciona para pruebas; se requiere una licencia comercial para producción.  
- **¿Qué versiones de .NET son compatibles?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Preguntas frecuentes

**Q: ¿Cómo abro un archivo DWFX en C#?**  
A: Usa `Aspose.CAD.Image.Load("file.dwfx")` para cargar el archivo, luego accede a sus páginas o entidades según sea necesario.

**Q: ¿Puedo leer un archivo DWG con C# sin convertirlo primero?**  
A: Sí – Aspose.CAD puede cargar directamente archivos DWG, permitiéndote consultar capas, entidades y metadatos.

**Q: ¿Cuál es la forma más sencilla de agregar texto a un dibujo DWG?**  
A: Crea una entidad `CadText`, establece su `Value` y posición, luego añádela a la colección del espacio modelo del dibujo.

**Q: ¿Existen límites de tamaño al trabajar con archivos DWFX o DWG grandes?**  
A: La biblioteca maneja archivos grandes de manera eficiente, pero deberías asegurar suficiente memoria y considerar la transmisión de imágenes raster grandes.

**Q: ¿Necesito una licencia separada para cada formato CAD?**  
A: No – una única licencia de Aspose.CAD cubre todos los formatos compatibles, incluidos DWG, DWFX, DWF, DXF y más.
