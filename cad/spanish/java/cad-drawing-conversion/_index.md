## ¿Qué es “export dwg to pdf”?

Exportar un archivo DWG a PDF crea un documento independiente del dispositivo, listo para compartir, que preserva la fidelidad visual exacta del dibujo original. Esto es especialmente útil para revisiones de clientes, documentación o archivado donde el receptor puede no tener instalado software CAD.

## Preguntas frecuentes

**P: ¿Cómo exporto un archivo DWG a PDF sin perder la información de capas?**  
R: Use `PdfOptions` y establezca `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` para conservar los datos vectoriales, luego llame a `image.save("output.pdf", options);`.

**P: ¿Puedo procesar por lotes varios archivos DWG a PDF en una sola ejecución?**  
R: Sí—itere sobre su colección de archivos, cargue cada uno con `Image.load` y guarde usando las mismas opciones PDF.

**P: ¿Existe una forma de convertir una plantilla DWT directamente a DXF?**  
R: Absolutamente. Cargue el DWT con `Image.load` y guárdelo como DXF usando `image.save("output.dxf");`.

**P: ¿Qué versiones de Java son compatibles con Aspose.CAD?**  
R: La biblioteca funciona con Java 8 y versiones posteriores, incluyendo Java 11, 17 y demás versiones LTS.

**P: ¿Necesito una licencia separada para la exportación a PDF?**  
R: No, la licencia de Aspose.CAD cubre todos los formatos de salida, incluidos PDF, imágenes raster y DXF.
