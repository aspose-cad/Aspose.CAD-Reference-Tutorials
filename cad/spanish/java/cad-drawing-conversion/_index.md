## Preguntas frecuentes

**Q: ¿Cómo exporto un archivo DWG a PDF sin perder la información de capas?**  
A: Utiliza `PdfOptions` y establece `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` para conservar los datos vectoriales, luego llama a `image.save("output.pdf", options);`.

**Q: ¿Puedo procesar por lotes varios archivos DWG a PDF en una sola ejecución?**  
A: Sí—recorre tu colección de archivos, carga cada uno con `Image.load` y guarda usando las mismas opciones PDF.

**Q: ¿Hay una forma de convertir una plantilla DWT directamente a DXF?**  
A: Absolutamente. Carga el DWT con `Image.load` y guárdalo como DXF usando `image.save("output.dxf");`.

**Q: ¿Qué versiones de Java son compatibles con Aspose.CAD?**  
A: La biblioteca funciona con Java 8 y versiones posteriores, incluyendo Java 11, 17 y posteriores versiones LTS.

**Q: ¿Necesito una licencia separada para la exportación a PDF?**  
A: No, la licencia de Aspose.CAD cubre todos los formatos de salida, incluyendo PDF, imágenes raster y DXF.

**Q: ¿Cómo puedo mejorar la velocidad de renderizado al convertir archivos DWG grandes?**  
A: Habilita el multihilo procesando cada diseño en un hilo separado y reutiliza una única instancia de `PdfOptions` para reducir la sobrecarga.

**Q: ¿Puedo establecer un tamaño de página personalizado para el PDF exportado?**  
A: Sí, configura `PdfOptions` con `PdfOptions.setPageSize(PageSize.A4)` (o cualquier otro tamaño) antes de guardar.
