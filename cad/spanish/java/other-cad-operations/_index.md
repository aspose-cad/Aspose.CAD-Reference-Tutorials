## Preguntas frecuentes

**Q: ¿Puedo agregar una marca de agua a un archivo DWG que ya está protegido con contraseña?**  
A: Sí. Carga el DWG con la contraseña adecuada, aplica la marca de agua usando la API y luego guarda el archivo.

**Q: ¿Agregar una marca de agua afecta significativamente el tamaño del archivo?**  
A: El aumento es mínimo porque la marca de agua se almacena como una superposición ligera.

**Q: ¿Es posible procesar por lotes varios archivos CAD para agregar la misma marca de agua?**  
A: Absolutamente. Recorre tu colección de archivos y ejecuta la misma rutina de marca de agua para cada dibujo.

**Q: ¿Puedo controlar la opacidad y la rotación de la marca de agua?**  
A: La API proporciona propiedades para establecer opacidad, ángulo de rotación y posicionamiento, dándote control visual total.

**Q: ¿Qué versiones de Java son compatibles con estos tutoriales?**  
A: Java 8, 11 y versiones LTS más recientes son totalmente compatibles.
