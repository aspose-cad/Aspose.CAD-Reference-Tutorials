## Pertanyaan yang Sering Diajukan

**Q: Bagaimana cara mengekspor file DWG ke PDF tanpa kehilangan informasi lapisan?**  
A: Gunakan `PdfOptions` dan atur `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` untuk mempertahankan data vektor, kemudian panggil `image.save("output.pdf", options);`.

**Q: Bisakah saya memproses batch beberapa file DWG ke PDF dalam satu kali jalan?**  
A: Ya—lakukan loop melalui koleksi file Anda, muat masing‑masing dengan `Image.load`, dan simpan menggunakan opsi PDF yang sama.

**Q: Apakah ada cara mengonversi template DWT langsung ke DXF?**  
A: Tentu. Muat DWT dengan `Image.load` dan simpan sebagai DXF menggunakan `image.save("output.dxf");`.

**Q: Versi Java apa yang didukung oleh Aspose.CAD?**  
A: Perpustakaan ini bekerja dengan Java 8 dan yang lebih baru, termasuk Java 11, 17, dan rilis LTS selanjutnya.

**Q: Apakah saya memerlukan lisensi terpisah untuk ekspor PDF?**  
A: Tidak, lisensi Aspose.CAD mencakup semua format output, termasuk PDF, gambar raster, dan DXF.

**Q: Bagaimana cara meningkatkan kecepatan rendering saat mengonversi file DWG besar?**  
A: Aktifkan multi‑threading dengan memproses setiap layout dalam thread terpisah dan gunakan kembali satu instance `PdfOptions` untuk mengurangi beban.

**Q: Bisakah saya mengatur ukuran halaman khusus untuk PDF yang diekspor?**  
A: Ya, konfigurasikan `PdfOptions` dengan `PdfOptions.setPageSize(PageSize.A4)` (atau ukuran lain) sebelum menyimpan.
