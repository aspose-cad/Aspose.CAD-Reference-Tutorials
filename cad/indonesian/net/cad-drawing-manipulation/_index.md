# Manipulasi Gambar CAD

## Pendahuluan

Jika Anda mencari **cara mengubah ukuran CAD** dengan cepat dan andal, Anda berada di tempat yang tepat. Panduan ini membawa Anda melalui tugas manipulasi CAD yang paling umum dengan Aspose.CAD untuk .NET, mulai dari mengubah ukuran kanvas hingga mengonversi gambar menjadi raster. Anda akan menemukan contoh praktis, kasus penggunaan dunia nyata, dan tip yang membuat alur kerja Anda tetap lancar.

## Jawaban Cepat
- **Bagaimana cara mengubah ukuran kanvas gambar CAD?** Gunakan metode `Resize` Aspose.CAD untuk menetapkan lebar dan tinggi baru.  
- **Apakah saya dapat mengonversi CAD ke gambar raster?** Ya—cukup muat gambar dan simpan sebagai PNG, JPEG, atau BMP.  
- **Format apa saja yang didukung Aspose.CAD untuk konversi?** DWG, DXF, DWF, DGN, STL, dan banyak lagi.  
- **Apakah saya memerlukan lisensi untuk penggunaan produksi?** Lisensi komersial diperlukan untuk penyebaran non‑trial.  
- **Versi .NET mana yang kompatibel?** .NET Framework 4.6+, .NET Core 3.1+, .NET 5/6/7.

## Apa itu “cara mengubah ukuran CAD”?
Mengubah ukuran gambar CAD berarti menyesuaikan sistem koordinat internal atau kanvas sehingga geometri cocok dengan dimensi output yang diinginkan. Dengan Aspose.CAD Anda dapat mengubah ukuran secara programatis tanpa kehilangan detail, menjadikannya ideal untuk pipeline otomatis atau pemrosesan batch.

## Mengapa mengubah ukuran kanvas CAD?
- **Presentasi konsisten** di berbagai perangkat dan printer.  
- **Pemrosesan lanjutan yang disederhanakan** saat mengonversi ke format raster.  
- **Ukuran file berkurang** untuk penampil berbasis web yang hanya memerlukan viewport tertentu.

## Prasyarat
- Visual Studio 2022 atau IDE yang kompatibel dengan .NET apa pun.  
- Perpustakaan Aspose.CAD untuk .NET (dipasang via NuGet).  
- Lisensi Aspose.CAD yang valid untuk penggunaan produksi (versi trial dapat dipakai untuk eksplorasi).

## Kesalahan Umum & Tips
- **Kesalahan:** Lupa mereset asal gambar setelah mengubah ukuran.  
  **Tip:** Gunakan `Drawing.SetOrigin(0, 0)` setelah menerapkan ukuran kanvas baru.  
- **Kesalahan:** Mengonversi file CAD besar tanpa menentukan resolusi raster menghasilkan output yang buram.  
  **Tip:** Tetapkan DPI tinggi (misalnya, 300) saat memanggil `Save` untuk mempertahankan detail.  
- **Kesalahan:** Mengabaikan pemeriksaan lisensi dapat menyebabkan pengecualian runtime.  
  **Tip:** Terapkan lisensi Anda di awal saat aplikasi dimulai.

## Pertanyaan yang Sering Diajukan

**T: Bisakah saya mengubah ukuran kanvas CAD tanpa mengubah geometri gambar?**  
J: Ya. Aspose.CAD memungkinkan Anda memodifikasi dimensi viewport sambil mempertahankan entitas asli.

**T: Apakah memungkinkan melakukan konversi batch banyak file CAD ke PNG?**  
J: Tentu saja. Loop melalui folder, muat setiap file dengan `Image.Load`, dan panggil `Save` dengan format raster yang diinginkan.

**T: Apakah perpustakaan ini mendukung format CAD 3D seperti STL?**  
J: Ya, Aspose.CAD menangani format 2D dan 3D, termasuk STL, OBJ, dan lainnya.

**T: Apakah mengubah ukuran akan memengaruhi ketebalan garis atau ukuran teks?**  
J: Mengubah ukuran kanvas tidak secara otomatis menskalakan ketebalan garis atau teks. Anda harus menyesuaikan properti tersebut secara manual bila diperlukan.

**T: Bagaimana cara mengambil dimensi tepat layout sebelum mengubah ukuran?**  
J: Gunakan properti `Layout.Size` untuk mendapatkan lebar dan tinggi dalam satuan gambar.
