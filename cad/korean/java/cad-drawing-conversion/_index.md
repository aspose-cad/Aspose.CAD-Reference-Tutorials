# DWG를 PDF로 내보내고 CAD 도면 변환

## 소개

Aspose.CAD for Java를 사용하여 CAD 도면의 잠재력을 최대한 활용하십시오. 숙련된 전문가이든 이제 시작하는 사용자이든, 단계별 튜토리얼을 통해 원활한 **export dwg to pdf** 변환, 래스터 이미지 생성 및 기타 일반적인 CAD 조작 작업을 안내합니다. DWG를 래스터로 변환하고, 특정 레이아웃을 내보내며, DWT 파일을 DXF로 변환하는 방법을 확인할 수 있습니다—모두 깔끔한 Java 코드와 외부 종속성 없이 수행됩니다.

## 빠른 답변
- **Java에서 DWG를 PDF로 내보내는 가장 쉬운 방법은 무엇인가요?** Aspose.CAD의 `PdfOptions`와 `Image.save` 메서드를 함께 사용하십시오.  
- **DWG 파일을 직접 래스터 이미지로 변환할 수 있나요?** 예, Aspose.CAD는 PNG, JPEG, BMP 등으로 변환을 지원합니다.  
- **DWG에서 단일 레이아웃만 내보낼 수 있나요?** 물론입니다—저장하기 전에 이름이나 인덱스로 레이아웃을 선택하면 됩니다.  
- **프로덕션 사용을 위해 라이선스가 필요합니까?** 프로덕션에서는 상용 라이선스가 필요합니다; 평가용 무료 체험판을 사용할 수 있습니다.  
- **어떤 보조 형식이 지원되나요?** PDF 외에도 래스터 형식, DXF 및 기타 CAD 호환 형식으로 내보낼 수 있습니다.

## “export dwg to pdf”란 무엇인가요?

DWG 파일을 PDF로 내보내면 장치에 독립적인 공유 준비 문서가 생성되어 원본 도면의 정확한 시각적 충실도를 유지합니다. 이는 클라이언트 검토, 문서화 또는 수신자가 CAD 소프트웨어를 설치하지 않은 경우에도 아카이빙에 특히 유용합니다.

## 자주 묻는 질문

**Q: DWG 파일을 레이어 정보를 잃지 않고 PDF로 내보내려면 어떻게 해야 하나요?**  
A: `PdfOptions`를 사용하고 `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)`를 설정하여 벡터 데이터를 유지한 다음 `image.save("output.pdf", options);`를 호출합니다.

**Q: 한 번에 여러 DWG 파일을 PDF로 배치 처리할 수 있나요?**  
A: 예—파일 컬렉션을 반복하면서 각 파일을 `Image.load`로 로드하고 동일한 PDF 옵션으로 저장하면 됩니다.

**Q: DWT 템플릿을 직접 DXF로 변환하는 방법이 있나요?**  
A: 물론입니다. `Image.load`로 DWT를 로드하고 `image.save("output.dxf");`를 사용해 DXF로 저장하면 됩니다.

**Q: Aspose.CAD가 지원하는 Java 버전은 무엇인가요?**  
A: 이 라이브러리는 Java 8 및 이후 버전, Java 11, 17 및 이후 LTS 릴리스를 지원합니다.

**Q: PDF 내보내기를 위해 별도의 라이선스가 필요합니까?**  
A: 아니요, Aspose.CAD 라이선스는 PDF, 래스터 이미지, DXF 등 모든 출력 형식을 포함합니다.
