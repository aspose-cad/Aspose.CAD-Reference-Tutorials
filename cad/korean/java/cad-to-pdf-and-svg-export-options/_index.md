## 자주 묻는 질문

**Q:** *큰 DWG 파일을 PDF/A로 변환할 때 메모리가 부족해지지 않나요?*  
A: 네. `CadImage` 클래스를 스트리밍 옵션과 함께 사용하고 이미지를 즉시 해제하여 메모리 사용량을 최소화하세요.

**Q:** *CAD 파일에서 특정 레이아웃만 SVG로 내보낼 수 있나요?*  
A: 물론 가능합니다. 이름으로 원하는 레이아웃을 로드한 뒤 SVG 옵션과 함께 `Save` 메서드를 호출하면 됩니다.

**Q:** *각 출력 형식마다 별도의 라이선스가 필요합니까?*  
A: 아닙니다. Aspose.CAD for Java 하나의 라이선스로 모든 지원 형식을 내보낼 수 있습니다.

**Q:** *한 번에 여러 CAD 파일을 배치 처리할 수 있나요?*  
A: 가능합니다. 파일 컬렉션을 순회하면서 `CadImage.load` 로 각각을 로드하고 대상 형식으로 저장하면 됩니다.

**Q:** *CAD 파일에 래스터 이미지가 포함되어 있으면 PDF에 보존되나요?*  
A: 래스터 이미지는 PDF로 내보낼 때 자동으로 삽입되어 시각적 충실도를 유지합니다.
