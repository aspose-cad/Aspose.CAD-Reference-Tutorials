## 快速解答
- **What is background color CAD?** 它是能為 CAD 渲染設定單色背景色調的功能，可提升可讀性與呈現效果。
- **Which library adds this feature?** Aspose.CAD for .NET 提供簡易的 API 來控制背景顏色。
- **Can I export CAD to PDF with custom backgrounds?** 可以——使用匯出 CAD 為 PDF 的選項並搭配背景設定。
- **Is auto layout scaling supported?** 當然；此函式庫包含自動版面縮放功能，以確保精確的渲染。
- **Do I need a license?** 生產環境使用需具備有效的 Aspose.CAD 授權。

## 常見問題

**Q: 如何變更 CAD 渲染的背景顏色？**  
A: 在呼叫匯出方法前，於渲染選項上設定 `BackgroundColor` 屬性。

**Q: 我可以在匯出 CAD 為 PDF 時保留自訂背景顏色嗎？**  
A: 可以——使用匯出 CAD 為 PDF 的功能，並搭配您的背景顏色設定。

**Q: 什麼是 Auto Layout Scaling，何時應使用？**  
A: Auto Layout Scaling 會自動調整繪圖區域以符合輸出尺寸，確保渲染的精確與比例正確。

**Q: Aspose.CAD 是否同時支援 2D 與 3D DGN 檔案？**  
A: 當然。此函式庫處理 2D 與 3D 元素，並完整支援背景顏色與縮放功能。

**Q: 生產環境使用是否需要授權？**  
A: 需要，商業部署必須擁有有效的 Aspose.CAD 授權；亦提供免費試用供評估使用。
