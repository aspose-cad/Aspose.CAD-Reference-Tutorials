## 常見問題

**Q: 如何在不遺失圖層資訊的情況下將 DWG 檔案匯出為 PDF？**  
A: 使用 `PdfOptions` 並設定 `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` 以保留向量資料，然後呼叫 `image.save("output.pdf", options);`。

**Q: 我可以一次批次處理多個 DWG 檔案匯出為 PDF 嗎？**  
A: 可以——遍歷您的檔案集合，使用 `Image.load` 載入每個檔案，並使用相同的 PDF 選項儲存。

**Q: 有沒有方法直接將 DWT 範本轉換為 DXF？**  
A: 當然可以。使用 `Image.load` 載入 DWT，然後使用 `image.save("output.dxf");` 儲存為 DXF。

**Q: Aspose.CAD 支援哪些 Java 版本？**  
A: 此函式庫支援 Java 8 及以上版本，包括 Java 11、17 以及之後的 LTS 版本。

**Q: 匯出 PDF 需要額外的授權嗎？**  
A: 不需要，Aspose.CAD 授權已涵蓋所有輸出格式，包括 PDF、點陣圖與 DXF。
