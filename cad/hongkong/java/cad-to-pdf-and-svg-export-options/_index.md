## 常見問題

**Q:** *我可以在不耗盡記憶體的情況下將大型 DWG 檔案轉換為 PDF/A 嗎？*  
A: 可以。使用 `CadImage` 類別搭配串流選項，並及時釋放影像，以降低記憶體使用量。

**Q:** *是否可以只匯出 CAD 檔案中的特定版面為 SVG？*  
A: 當然可以。依名稱載入所需的版面，然後使用 SVG 選項呼叫 `Save` 方法。

**Q:** *每種輸出格式都需要單獨的授權嗎？*  
A: 不需要。單一 Aspose.CAD for Java 授權即可涵蓋所有支援的匯出格式。

**Q:** *我可以在一次執行中批次處理多個 CAD 檔案嗎？*  
A: 可以。遍歷檔案集合，使用 `CadImage.load` 載入每個檔案，並儲存為目標格式。

**Q:** *如果我的 CAD 檔案包含點陣圖，匯出為 PDF 時會保留嗎？*  
A: 匯出為 PDF 時，點陣圖會自動嵌入，保持視覺忠實度。
