# CAD Custom Properties and Metadata Tutorials for Aspose.CAD Java

Our custom properties tutorials demonstrate how to manage CAD document metadata using Aspose.CAD in Java. These practical guides show you how to read and write drawing properties, create custom metadata fields, extract XREF information, and implement document information systems. Each tutorial includes working Java code examples for common metadata scenarios, helping you build applications that maintain comprehensive CAD document information for better organization, tracking, and workflow management.
