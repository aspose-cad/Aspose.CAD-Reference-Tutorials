## Часто задаваемые вопросы

**Q: Можно ли добавить водяной знак в DWG‑файл, который уже защищён паролем?**  
A: Да. Загрузите DWG с соответствующим паролем, примените водяной знак через API и затем сохраните файл.

**Q: Влияет ли добавление водяного знака существенно на размер файла?**  
A: Увеличение минимально, так как водяной знак хранится как лёгкое наложение.

**Q: Можно ли пакетно обрабатывать несколько CAD‑файлов, добавляя один и тот же водяной знак?**  
A: Абсолютно. Пройдите по коллекции файлов и вызовите одну и ту же процедуру наложения водяного знака для каждого чертежа.

**Q: Можно ли контролировать непрозрачность и вращение водяного знака?**  
A: API предоставляет свойства для установки непрозрачности, угла вращения и позиционирования, давая вам полный визуальный контроль.

**Q: Какие версии Java поддерживаются в этих учебниках?**  
A: Полностью поддерживаются Java 8, 11 и более новые LTS‑выпуски.
