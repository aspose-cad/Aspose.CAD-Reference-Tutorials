# Включение отслеживания процесса рендеринга CAD

## Введение

В этом руководстве вы узнаете, как **установить размер страницы PDF** при **конвертации CAD в PDF** с помощью **Aspose.CAD for Java**. Включив отслеживание, вы получаете полную видимость конвейера рендеринга, что упрощает отладку и оптимизацию преобразования файлов CAD (например, DXF) в PDF. Независимо от того, нужно ли вам **сохранить CAD как PDF**, создать PDF из DXF или просто контролировать размеры вывода, нижеописанные шаги проведут вас через весь процесс.

## Быстрые ответы
- **Что делает «установить размер страницы PDF»?** Он определяет ширину и высоту результирующей страницы PDF во время рендеринга CAD.  
- **Зачем включать отслеживание?** Отслеживание записывает каждый этап конвертации, помогая выявлять узкие места в производительности или ошибки.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для оценки; для продакшн требуется коммерческая лицензия.  
- **Какие форматы CAD поддерживаются?** DWG, DXF, DGN и многие другие — см. документацию Aspose.CAD для полного списка.  
- **Можно ли менять размеры страницы «на лету»?** Да — просто измените значения `PageWidth` и `PageHeight` в `CadRasterizationOptions`.

## Что такое «установить размер страницы PDF» при рендеринге CAD?

Установка размера страницы PDF указывает растеризатору, насколько большим должен быть холст, когда векторные данные CAD преобразуются в страницу PDF. Это критически важно для сохранения визуальной точности, особенно при работе с детальными инженерными чертежами.

## Зачем включать отслеживание при рендеринге CAD?

Включение отслеживания предоставляет подробный журнал каждого шага — от загрузки исходного файла до записи PDF‑вывода. Это помогает вам:
- Диагностировать, почему конкретный чертеж может отображаться некорректно.  
- Измерять время, затраченное на каждый этап, что полезно для настройки производительности.  
- Проверять, что настроенный размер страницы действительно применяется.

## Предварительные требования

Прежде чем приступить к настройке отслеживания, убедитесь, что у вас есть следующие требования:

1. **Среда разработки Java** — установлен Java 8 или новее.  
2. **Библиотека Aspose.CAD** — скачайте и интегрируйте библиотеку Aspose.CAD в ваш Java‑проект. Ссылка для скачивания доступна [здесь](https://releases.aspose.com/cad/java/).  
3. **Каталог документов** — подготовьте каталог для хранения ваших файлов CAD и сгенерированных PDF.

## Импорт пространств имён

В вашем Java‑проекте импортируйте необходимые пространства имён, чтобы использовать возможности Aspose.CAD. Добавьте следующие строки в начало вашего кода:

```java
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.OutputStream;

import com.aspose.cad.Image;

import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PdfOptions;
```

## Установка пути к каталогу ресурсов

Замените `"Your Document Directory"` фактическим путем к вашему каталогу документов.

```java
String dataDir = "Your Document Directory" + "CADConversion/";
```

## Загрузка файла CAD

Укажите путь к вашему файлу CAD, убедившись, что он находится в указанном каталоге документов.

```java
String srcFile = dataDir + "conic_pyramid.dxf";
Image image = Image.load(srcFile);
```

## Установка параметров вывода PDF

Создайте поток вывода и задайте параметры PDF для конвертации.

```java
OutputStream stream = new FileOutputStream(dataDir + "conic_pyramid.pdf");
PdfOptions pdfOptions = new PdfOptions();
```

## Настройка CadRasterizationOptions (Установка размера страницы PDF)

Здесь мы **устанавливаем размер страницы PDF**, задавая `PageWidth` и `PageHeight`. Отрегулируйте эти значения в соответствии с требуемыми размерами вашего инженерного чертежа. Это основной шаг, отвечающий на вопрос **как установить размер PDF**, когда вы **java cad to pdf**.

```java
CadRasterizationOptions cadRasterizationOptions = new CadRasterizationOptions();
pdfOptions.setVectorRasterizationOptions(cadRasterizationOptions);
cadRasterizationOptions.setPageWidth(800);
cadRasterizationOptions.setPageHeight(600);
```

## Сохранение файла PDF

Сохраните отрендеренный PDF‑файл с указанными параметрами.

```java
image.save(stream, pdfOptions);
```

## Проверка включения отслеживания

Подтвердите, что отслеживание успешно включено для процесса рендеринга CAD.

```java
System.out.println("Tracking enabled successfully for CAD rendering process.");
```

## Распространённые проблемы и их устранение

| Симптом | Возможная причина | Решение |
|---------|-------------------|--------|
| Страница PDF отображается пустой | `PageWidth`/`PageHeight` установлены в 0 | Убедитесь, что указаны ненулевые размеры. |
| Выходной файл повреждён | Поток вывода не закрыт | Вызовите `stream.close()` после `image.save(...)`. |
| Отсутствуют слои в PDF | Файл CAD содержит неподдерживаемые сущности | Проверьте, что формат файла полностью поддерживается Aspose.CAD. |
