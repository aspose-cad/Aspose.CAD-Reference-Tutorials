# Включение отслеживания процесса рендеринга CAD

## Введение

В этом руководстве вы узнаете, как **установить размер страницы PDF** при **конвертации CAD в PDF** с помощью **Aspose.CAD for Java**. Включив отслеживание, вы получаете полную видимость конвейера рендеринга, что упрощает отладку и оптимизацию преобразования файлов CAD (например, DXF) в PDF. Независимо от того, нужно ли вам **сохранить CAD как PDF**, сгенерировать PDF из DXF или просто контролировать размеры вывода, нижеописанные шаги проведут вас через весь процесс.

## Быстрые ответы
- **Что делает «set PDF page size»?** Он определяет ширину и высоту результирующей страницы PDF во время рендеринга CAD.  
- **Зачем включать отслеживание?** Отслеживание записывает каждый этап конвертации, помогая выявлять узкие места производительности или ошибки.  
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для оценки; для продакшн требуется коммерческая лицензия.  
- **Какие форматы CAD поддерживаются?** DWG, DXF, DGN и многие другие — см. документацию Aspose.CAD для полного списка.  
- **Можно ли менять размеры страницы «на лету»?** Да — просто измените значения `PageWidth` и `PageHeight` в `CadRasterizationOptions`.

## Что такое «set PDF page size» в рендеринге CAD?

Установка размера страницы PDF сообщает растеризатору, какого размера должен быть холст, когда векторные данные CAD преобразуются в страницу PDF. Это критически важно для сохранения визуальной точности, особенно при работе с детализированными инженерными чертежами.

## Зачем включать отслеживание для рендеринга CAD?

Включение отслеживания предоставляет подробный журнал каждого шага — от загрузки исходного файла до записи PDF‑вывода. Это помогает вам:

- Диагностировать, почему конкретный чертеж может отображаться некорректно.  
- Измерять время, затраченное на каждый этап, полезно для настройки производительности.  
- Проверить, что настроенный размер страницы действительно применён.

## Предварительные требования

Перед тем как приступить к настройке отслеживания, убедитесь, что у вас есть следующие требования:

1. **Среда разработки Java** — Java 8 или новее, установленная на вашем компьютере.  
2. **Библиотека Aspose.CAD** — Скачайте и интегрируйте библиотеку Aspose.CAD в ваш Java‑проект. Ссылка для загрузки находится [здесь](https://releases.aspose.com/cad/java/).  
3. **Каталог документов** — Подготовьте каталог для хранения ваших CAD‑файлов и сгенерированных PDF.

## Импорт пространств имён

В вашем Java‑проекте импортируйте необходимые пространства имён, чтобы воспользоваться функциональностью Aspose.CAD. Добавьте следующие строки в начале вашего кода:

```java
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.OutputStream;

import com.aspose.cad.Image;

import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PdfOptions;
```

## Установка пути к каталогу ресурсов

```java
String dataDir = "Your Document Directory" + "CADConversion/";
```

Замените `"Your Document Directory"` фактическим путём к вашему каталогу документов.

## Загрузка CAD‑файла

```java
String srcFile = dataDir + "conic_pyramid.dxf";
Image image = Image.load(srcFile);
```

Укажите путь к вашему CAD‑файлу, убедившись, что он находится в указанном каталоге документов.

## Установка параметров вывода PDF

```java
OutputStream stream = new FileOutputStream(dataDir + "conic_pyramid.pdf");
PdfOptions pdfOptions = new PdfOptions();
```

## Настройка CadRasterizationOptions (Установка размера страницы PDF)

```java
CadRasterizationOptions cadRasterizationOptions = new CadRasterizationOptions();
pdfOptions.setVectorRasterizationOptions(cadRasterizationOptions);
cadRasterizationOptions.setPageWidth(800);
cadRasterizationOptions.setPageHeight(600);
```

Здесь мы **устанавливаем размер страницы PDF**, задавая `PageWidth` и `PageHeight`. Отрегулируйте эти значения в соответствии с требуемыми размерами вашего инженерного чертежа. Этот шаг напрямую влияет на масштабирование и рендеринг содержимого CAD в конечном PDF.

## Сохранение PDF‑файла

```java
image.save(stream, pdfOptions);
```

Сохраните отрендеренный PDF‑файл с указанными параметрами.

## Проверка включения отслеживания

```java
System.out.println("Tracking enabled successfully for CAD rendering process.");
```

Убедитесь, что отслеживание успешно включено для процесса рендеринга CAD.

## Распространённые проблемы и их устранение

| Симптом | Вероятная причина | Решение |
|---------|-------------------|---------|
| Страница PDF отображается пустой | `PageWidth`/`PageHeight` установлены в 0 | Убедитесь, что указаны ненулевые размеры. |
| Файл вывода повреждён | Поток вывода не закрыт | Вызовите `stream.close()` после `image.save(...)`. |
| Отсутствуют слои в PDF | CAD‑файл содержит неподдерживаемые сущности | Убедитесь, что формат файла полностью поддерживается Aspose.CAD. |
