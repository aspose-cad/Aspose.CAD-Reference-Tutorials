- **Можно ли экспортировать только один макет из DWG?** Конечно — выберите макет по имени или индексу перед сохранением.  
- **Нужна ли лицензия для использования в продакшене?** Для продакшена требуется коммерческая лицензия; бесплатная пробная версия доступна для оценки.  
- **Какие вторичные форматы поддерживаются?** Помимо PDF, вы можете экспортировать в растровые форматы, DXF и другие совместимые с CAD типы.

## Часто задаваемые вопросы

**Q: Как экспортировать DWG‑файл в PDF без потери информации о слоях?**  
A: Use `PdfOptions` and set `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` to retain vector data, then call `image.save("output.pdf", options);`.

**Q: Могу ли я пакетно обрабатывать несколько DWG‑файлов в PDF за один запуск?**  
A: Yes—loop through your file collection, load each with `Image.load`, and save using the same PDF options.

**Q: Есть ли способ напрямую конвертировать шаблон DWT в DXF?**  
A: Absolutely. Load the DWT with `Image.load` and save it as DXF using `image.save("output.dxf");`.

**Q: Какие версии Java поддерживает Aspose.CAD?**  
A: The library works with Java 8 and newer, including Java 11, 17, and later LTS releases.

**Q: Нужна ли отдельная лицензия для экспорта в PDF?**  
A: No, the Aspose.CAD license covers all output formats, including PDF, raster images, and DXF.
