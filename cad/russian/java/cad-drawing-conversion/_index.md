## Часто задаваемые вопросы

**В: Как экспортировать DWG‑файл в PDF без потери информации о слоях?**  
О: Используйте `PdfOptions` и установите `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)`, чтобы сохранить векторные данные, затем вызовите `image.save("output.pdf", options);`.

**В: Можно ли пакетно обрабатывать несколько DWG‑файлов в PDF за один запуск?**  
О: Да — пройдитесь циклом по коллекции файлов, загрузите каждый с помощью `Image.load` и сохраните, используя те же PDF‑опции.

**В: Есть ли способ напрямую конвертировать шаблон DWT в DXF?**  
О: Абсолютно. Загрузите DWT через `Image.load` и сохраните его как DXF, вызвав `image.save("output.dxf");`.

**В: Какие версии Java поддерживает Aspose.CAD?**  
О: Библиотека работает с Java 8 и новее, включая Java 11, 17 и последующие LTS‑выпуски.

**В: Нужна ли отдельная лицензия для экспорта в PDF?**  
О: Нет, лицензия Aspose.CAD покрывает все форматы вывода, включая PDF, растровые изображения и DXF.

**В: Как улучшить скорость рендеринга при конвертации больших DWG‑файлов?**  
О: Включите многопоточность, обрабатывая каждый макет в отдельном потоке, и переиспользуйте один экземпляр `PdfOptions`, чтобы снизить накладные расходы.

**В: Можно ли задать пользовательский размер страницы для экспортируемого PDF?**  
О: Да, настройте `PdfOptions` с помощью `PdfOptions.setPageSize(PageSize.A4)` (или любого другого размера) перед сохранением.
