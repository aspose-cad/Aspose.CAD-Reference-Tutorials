## Часто задаваемые вопросы

**Q:** *Могу ли я конвертировать большие файлы DWG в PDF/A без исчерпания памяти?*  
A: Да. Используйте класс `CadImage` с параметрами потоковой передачи и своевременно освобождайте изображения, чтобы снизить потребление памяти.

**Q:** *Можно ли экспортировать только конкретный макет из файла CAD в SVG?*  
A: Абсолютно. Загрузите нужный макет по имени и вызовите метод `Save` с параметрами SVG.

**Q:** *Нужна ли отдельная лицензия для каждого формата вывода?*  
A: Нет. Одна лицензия Aspose.CAD for Java покрывает все поддерживаемые форматы экспорта.

**Q:** *Могу ли я пакетно обрабатывать несколько файлов CAD за один запуск?*  
A: Да. Проходите по коллекции файлов, загружая каждый с помощью `CadImage.load` и сохраняйте в целевой формат.

**Q:** *Что если мой файл CAD содержит растровые изображения — сохраняются ли они в PDF?*  
A: Растровые изображения автоматически встраиваются при экспорте в PDF, сохраняя визуальную точность.
