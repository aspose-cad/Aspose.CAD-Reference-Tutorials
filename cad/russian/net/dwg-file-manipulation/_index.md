## Часто задаваемые вопросы

**В: Как открыть файл DWFX в C#?**  
О: Используйте `Aspose.CAD.Image.Load("file.dwfx")` для загрузки файла, затем при необходимости получайте доступ к его страницам или сущностям.

**В: Можно ли прочитать файл DWG с помощью C# без предварительного конвертирования?**  
О: Да — Aspose.CAD может напрямую загружать файлы DWG, позволяя запрашивать слои, сущности и метаданные.

**В: Какой самый простой способ добавить текст в чертеж DWG?**  
О: Создайте сущность `CadText`, задайте её `Value` и позицию, затем добавьте её в коллекцию model space чертежа.

**В: Существуют ли ограничения по размеру при работе с большими файлами DWFX или DWG?**  
О: Библиотека эффективно обрабатывает большие файлы, однако следует обеспечить достаточный объём памяти и рассмотреть возможность потоковой обработки больших растровых изображений.

**В: Нужна ли отдельная лицензия для каждого формата CAD?**  
О: Нет — одна лицензия Aspose.CAD покрывает все поддерживаемые форматы, включая DWG, DWFX, DWF, DXF и другие.
