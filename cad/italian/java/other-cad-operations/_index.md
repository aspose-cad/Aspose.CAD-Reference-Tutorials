## Domande frequenti

**D: Posso aggiungere una filigrana a un file DWG già protetto da password?**  
R: Sì. Carica il DWG con la password appropriata, applica la filigrana usando l’API e poi salva il file.

**D: L’aggiunta di una filigrana influisce significativamente sulla dimensione del file?**  
R: L’aumento è minimo perché la filigrana è memorizzata come un overlay leggero.

**D: È possibile elaborare in batch più file CAD per aggiungere la stessa filigrana?**  
R: Assolutamente. Scorri la tua collezione di file ed esegui la stessa routine di filigrana per ogni disegno.

**D: Posso controllare l’opacità e la rotazione della filigrana?**  
R: L’API fornisce proprietà per impostare opacità, angolo di rotazione e posizionamento, offrendoti pieno controllo visivo.

**D: Quali versioni di Java sono supportate per questi tutorial?**  
R: Java 8, 11 e le versioni LTS più recenti sono pienamente supportate.
