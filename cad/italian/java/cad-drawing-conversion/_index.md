## Domande frequenti

**D: Come posso esportare un file DWG in PDF senza perdere le informazioni sui livelli?**  
R: Usa `PdfOptions` e imposta `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` per mantenere i dati vettoriali, quindi chiama `image.save("output.pdf", options);`.

**D: Posso elaborare in batch più file DWG in PDF in un'unica esecuzione?**  
R: Sì—itera sulla tua collezione di file, carica ciascuno con `Image.load` e salva usando le stesse opzioni PDF.

**D: Esiste un modo per convertire direttamente un modello DWT in DXF?**  
R: Assolutamente. Carica il DWT con `Image.load` e salvalo come DXF usando `image.save("output.dxf");`.

**D: Quali versioni di Java sono supportate da Aspose.CAD?**  
R: La libreria funziona con Java 8 e versioni successive, incluse Java 11, 17 e le successive release LTS.

**D: È necessaria una licenza separata per l'esportazione PDF?**  
R: No, la licenza di Aspose.CAD copre tutti i formati di output, inclusi PDF, immagini raster e DXF.

**D: Come posso migliorare la velocità di rendering quando converto file DWG di grandi dimensioni?**  
R: Abilita il multi‑threading elaborando ogni layout in un thread separato e riutilizza un'unica istanza di `PdfOptions` per ridurre l'overhead.

**D: Posso impostare una dimensione pagina personalizzata per il PDF esportato?**  
R: Sì, configura `PdfOptions` con `PdfOptions.setPageSize(PageSize.A4)` (o qualsiasi altra dimensione) prima di salvare.
