## Domande frequenti

**Q:** *Posso convertire file DWG di grandi dimensioni in PDF/A senza esaurire la memoria?*  
A: Sì. Usa la classe `CadImage` con opzioni di streaming e rilascia le immagini prontamente per mantenere basso l'uso della memoria.

**Q:** *È possibile esportare solo un layout specifico da un file CAD in SVG?*  
A: Assolutamente. Carica il layout desiderato per nome e chiama il metodo `Save` con le opzioni SVG.

**Q:** *Ho bisogno di una licenza separata per ogni formato di output?*  
A: No. Una singola licenza Aspose.CAD for Java copre tutti i formati di esportazione supportati.

**Q:** *Posso elaborare in batch più file CAD in un'unica esecuzione?*  
A: Sì. Itera su una collezione di file, caricando ciascuno con `CadImage.load` e salvando nel formato di destinazione.

**Q:** *Cosa succede se il mio file CAD contiene immagini raster—vengono preservate nel PDF?*  
A: Le immagini raster vengono incorporate automaticamente quando esporti in PDF, mantenendo la fedeltà visiva.
