# Πώς να ορίσετε χρονικό όριο στην αποθήκευση - Aspose.CAD Οδηγός

## Εισαγωγή

Σε αυτόν τον οδηγό, θα μάθετε **πώς να ορίσετε χρονικό όριο** σε λειτουργίες αποθήκευσης CAD, μια τεχνική που σας βοηθά να αποτρέψετε τις μακροχρόνιες μετατροπές από το να γίνουν μη ανταποκρινόμενες διεργασίες. Η διαχείριση του χρονικού ορίου είναι ιδιαίτερα χρήσιμη όταν **μετατρέπετε DWG σε PDF** ή **εξάγετε CAD PDF** αρχεία σε παρτίδες ή υπηρεσίες web. Το Aspose.CAD για .NET παρέχει ένα καθαρό API που σας επιτρέπει να ελέγχετε τη λειτουργία αποθήκευσης ενώ εξακολουθείτε να εκμεταλλεύεστε τη δυνατότητα rasterization του.

## Γρήγορες Απαντήσεις
- **Τι ελέγχει το χρονικό όριο;** Διακόπτει τη λειτουργία αποθήκευσης/εξαγωγής εάν εκτελείται περισσότερο από την καθορισμένη περίοδο.  
- **Ποιοι τύποι αρχείων ωφελούνται περισσότερο;** Η μετατροπή μεγάλων αρχείων DWG σε PDF ή εικόνες raster όπου ο χρόνος επεξεργασίας μπορεί να διαφέρει.  
- **Χρειάζομαι άδεια;** Απαιτείται έγκυρη άδεια Aspose.CAD για χρήση σε παραγωγή· μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση.  
- **Μπορώ να προσαρμόσω τη διάρκεια;** Ναι—απλώς αλλάξτε την τιμή του `Thread.Sleep` (σε χιλιοστά του δευτερολέπτου) ώστε να ταιριάζει στο σενάριό σας.  
- **Είναι αυτή η προσέγγιση φιλική προς το async;** Το παράδειγμα χρησιμοποιεί `Task.Factory.StartNew`, καθιστώντας εύκολη την ενσωμάτωση σε async ροές εργασίας.

## Τι σημαίνει “πώς να ορίσετε χρονικό όριο” στο πλαίσιο της αποθήκευσης CAD;
Η ρύθμιση ενός χρονικού ορίου σημαίνει την προσάρτηση ενός `InterruptionToken` στις επιλογές αποθήκευσης και την ενεργοποίηση μιας διακοπής μετά από ένα προκαθορισμένο διάστημα. Αυτό αποτρέπει τη διεργασία από το να κολλάει επ' άπειρο, προσφέροντας προβλέψιμη απόδοση και καλύτερη διαχείριση πόρων.

## Γιατί να χρησιμοποιήσετε Aspose.CAD για μετατροπή DWG σε PDF με χρονικό όριο;
- **Αξιοπιστία:** Εγγυάται ότι μια ασταθής μετατροπή δεν θα μπλοκάρει την υπηρεσία σας.  
- **Κλιμακωσιμότητα:** Σας επιτρέπει να επεξεργάζεστε πολλά αρχεία ταυτόχρονα χωρίς τον κίνδυνο ένα αρχείο να καθυστερήσει τη διαδικασία.  
- **Ποιότητα:** Διατηρεί τη υψηλής πιστότητας rasterization του Aspose.CAD ενώ προσθέτει ελέγχους ασφαλείας.

## Προαπαιτούμενα

Πριν προχωρήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

- **Aspose.CAD for .NET** – ενσωματωμένο στο έργο σας. Μπορείτε να το κατεβάσετε [εδώ](https://releases.aspose.com/cad/net/).  
- **Document Directory** – ένας φάκελος όπου θα βρίσκονται τα πηγαία αρχεία DWG και τα παραγόμενα PDF.

## Εισαγωγή Namespaces

Πρώτα, εισάγετε τα namespaces που παρέχουν τις κλάσεις που θα χρειαστούμε για rasterization, επιλογές PDF και τον μηχανισμό διακοπής.

```csharp
using Aspose.CAD.ImageOptions;
using System;
using System.Threading;
using System.Threading.Tasks;
```

## Πώς να ορίσετε χρονικό όριο στην αποθήκευση

Παρακάτω ακολουθεί ένας οδηγός βήμα‑βήμα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση, ακολουθούμενη από το αρχικό τμήμα κώδικα (αμετάβλητο).

### Βήμα 1: Φόρτωση Σχεδίου CAD

Φορτώνουμε το πηγαίο αρχείο DWG από τον καθορισμένο φάκελο. Η μέθοδος `Image.Load` επιστρέφει ένα αντικείμενο `Image` που αντιπροσωπεύει το σχέδιο CAD.

```csharp
// Example: Loading CAD Drawing
string SourceDir = "Your Document Directory";
string OutputDir = "Your Document Directory";

using (Image cadDrawing = Image.Load(SourceDir + "Drawing11.dwg"))
{
    // Code for subsequent steps will be placed here
}
```

### Βήμα 2: Διαμόρφωση Ρυθμίσεων Rasterization

Ορίστε τις ρυθμίσεις rasterization ώστε το εξαγόμενο PDF να ταιριάζει με τις αρχικές διαστάσεις του σχεδίου. Εδώ ελέγχετε το πλάτος, το ύψος και άλλες ρυθμίσεις απόδοσης.

```csharp
// Example: Configuring Rasterization Options
var rasterizationOptions = new CadRasterizationOptions();
rasterizationOptions.PageWidth = cadDrawing.Size.Width;
rasterizationOptions.PageHeight = cadDrawing.Size.Height;
```

### Βήμα 3: Δημιουργία PDF Options (Εξαγωγή CAD PDF)

Δημιουργήστε μια παρουσία `PdfOptions` και συνδέστε τις ρυθμίσεις rasterization. Αυτό το αντικείμενο θα περάσει στη μέθοδο `Save` για τη δημιουργία μιας έκδοσης PDF του αρχείου DWG.

```csharp
// Example: Creating PDF Options
PdfOptions CADf = new PdfOptions();
CADf.VectorRasterizationOptions = rasterizationOptions;
```

### Βήμα 4: Υλοποίηση Μηχανισμού Χρονικού Ορίου

Χρησιμοποιούμε `InterruptionTokenSource` για να αποκτήσουμε ένα token που μπορεί να διακόψει τη λειτουργία αποθήκευσης. Μια εργασία στο παρασκήνιο εκτελεί την εξαγωγή, ενώ το κύριο νήμα κοιμάται για τη ζητούμενη διάρκεια (π.χ., 10 δευτερόλεπτα). Μετά τον ύπνο, καλούμε `Interrupt()` για να ακυρώσουμε τη λειτουργία εάν εξακολουθεί να εκτελείται.

```csharp
// Example: Implementing Timeout Mechanism
using (var its = new InterruptionTokenSource())
{
    CADf.InterruptionToken = its.Token;

    var exportTask = Task.Factory.StartNew(() =>
    {
        cadDrawing.Save(OutputDir + "PutTimeoutOnSave_out.pdf", CADf);
    });

    Thread.Sleep(10000); // Set your desired timeout duration in milliseconds
    its.Interrupt();

    exportTask.Wait();
}
```

### Βήμα 5: Ολοκλήρωση και Επιβεβαίωση

Μετά την εξαγωγή (ή τη διακοπή), γράψτε ένα μήνυμα επιβεβαίωσης στην κονσόλα. Σε μια πραγματική εφαρμογή μπορεί να καταγράψετε το αποτέλεσμα ή να εκκινήσετε ένα συμβάν.

```csharp
// Example: Finalizing and Confirming
Console.WriteLine("PutTimeoutOnSave executed successfully");
```

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| Η εξαγωγή κολλάει επ' άπειρο | Δεν έχει προσαρτηθεί token διακοπής | Βεβαιωθείτε ότι `CADf.InterruptionToken = its.Token;` έχει οριστεί πριν την εκκίνηση της εργασίας. |
| Το PDF είναι κενό ή κατεστραμμένο | Λανθασμένη διαδρομή φακέλου εξόδου | Επαληθεύστε ότι `OutputDir` λήγει με διαχωριστικό διαδρομής και ότι το όνομα αρχείου είναι έγκυρο. |
| Το χρονικό όριο είναι πολύ μικρό, προκαλώντας πρόωρη διακοπή | Η τιμή `Thread.Sleep` είναι πολύ χαμηλή για μεγάλα αρχεία | Αυξήστε τη διάρκεια του ύπνου ή υπολογίστε προσαρμοστικό χρονικό όριο βάσει του μεγέθους του αρχείου. |

## Συχνές Ερωτήσεις

**Q1: Μπορώ να προσαρμόσω τη διάρκεια του χρονικού ορίου;**  
A: Απόλυτα. Αλλάξτε την τιμή που περνάτε στο `Thread.Sleep` (σε χιλιοστά του δευτερολέπτου) ώστε να ταιριάζει με τις προσδοκίες απόδοσής σας.

**Q2: Υπάρχουν άλλες επιλογές για rasterization;**  
A: Ναι. Το `CadRasterizationOptions` προσφέρει ιδιότητες όπως `Resolution`, `BackgroundColor` και `DrawType` για λεπτομερή ρύθμιση της εξόδου PDF.

**Q3: Πώς μπορώ να διαχειριστώ τις διακοπές στην εφαρμογή μου;**  
A: Χρησιμοποιήστε τις κλάσεις `InterruptionToken` και `InterruptionTokenSource` όπως φαίνεται. Παρέχουν έναν thread‑safe τρόπο για την ακύρωση μακροχρόνιων λειτουργιών.

**Q4: Είναι το Aspose.CAD κατάλληλο για αρχεία CAD 2D και 3D;**  
A: Υποστηρίζει μια ευρεία γκάμα μορφών 2D και 3D, συμπεριλαμβανομένων των DWG, DXF, DGN και άλλων.

**Q5: Πού μπορώ να βρω περαιτέρω βοήθεια ή υποστήριξη από την κοινότητα;**  
A: Επισκεφθείτε το [Aspose.CAD forum](https://forum.aspose.com/c/cad/19) για συζητήσεις της κοινότητας, δείγματα κώδικα και συμβουλές αντιμετώπισης προβλημάτων.

Ενσωματώστε αυτό το πρότυπο σε παρτίδες μετατροπής, υπηρεσίες web ή οποιοδήποτε αυτοματοποιημένο pipeline όπου η προβλεψιμότητα είναι κρίσιμη.
