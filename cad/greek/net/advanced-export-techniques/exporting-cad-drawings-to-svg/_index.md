# Εξαγωγή CAD σε SVG – Εξαγωγή Σχεδίων CAD σε Μορφή SVG

## Εισαγωγή

Η εξαγωγή σχεδίων CAD σε SVG είναι μια συχνή απαίτηση όταν χρειάζεστε γραφικά ανεξάρτητα από την ανάλυση για ιστοσελίδες, αναφορές ή διαδραστικές απεικονίσεις. Σε αυτό το σεμινάριο θα μάθετε **πώς να εξάγετε CAD σε SVG** με το Aspose.CAD for .NET, θα εξερευνήσετε επιλογές για **προσαρμογή χρώματος SVG**, και θα δείτε πώς να **μετατρέψετε DWG σε SVG** (ή DXF σε SVG) με λίγες μόνο γραμμές κώδικα. Τα βήματα είναι γρήγορα, το API είναι διαισθητικό, και τα παραγόμενα αρχεία SVG κλιμακώνονται τέλεια σε οποιαδήποτε συσκευή.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται τη μετατροπή;** Aspose.CAD for .NET.  
- **Μπορώ να αλλάξω τη λειτουργία χρώματος;** Ναι – χρησιμοποιήστε `SvgColorMode` για να ορίσετε αποχρώσεις του γκρι, ασπρόμαυρο ή πλήρες χρώμα.  
- **Ποιες μορφές CAD υποστηρίζονται;** DWG, DXF και πολλές άλλες (δείτε την τεκμηρίωση Aspose.CAD).  
- **Χρειάζομαι άδεια για ανάπτυξη;** Διατίθεται προσωρινή άδεια για δοκιμή.  
- **Πόσο διαρκεί η μετατροπή;** Συνήθως κάτω από ένα δευτερόλεπτο για τυπικά σχέδια.

## Τι είναι η «εξαγωγή CAD σε SVG»;
Η εξαγωγή CAD σε SVG σημαίνει τη λήψη ενός εγγενούς αρχείου CAD (όπως DWG ή DXF) και η απόδοσή του στη μορφή Scalable Vector Graphics. Το SVG είναι βασισμένο σε XML, ελαφρύ και μπορεί να μορφοποιηθεί με CSS—κάτι που το καθιστά ιδανικό για σύγχρονες web και mobile εφαρμογές.

## Γιατί να χρησιμοποιήσετε το Aspose.CAD για εξαγωγή CAD σε SVG;
- **Χωρίς εξωτερικές εξαρτήσεις** – καθαρό .NET API, δεν χρειάζονται εγκαταστάσεις AutoCAD.  
- **Ακριβής έλεγχος** – μπορείτε να **προσαρμόσετε το χρώμα SVG**, να αποφασίσετε αν το κείμενο θα διατηρηθεί ως σχήματα, και να επιλέξετε επιλογές rasterization.  
- **Ευρεία υποστήριξη μορφών** – ο ίδιος κώδικας λειτουργεί για **μετατροπή DWG σε SVG**, **μετατροπή DXF σε SVG**, και άλλες μορφές, απλοποιώντας τη βάση κώδικα.  
- **Υψηλή απόδοση** – βελτιστοποιημένο για ταχύτητα και χαμηλή χρήση μνήμης, κατάλληλο για επεξεργασία σε παρτίδες.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Aspose.CAD for .NET** εγκατεστημένο. Μπορείτε να κατεβάσετε τη βιβλιοθήκη [εδώ](https://releases.aspose.com/cad/net/).  
- Περιβάλλον ανάπτυξης .NET (Visual Studio, Rider ή .NET CLI).  
- Αρχείο CAD (DWG ή DXF) που θέλετε να μετατρέψετε.

## Εισαγωγή Ονομάτων Χώρων

Πρώτα, φέρετε τα απαιτούμενα namespaces στο έργο σας ώστε οι κλάσεις να είναι διαθέσιμες.

```csharp
using Aspose.CAD.ImageOptions;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου  
Ορίστε το φάκελο όπου βρίσκεται το αρχικό αρχείο CAD και όπου θα αποθηκευτεί το αρχείο SVG.

```csharp
// The path to the documents directory.
string MyDir = "Your Document Directory";
```

### Βήμα 2: Φόρτωση Σχεδίου CAD  
Χρησιμοποιήστε `Image.Load` για να διαβάσετε το αρχείο DWG (ή DXF). Αυτό λειτουργεί για σενάρια **load DWG .NET**.

```csharp
using (Image image = Image.Load(MyDir + "sample.dwg"))
{
```

### Βήμα 3: Διαμόρφωση Επιλογών Εξαγωγής SVG  
Ρυθμίστε τις επιλογές εξαγωγής ώστε να ταιριάζουν στις ανάγκες σας. Εδώ ορίζουμε τη λειτουργία χρώματος σε αποχρώσεις του γκρι και αναγκάζουμε όλο το κείμενο να αποδοθεί ως διανυσματικά σχήματα.

```csharp
    var options = new SvgOptions();
    options.ColorType = SvgColorMode.Grayscale;   // customize SVG color mode
    options.TextAsShapes = true;                  // ensures text appears as shapes
```

### Βήμα 4: Αποθήκευση Αρχείου SVG  
Τέλος, γράψτε το αρχείο SVG στο δίσκο. Αυτό το βήμα **αποθηκεύει CAD ως SVG** και ολοκληρώνει τη μετατροπή.

```csharp
    image.Save(MyDir + "sample.svg", options);
}
```

Ακολουθώντας αυτά τα τέσσερα σύντομα βήματα, μπορείτε να **μετατρέψετε DWG σε SVG** (ή **μετατρέψετε DXF σε SVG**) με πλήρη έλεγχο της εμφάνισης του αποτελέσματος.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Κενό SVG αποτέλεσμα** | Η διαδρομή του αρχείου προέλευσης είναι λανθασμένη ή το αρχείο είναι κατεστραμμένο. | Επαληθεύστε το `MyDir` και βεβαιωθείτε ότι το αρχείο CAD ανοίγει χωρίς σφάλματα. |
| **Τα χρώματα φαίνονται λανθασμένα** | `SvgColorMode` έχει οριστεί σε αποχρώσεις του γκρι κατά λάθος. | Αλλάξτε το `options.ColorType` σε `SvgColorMode.FullColor` για να διατηρήσετε τα αρχικά χρώματα. |
| **Λείπει το κείμενο** | `TextAsShapes` ορίστηκε σε `false` και ο προβολέας δεν υποστηρίζει ενσωματωμένες γραμματοσειρές. | Διατηρήστε το `TextAsShapes = true` ή ενσωματώστε τις απαιτούμενες γραμματοσειρές στο SVG. |

## Συχνές Ερωτήσεις

**Q: Μπορώ να χρησιμοποιήσω αυτόν τον κώδικα με .NET Core ή .NET 6;**  
A: Απολύτως. Το Aspose.CAD for .NET είναι συμβατό με .NET Framework, .NET Core και .NET 5/6+.

**Q: Είναι δυνατόν να εξάγω μόνο μια συγκεκριμένη διάταξη ή στρώση;**  
A: Ναι. Χρησιμοποιήστε `SvgOptions` για να ορίσετε `PageIndex` ή να φιλτράρετε στρώματα πριν από την αποθήκευση.

**Q: Πώς ενσωματώνω προσαρμοσμένα στυλ CSS στο παραγόμενο SVG;**  
A: Μετά την αποθήκευση, μπορείτε να επεξεργαστείτε το XML του SVG για να εισάγετε στοιχεία `<style>`, ή να χρησιμοποιήσετε `options.CustomCss` εάν είναι διαθέσιμο σε νεότερες εκδόσεις.

**Q: Ποιοι είναι οι περιορισμοί μεγέθους για το αρχείο CAD προέλευσης;**  
A: Η βιβλιοθήκη διαχειρίζεται μεγάλα αρχεία, αλλά η κατανάλωση μνήμης αυξάνεται με την πολυπλοκότητα. Για πολύ μεγάλα σχέδια, σκεφτείτε την επεξεργασία σελίδων ξεχωριστά.

**Q: Διατηρεί η μετατροπή τα βάρη γραμμών και τους τύπους γραμμών;**  
A: Από προεπιλογή, τα βάρη γραμμών διατηρούνται. Μπορείτε να ρυθμίσετε τις επιλογές απόδοσης στο `SvgOptions` για πιο λεπτομερή έλεγχο.
