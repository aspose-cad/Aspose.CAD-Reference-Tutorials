# Πώς να Αντικαταστήσετε τις Γραμματοσειρές στο Aspose.CAD για .NET

## Εισαγωγή

Στον χώρο της ανάπτυξης CAD με .NET, η εκμάθηση **πώς να αντικαταστήσετε τις γραμματοσειρές** είναι μια κρίσιμη δεξιότητα που μπορεί να βελτιώσει δραματικά την οπτική ποιότητα των σχεδίων σας. Το Aspose.CAD για .NET προσφέρει ένα απλό API που σας επιτρέπει να **ορίσετε το primary font name** για οποιοδήποτε στυλ, καθιστώντας την παγκόσμια ή επιλεκτική αντικατάσταση γραμματοσειρών παιχνιδάκι. Σε αυτό το tutorial θα περάσουμε από τη διαδικασία φόρτωσης ενός σχεδίου μέχρι την ανταλλαγή γραμματοσειρών είτε παγκοσμίως είτε με βάση συγκεκριμένο όνομα στυλ.

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για τη διαχείριση CAD;** `Aspose.CAD.Image` (ή η παράγωγό της `CadImage`).
- **Ποια ιδιότητα αλλάζει τη γραμματοσειρά;** `PrimaryFontName` στο `CadStyleTableObject`.
- **Μπορώ να αντικαταστήσω τις γραμματοσειρές για όλα τα στυλ ταυτόχρονα;** Ναι, επαναλάβετε μέσω `cadImage.Styles` και ορίστε την ιδιότητα.
- **Χρειάζομαι άδεια για παραγωγή;** Απαιτείται έγκυρη άδεια Aspose.CAD για μη‑δοκιμαστική χρήση.
- **Υποστηριζόμενες εκδόσεις .NET;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Τι είναι η Αντικατάσταση Γραμματοσειρών στο CAD;
Η αντικατάσταση γραμματοσειρών σημαίνει την αντικατάσταση της αρχικής γραμματοσειράς που χρησιμοποιείται σε ένα σχέδιο CAD με μια άλλη που είναι διαθέσιμη στο σύστημα-στόχο. Αυτό είναι ιδιαίτερα χρήσιμο όταν η αρχική γραμματοσειρά λείπει, όταν χρειάζεστε συνεπές εταιρικό στυλ ή όταν προετοιμάζετε σχέδια για εκτύπωση σε συσκευές που υποστηρίζουν μόνο περιορισμένο σύνολο γραμματοσειρών.

## Γιατί να Αντικαταστήσετε τις Γραμματοσειρές με το Aspose.CAD;
- **Καμία εξωτερική εξάρτηση** – η βιβλιοθήκη διαχειρίζεται την αντιστοίχιση γραμματοσειρών εσωτερικά.
- **Λειτουργεί με πολλές μορφές** – DXF, DWG, DGN, κ.ά.
- **Προγραμματιστικός έλεγχος** – αυτοματοποιήστε τη διαδικασία για μαζικές μετατροπές ή CI pipelines.
- **Διατηρεί τη γεωμετρία του σχεδίου** – μόνο η οπτική αναπαράσταση του κειμένου αλλάζει.

## Προαπαιτούμενα

- Το Aspose.CAD για .NET είναι εγκατεστημένο. Εάν δεν το έχετε εγκαταστήσει ακόμη, κατεβάστε το [εδώ](https://releases.aspose.com/cad/net/).
- Ένα αρχείο σχεδίου CAD (DXF, DWG, κ.λπ.) για πειραματισμό.

## Βήμα 1: Φόρτωση Σχεδίου CAD

Ξεκινήστε φορτώνοντας το σχέδιο CAD σε μια παρουσία του `CadImage`. Βεβαιωθείτε ότι παρέχετε τη σωστή διαδρομή προς το φάκελο του εγγράφου σας.

```csharp
string MyDir = "Your Document Directory";
string sourceFilePath = MyDir + "conic_pyramid.dxf";

using (CadImage cadImage = (CadImage)Aspose.CAD.Image.Load(sourceFilePath))
{
    // Your code for further actions goes here

```

## Βήμα 2: Επανάληψη πάνω από τα Στυλ

Στη συνέχεια, επαναλάβετε τα στυλ στο σχέδιο CAD χρησιμοποιώντας βρόχο `foreach`. Αυτό σας επιτρέπει να έχετε πρόσβαση και να διαχειριστείτε μεμονωμένα στυλ γραμματοσειρών.

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    // Your code for style manipulation goes here
}
```

## Πώς να Ορίσετε το Primary Font Name για την Αντικατάσταση Γραμματοσειρών

Η ιδιότητα `PrimaryFontName` σε κάθε `CadStyleTableObject` ελέγχει ποια γραμματοσειρά χρησιμοποιείται όταν αποδίδεται το σχέδιο. Ορίζοντας αυτή την ιδιότητα αντικαθιστάτε ουσιαστικά την αρχική γραμματοσειρά.

### Βήμα 3: Αντικατάσταση Γραμματοσειρών Καθολικά

Για να αντικαταστήσετε τις γραμματοσειρές για **όλα** τα στυλ, ορίστε την ιδιότητα `PrimaryFontName` για κάθε στυλ στο επιθυμητό όνομα γραμματοσειράς, π.χ., `"Arial"`.

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    style.PrimaryFontName = "Arial";
}
```

### Βήμα 4: Αντικατάσταση Γραμματοσειράς με Όνομα Στυλ

Εάν χρειάζεται να αντικαταστήσετε τη γραμματοσειρά μόνο για ένα συγκεκριμένο στυλ (π.χ., το στυλ με όνομα `"Roman"`), προσθέστε έναν έλεγχο συνθήκης μέσα στον βρόχο.

```csharp
foreach (CadStyleTableObject style in cadImage.Styles)
{
    if (style.StyleName == "Roman")
    {
        style.PrimaryFontName = "Arial";
    }
}
```

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Η γραμματοσειρά δεν αλλάζει μετά την εκτέλεση του κώδικα | Το σχέδιο είναι στην κρυφή μνήμη ή ανοίχτηκε σε λειτουργία μόνο για ανάγνωση | Βεβαιωθείτε ότι αποθηκεύετε την εικόνα μετά την τροποποίηση (`cadImage.Save(...)`) ή επαναφορτώστε το αρχείο για επαλήθευση. |
| Η επιθυμητή γραμματοσειρά δεν βρέθηκε στο σύστημα | Η γραμματοσειρά δεν είναι εγκατεστημένη στον υπολογιστή που εκτελεί τον κώδικα | Εγκαταστήστε την απαιτούμενη γραμματοσειρά TrueType/OpenType ή ενσωματώστε την στους πόρους της εφαρμογής. |
| Το κείμενο εμφανίζεται παραμορφωμένο | Λανθασμένη κωδικοποίηση ή έλλειψη υποστήριξης Unicode | Χρησιμοποιήστε μια γραμματοσειρά που υποστηρίζει το απαιτούμενο σύνολο χαρακτήρων (π.χ., Arial Unicode MS). |
