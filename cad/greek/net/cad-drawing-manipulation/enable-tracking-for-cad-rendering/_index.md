# Δημιουργία PDF από CAD με Παρακολούθηση χρησιμοποιώντας Aspose.CAD για .NET

## Εισαγωγή

Σε σύγχρονες εφαρμογές .NET, η **δημιουργία PDF από αρχεία CAD** είναι συχνή απαίτηση — είτε χρειάζεται να μοιραστείτε σχέδια με μη‑τεχνικούς ενδιαφερόμενους είτε να αρχειοθετήσετε σχέδια σε φορητό φορμάτ. Το Aspose.CAD για .NET κάνει αυτή τη μετατροπή απλή, ενώ προσφέρει επίσης δυνατότητα **παρακολούθησης** που σας επιτρέπει να παρακολουθείτε την πρόοδο της απόδοσης και να καταγράφετε διαγνωστικές πληροφορίες. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα όλη τη διαδικασία: φόρτωση αρχείου DXF, ρύθμιση μεγέθους σελίδας, μετατροπή σε PDF και ενεργοποίηση παρακολούθησης για πλήρη ορατότητα του pipeline απόδοσης.

## Γρήγορες Απαντήσεις
- **Μπορώ να μετατρέψω DXF σε PDF με το Aspose.CAD;** Ναι, η βιβλιοθήκη υποστηρίζει τη μετατροπή DXF‑σε‑PDF έτοιμη προς χρήση.  
- **Πώς ορίζω το μέγεθος σελίδας CAD κατά τη μετατροπή;** Χρησιμοποιήστε `CadRasterizationOptions.PageWidth` και `PageHeight`.  
- **Είναι η παρακολούθηση υποχρεωτική;** Όχι, αλλά παρέχει πολύτιμες διαγνωστικές πληροφορίες για μεγάλα ή σύνθετα σχέδια.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Χρειάζομαι άδεια για παραγωγική χρήση;** Απαιτείται εμπορική άδεια· διατίθεται δωρεάν δοκιμαστική έκδοση.

## Τι σημαίνει «δημιουργία PDF από CAD»;
Η δημιουργία PDF από CAD σημαίνει απόδοση ενός σχεδίου CAD (π.χ. DXF, DWG) σε έγγραφο PDF raster ή vector. Η διαδικασία διατηρεί την οπτική πιστότητα ενώ παρέχει ένα φορμάτ που μπορεί να ανοιχθεί σε σχεδόν οποιαδήποτε συσκευή χωρίς εξειδικευμένο λογισμικό CAD.

## Γιατί να ενεργοποιήσετε την παρακολούθηση για την απόδοση CAD;
Η παρακολούθηση σας δίνει πραγματικό‑χρόνο εικόνα για τη μηχανή απόδοσης — χρήσιμη για εντοπισμό σφαλμάτων, βελτιστοποίηση απόδοσης και έλεγχο. Όταν ενεργοποιείτε την παρακολούθηση, το Aspose.CAD καταγράφει κάθε βήμα απόδοσης, βοηθώντας σας να εντοπίσετε σημεία συμφόρησης ή σφάλματα σε μεγάλα έργα.

## Προαπαιτούμενα

- **Aspose.CAD για .NET** – κατεβάστε το από [εδώ](https://releases.aspose.com/cad/net/).  
- **Περιβάλλον ανάπτυξης** – Visual Studio (οποιαδήποτε πρόσφατη έκδοση) με υποστήριξη .NET.  
- **Δείγμα αρχείου CAD** – ένα αρχείο DXF όπως το `conic_pyramid.dxf` που θα μετατρέψετε και θα παρακολουθήσετε.

using System;

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου (set page size CAD)

```csharp
string MyDir = "Your Document Directory";
string sourceFilePath = MyDir + "conic_pyramid.dxf";
```

Αντικαταστήστε **Your Document Directory** με την απόλυτη διαδρομή όπου βρίσκεται το αρχείο DXF. Εδώ μπορείτε επίσης να προσαρμόσετε τις διαστάσεις σελίδας μέσω των επιλογών rasterization.

### Βήμα 2: Φόρτωση του Αρχείου CAD

```csharp
using (Image image = Image.Load(sourceFilePath))
{
    // Further steps will be implemented here
}
```

Η μέθοδος `Image.Load` διαβάζει το σχέδιο CAD σε ένα αντικείμενο `Aspose.CAD.Image`, προετοιμάζοντάς το για απόδοση.

### Βήμα 3: Δημιουργία PDF Options (prepare for convert dxf to pdf)

```csharp
MemoryStream stream = new MemoryStream();
PdfOptions pdfOptions = new PdfOptions();
```

Ένα `MemoryStream` σας επιτρέπει να κρατήσετε το παραγόμενο PDF στη μνήμη, ενώ το `PdfOptions` περιέχει όλες τις ρυθμίσεις ειδικές για PDF.

### Βήμα 4: Ρύθμιση Rasterization Options (set page size CAD)

```csharp
CadRasterizationOptions cadRasterizationOptions = new CadRasterizationOptions();
pdfOptions.VectorRasterizationOptions = cadRasterizationOptions;
cadRasterizationOptions.PageWidth = 800;
cadRasterizationOptions.PageHeight = 600;
```

Εδώ ορίζετε το μέγεθος εξόδου σελίδας (`PageWidth` & `PageHeight`). Προσαρμόστε αυτές τις τιμές ώστε να ταιριάζουν στις επιθυμητές διαστάσεις PDF — αυτό αποτελεί τον πυρήνα της απαίτησης **set page size CAD**.

### Βήμα 5: Αποθήκευση της Αποδοθείσας Εικόνας (complete the create pdf from cad workflow)

```csharp
image.Save(stream, pdfOptions);
```

Η κλήση `Save` γράφει το αποδοθέν περιεχόμενο στο memory stream χρησιμοποιώντας τις PDF options που διαμορφώσατε. Σε αυτό το σημείο έχετε μια PDF αναπαράσταση του αρχικού αρχείου CAD, και οι πληροφορίες παρακολούθησης έχουν καταγραφεί αυτόματα από τη βιβλιοθήκη.

## Συνηθισμένα Προβλήματα & Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **Κενό PDF εξόδου** | Οι επιλογές rasterization δεν συνδέθηκαν με το `PdfOptions` | Βεβαιωθείτε ότι έχετε ορίσει `pdfOptions.VectorRasterizationOptions = cadRasterizationOptions;`. |
| **Λάθος μέγεθος σελίδας** | Τα `PageWidth`/`PageHeight` παρέμειναν στα προεπιλεγμένα | Ορίστε ρητά και τις δύο ιδιότητες πριν την αποθήκευση. |
| **Μείωση απόδοσης σε μεγάλα DXF** | Τα αρχεία καταγραφής παρακολούθησης μπορούν να είναι εκτενείς | Απενεργοποιήστε ή περιορίστε την παρακολούθηση στην παραγωγή ρυθμίζοντας τις ρυθμίσεις `Aspose.CAD.Logging`. |
