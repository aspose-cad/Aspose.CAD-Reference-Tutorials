## Πώς να εξάγετε DXF σε PDF χρησιμοποιώντας το Aspose.CAD;

Φορτώστε το DXF, διαμορφώστε τη ραστεροποίηση και αποθηκεύστε ως PDF—όλα σε λίγες απλές γραμμές. Πρώτα, δημιουργήστε το αντικείμενο `Image` με το αρχείο DXF, στη συνέχεια ορίστε τις `CadRasterizationOptions` (χρώμα φόντου, μέγεθος σελίδας, DPI), τυλίξτε αυτές τις επιλογές σε ένα αντικείμενο `PdfOptions` και τέλος καλέστε το `Save`. Αυτό το μοτίβο λειτουργεί για οποιαδήποτε υποστηριζόμενη μορφή CAD και σας δίνει πλήρη έλεγχο της ποιότητας εξόδου.

`Image` αντιπροσωπεύει ένα σχέδιο CAD που φορτώνεται στη μνήμη.  
`CadRasterizationOptions` καθορίζει τις ρυθμίσεις ραστεροποίησης όπως το χρώμα φόντου και τις διαστάσεις της σελίδας.  
`PdfOptions` περιέχει ρυθμίσεις εξόδου ειδικές για PDF, συμπεριλαμβανομένων των επιλογών ραστεροποίησης.  
`Save` γράφει την εικόνα στην επιλεγμένη μορφή και διαδρομή αρχείου.

Τα παρακάτω namespaces σας δίνουν πρόσβαση στις βασικές κλάσεις μετατροπής.

```csharp
using System;
```

### Βήμα 1: Φόρτωση του αρχείου DXF

`Image` είναι η κύρια κλάση του Aspose.CAD που αντιπροσωπεύει ένα σχέδιο CAD στη μνήμη. Η φόρτωση του αρχείου το προετοιμάζει για περαιτέρω επεξεργασία.

```csharp
string MyDir = "Your Document Directory";
string sourceFilePath = MyDir + "conic_pyramid.dxf";

using (Image image = Image.Load(sourceFilePath))
{
    // Your code for subsequent steps will go here
}
```

### Βήμα 2: Ορισμός Ρυθμίσεων Ραστεροποίησης

`CadRasterizationOptions` ορίζει πώς τα διανυσματικά δεδομένα ραστεροποιούνται στο PDF. Μπορείτε να ελέγξετε το χρώμα φόντου, τις διαστάσεις της σελίδας και το DPI για να ισορροπήσετε την ποιότητα και το μέγεθος του αρχείου.

```csharp
CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
rasterizationOptions.BackgroundColor = Color.White;
rasterizationOptions.PageWidth = 1600;
rasterizationOptions.PageHeight = 1600;
```

### Βήμα 3: Δημιουργία Ρυθμίσεων PDF

`PdfOptions` περιέχει τις ρυθμίσεις ραστεροποίησης και λέει στο Aspose.CAD να δημιουργήσει ένα έγγραφο PDF. Αναθέστε τις προηγουμένως δημιουργημένες `CadRasterizationOptions` στην ιδιότητα `VectorRasterizationOptions` του.

```csharp
PdfOptions pdfOptions = new PdfOptions();
pdfOptions.VectorRasterizationOptions = rasterizationOptions;
```

### Βήμα 4: Καθορισμός Διαδρομής Εξόδου

Επιλέξτε έναν φάκελο με δικαιώματα εγγραφής και όνομα αρχείου για το παραγόμενο PDF. Το Aspose.CAD θα δημιουργήσει το αρχείο αν δεν υπάρχει ήδη.

```csharp
MyDir = MyDir + "conic_pyramid_out.pdf";
```

### Βήμα 5: Εξαγωγή DXF σε PDF

Καλώντας το `Save` στο αντικείμενο `Image` με την παρουσία `PdfOptions` εκτελεί τη μετατροπή. Η μέθοδος διαχειρίζεται αυτόματα τη γεωμετρία, τα βάρη των γραμμών και τα χρώματα, παρέχοντας μια ακριβή αναπαράσταση PDF του αρχικού σχεδίου CAD.

```csharp
image.Save(MyDir, pdfOptions);
```

## Συχνά Προβλήματα και Λύσεις

- **Κενό PDF αποτέλεσμα** – Βεβαιωθείτε ότι το `BackgroundColor` έχει οριστεί (π.χ., `Color.White`) και ότι το `PageWidth`/`PageHeight` ταιριάζει με τις διαστάσεις του αρχικού σχεδίου.
- **Σφάλματα μνήμης με τεράστια αρχεία** – Αυξήστε την ιδιότητα `MemoryLimit` στο `CadRasterizationOptions` ή επεξεργαστείτε το αρχείο σε τμήματα αν ξεπεράσετε τα 2 GB.
- **Λανθασμένη κλιμάκωση** – Προσαρμόστε το `PageWidth` και το `PageHeight` ή ορίστε το `LayoutOptions` σε `FitToPage` για να διατηρήσετε την αναλογία διαστάσεων.
