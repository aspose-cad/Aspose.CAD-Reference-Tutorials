# Αποθήκευση CAD ως PNG και Εξαγωγή Αντικειμένων OLE με Aspose.CAD για Java

## Εισαγωγή

Στον ταχύτατα εξελισσόμενο κόσμο του Computer‑Aided Design (CAD), η δυνατότητα **αποθήκευσης CAD ως PNG** ενώ εξάγονται ενσωματωμένα αντικείμενα OLE (Object Linking and Embedding) μπορεί να βελτιώσει δραστικά τη ροή εργασίας σας. Είτε δημιουργείτε ένα εργαλείο μαζικής μετατροπής, παράγετε μικρογραφίες για μια διαδικτυακή πύλη, είτε απλώς χρειάζεστε να αρχειοθετήσετε περιεχόμενο OLE, το Aspose.CAD για Java σας προσφέρει έναν καθαρό, προγραμματιζόμενο τρόπο για να το κάνετε. Σε αυτό το tutorial θα περάσουμε από τη ρύθμιση του έργου σας μέχρι την εξαγωγή κάθε αντικειμένου OLE ως εικόνα PNG.

## Γρήγορες Απαντήσεις
- **Μπορώ να εξάγω αντικείμενα OLE απευθείας σε PNG;** Ναι – το Aspose.CAD φορτώνει το αρχείο CAD και σας επιτρέπει να ραστεροποιήσετε κάθε layout σε PNG.  
- **Ποια έκδοση Java απαιτείται;** Java 8 ή νεότερη.  
- **Χρειάζεται άδεια για αυτή τη λειτουργία;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται άδεια για παραγωγική χρήση.  
- **Θα διατηρηθούν τα διανυσματικά δεδομένα;** Η ραστεροποίηση PNG διατηρεί την οπτική πιστότητα, αλλά το αποτέλεσμα είναι ραστερ, όχι διανυσματικό.  
- **Υποστηρίζεται η μαζική επεξεργασία;** Απόλυτα – επαναλάβετε πάνω σε έναν πίνακα αρχείων όπως φαίνεται στο δείγμα κώδικα.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:

- **Java Development Kit (JDK)** – Εγκατεστημένο και ρυθμισμένο Java 8+.  
- **Aspose.CAD για Java** – Κατεβάστε τη νεότερη βιβλιοθήκη από το [download link](https://releases.aspose.com/cad/java/).  
- **Αρχεία πηγής CAD** – Οποιαδήποτε αρχεία DWG/DXF/DGN που περιέχουν αντικείμενα OLE που θέλετε να εξάγετε.

## Εισαγωγή Namespaces

Για να εργαστείτε με αρχεία CAD πρέπει να εισάγετε τις βασικές κλάσεις του Aspose.CAD. Διατηρήστε το μπλοκ εισαγωγών ακριβώς όπως φαίνεται – παρέχει τις κλάσεις που χρησιμοποιούνται αργότερα στο tutorial.

```java
import com.aspose.cad.Image;

import com.aspose.cad.fileformats.cad.CadImage;
import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PngOptions;
```

## Πώς να Αποθηκεύσετε CAD ως PNG

Παρακάτω υπάρχει ένας σύντομος, βήμα‑βήμα οδηγός που δείχνει **πώς να εξάγετε αντικείμενα OLE και να αποθηκεύσετε CAD ως PNG**. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από τον ακριβή κώδικα που πρέπει να αντιγράψετε στο έργο σας.

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων σας

Καθορίστε το φάκελο που περιέχει τα σχέδια CAD. Αντικαταστήστε το placeholder με την πραγματική διαδρομή στο σύστημά σας.

```java
String dataDir = "Your Document Directory" + "DWGDrawings/";
```

### Βήμα 2: Ορίστε Τα Ονόματα Αρχείων CAD

Καταγράψτε κάθε αρχείο CAD που θέλετε να επεξεργαστείτε. Μπορείτε να προσθέσετε όσες καταχωρήσεις χρειάζεστε.

```java
String[] files = new String[] { "D ZD junior D10m H2m.dwg", "ZD - Senior D6m H2m45.dwg" };
```

### Βήμα 3: Ορίστε Επιλογές Εξαγωγής PNG

Διαμορφώστε τις ρυθμίσεις ραστεροποίησης. Εδώ στοχεύουμε σε ένα συγκεκριμένο layout (`Layout1`) και χρησιμοποιούμε τις προεπιλεγμένες επιλογές PNG.

```java
PngOptions pngOptions = new PngOptions();
CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
pngOptions.setVectorRasterizationOptions(rasterizationOptions);
rasterizationOptions.setLayouts(new String[] { "Layout1" });
```

### Βήμα 4: Επανάληψη Στα Αρχεία CAD και Αποθήκευση ως PNG

Φορτώστε κάθε αρχείο CAD, ραστεροποιήστε τα αντικείμενα OLE και γράψτε το αρχείο PNG εξόδου. Το επίθημα `_out.png` σας βοηθά να αναγνωρίσετε τις παραγόμενες εικόνες.

```java
for(String file : files)
{
    CadImage cadImage = (CadImage)Image.load(dataDir + file);
    cadImage.save(dataDir + file + "_out.png", pngOptions);
}
```

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί Συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **`Image.load` throws `FileNotFoundException`** | Η διαδρομή `dataDir` είναι λανθασμένη ή το όνομα αρχείου έχει τυπογραφικό λάθος. | Ελέγξτε ξανά τη συμβολοσειρά καταλόγου και βεβαιωθείτε ότι τα ονόματα αρχείων ταιριάζουν ακριβώς, συμπεριλαμβανομένων των κενών. |
| **Το PNG εξόδου είναι κενό** | Το όνομα του layout δεν υπάρχει στο πηγαίο αρχείο CAD. | Χρησιμοποιήστε `cadImage.getLayouts()` για να εμφανίσετε τα διαθέσιμα layouts, έπειτα ενημερώστε το `rasterizationOptions.setLayouts(...)`. |
| **Μεγάλα αρχεία CAD προκαλούν OutOfMemoryError** | Η ραστεροποίηση εικόνων υψηλής ανάλυσης καταναλώνει πολύ heap. | Αυξήστε το heap της JVM (`-Xmx2g`) ή μειώστε την ανάλυση ραστεροποίησης μέσω `rasterizationOptions.setResolution(...)`. |
