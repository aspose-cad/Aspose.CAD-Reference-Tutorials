# Ενεργοποίηση Καταγραφής για τη Διαδικασία Απόδοσης CAD

## Εισαγωγή

Σε αυτό το tutorial θα μάθετε πώς να **ορίσετε το μέγεθος σελίδας PDF** ενώ **μετατρέπετε CAD σε PDF** χρησιμοποιώντας το **Aspose.CAD for Java**. Ενεργοποιώντας την καταγραφή αποκτάτε πλήρη ορατότητα πάνω στην αλυσίδα απόδοσης, καθιστώντας ευκολότερο τον εντοπισμό σφαλμάτων και τη βελτιστοποίηση της μετατροπής από αρχεία CAD (όπως DXF) σε PDF. Είτε χρειάζεστε **αποθήκευση CAD ως PDF**, δημιουργία PDF από DXF, είτε απλώς έλεγχο των διαστάσεων εξόδου, τα παρακάτω βήματα θα σας καθοδηγήσουν σε όλη τη διαδικασία.

## Γρήγορες Απαντήσεις
- **Τι κάνει η λειτουργία “set PDF page size”;** Ορίζει το πλάτος και το ύψος της τελικής σελίδας PDF κατά την απόδοση του CAD.  
- **Γιατί να ενεργοποιήσω την καταγραφή;** Η καταγραφή καταγράφει κάθε στάδιο της μετατροπής, βοηθώντας σας να εντοπίσετε σημεία συμφόρησης ή σφάλματα.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγική χρήση.  
- **Ποιες μορφές CAD υποστηρίζονται;** DWG, DXF, DGN και πολλές άλλες – δείτε την τεκμηρίωση του Aspose.CAD για την πλήρη λίστα.  
- **Μπορώ να αλλάξω τις διαστάσεις της σελίδας “on the fly”;** Ναι – απλώς προσαρμόστε τις τιμές `PageWidth` και `PageHeight` στο `CadRasterizationOptions`.

## Τι είναι το “set PDF page size” στην απόδοση CAD;

Η ρύθμιση του μεγέθους σελίδας PDF λέει στον rasterizer πόσο μεγάλο πρέπει να είναι το καμβάς όταν τα διανυσματικά δεδομένα CAD μετατρέπονται σε σελίδα PDF. Αυτό είναι κρίσιμο για τη διατήρηση της οπτικής πιστότητας, ειδικά σε λεπτομερή τεχνικά σχέδια.

## Γιατί να ενεργοποιήσετε την καταγραφή για την απόδοση CAD;

Η ενεργοποίηση της καταγραφής παρέχει λεπτομερή log κάθε βήματος—από τη φόρτωση του αρχικού αρχείου μέχρι τη δημιουργία του PDF. Σας βοηθά να:

- Διαγνώσετε γιατί ένα συγκεκριμένο σχέδιο μπορεί να αποδίδεται λανθασμένα.  
- Μετρήσετε το χρόνο που απαιτείται για κάθε στάδιο, χρήσιμο για βελτιστοποίηση απόδοσης.  
- Επαληθεύσετε ότι το μέγεθος σελίδας που διαμορφώσατε εφαρμόζεται πράγματι.

## Προαπαιτούμενα

Πριν προχωρήσετε στη ρύθμιση της καταγραφής, βεβαιωθείτε ότι έχετε τα εξής:

1. **Περιβάλλον Ανάπτυξης Java** – Java 8 ή νεότερη έκδοση εγκατεστημένη στον υπολογιστή σας.  
2. **Βιβλιοθήκη Aspose.CAD** – Κατεβάστε και ενσωματώστε τη βιβλιοθήκη Aspose.CAD στο έργο Java. Μπορείτε να βρείτε το σύνδεσμο λήψης [εδώ](https://releases.aspose.com/cad/java/).  
3. **Φάκελος Εγγράφων** – Προετοιμάστε έναν φάκελο για την αποθήκευση των αρχείων CAD και των παραγόμενων PDF.

## Εισαγωγή Ονομάτων Χώρου

Στο έργο Java, εισάγετε τα απαραίτητα ονόματα χώρου για να αξιοποιήσετε τις λειτουργίες του Aspose.CAD. Προσθέστε τις παρακάτω γραμμές στην αρχή του κώδικά σας:

```java
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.OutputStream;

import com.aspose.cad.Image;

import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.PdfOptions;
```

## Ορισμός Διαδρομής Καταλόγου Πόρων

```java
String dataDir = "Your Document Directory" + "CADConversion/";
```

Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή του φακέλου εγγράφων σας.

## Φόρτωση του Αρχείου CAD

```java
String srcFile = dataDir + "conic_pyramid.dxf";
Image image = Image.load(srcFile);
```

Καθορίστε τη διαδρομή του αρχείου CAD, βεβαιώνοντας ότι βρίσκεται εντός του καθορισμένου φακέλου εγγράφων.

## Ορισμός Επιλογών Εξόδου PDF

```java
OutputStream stream = new FileOutputStream(dataDir + "conic_pyramid.pdf");
PdfOptions pdfOptions = new PdfOptions();
```

Δημιουργήστε ένα stream εξόδου και ορίστε τις επιλογές PDF για τη μετατροπή.

## Διαμόρφωση CadRasterizationOptions (Ορισμός Μεγέθους Σελίδας PDF)

```java
CadRasterizationOptions cadRasterizationOptions = new CadRasterizationOptions();
pdfOptions.setVectorRasterizationOptions(cadRasterizationOptions);
cadRasterizationOptions.setPageWidth(800);
cadRasterizationOptions.setPageHeight(600);
```

Εδώ **ορίζουμε το μέγεθος σελίδας PDF** ορίζοντας τις τιμές `PageWidth` και `PageHeight`. Προσαρμόστε αυτές τις τιμές ώστε να ταιριάζουν με τις διαστάσεις που απαιτούνται για το τεχνικό σας σχέδιο. Αυτό είναι το κεντρικό βήμα που απαντά στο **πώς να ορίσετε το μέγεθος PDF** όταν **java cad to pdf**.

## Αποθήκευση του Αρχείου PDF

```java
image.save(stream, pdfOptions);
```

Αποθηκεύστε το παραχθέν PDF με τις καθορισμένες επιλογές.

## Επαλήθευση Ενεργοποίησης Καταγραφής

```java
System.out.println("Tracking enabled successfully for CAD rendering process.");
```

Επιβεβαιώστε ότι η καταγραφή έχει ενεργοποιηθεί επιτυχώς για τη διαδικασία απόδοσης CAD.

## Συχνά Προβλήματα & Αντιμετώπιση

| Σύμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| Η σελίδα PDF εμφανίζεται κενή | `PageWidth`/`PageHeight` ορίστηκαν σε 0 | Βεβαιωθείτε ότι παρέχονται μη‑μηδενικές διαστάσεις. |
| Το αρχείο εξόδου είναι κατεστραμμένο | Το stream εξόδου δεν κλείνει | Καλέστε `stream.close()` μετά το `image.save(...)`. |
| Λείπουν στρώματα στο PDF | Το αρχείο CAD χρησιμοποιεί μη υποστηριζόμενες οντότητες | Επαληθεύστε ότι η μορφή αρχείου υποστηρίζεται πλήρως από το Aspose.CAD. |
