## Συχνές Ερωτήσεις

**Q: Πώς μπορώ να εξάγω ένα αρχείο DWG σε PDF χωρίς να χάσω τις πληροφορίες των επιπέδων;**  
A: Χρησιμοποιήστε `PdfOptions` και ορίστε `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` για να διατηρήσετε τα διανυσματικά δεδομένα, στη συνέχεια καλέστε `image.save("output.pdf", options);`.

**Q: Μπορώ να επεξεργαστώ παρτίδα πολλαπλών αρχείων DWG σε PDF σε μία εκτέλεση;**  
A: Ναι—περιηγηθείτε στη συλλογή αρχείων σας, φορτώστε το καθένα με `Image.load`, και αποθηκεύστε το χρησιμοποιώντας τις ίδιες επιλογές PDF.

**Q: Υπάρχει τρόπος να μετατρέψω ένα πρότυπο DWT απευθείας σε DXF;**  
A: Απόλυτα. Φορτώστε το DWT με `Image.load` και αποθηκεύστε το ως DXF χρησιμοποιώντας `image.save("output.dxf");`.

**Q: Ποιες εκδόσεις Java υποστηρίζονται από το Aspose.CAD;**  
A: Η βιβλιοθήκη λειτουργεί με Java 8 και νεότερες, συμπεριλαμβανομένων των Java 11, 17 και μεταγενέστερων LTS εκδόσεων.

**Q: Χρειάζομαι ξεχωριστή άδεια για την εξαγωγή PDF;**  
A: Όχι, η άδεια Aspose.CAD καλύπτει όλες τις μορφές εξόδου, συμπεριλαμβανομένων των PDF, raster εικόνων και DXF.

**Q: Πώς μπορώ να βελτιώσω την ταχύτητα απόδοσης κατά τη μετατροπή μεγάλων αρχείων DWG;**  
A: Ενεργοποιήστε το multi‑threading επεξεργάζοντας κάθε διάταξη σε ξεχωριστό νήμα και επαναχρησιμοποιήστε ένα μόνο αντικείμενο `PdfOptions` για μείωση του κόστους.

**Q: Μπορώ να ορίσω προσαρμοσμένο μέγεθος σελίδας για το εξαγόμενο PDF;**  
A: Ναι, διαμορφώστε το `PdfOptions` με `PdfOptions.setPageSize(PageSize.A4)` (ή οποιοδήποτε άλλο μέγεθος) πριν την αποθήκευση.
