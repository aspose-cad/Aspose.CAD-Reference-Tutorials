# Πώς να Προσαρμόσετε τις Γραμματοσειρές στο Κείμενο CAD και τις Σχόλια

## Εισαγωγή 

Αν ψάχνετε να **προσαρμόσετε τις γραμματοσειρές** στα DWG σχέδιά σας, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα σας καθοδηγήσουμε στη προσθήκη κειμένου, την αντικατάσταση γραμματοσειρών και τη ρύθμιση των στυλ γραμματοσειρών χρησιμοποιώντας το Aspose.CAD for Java. Είτε βελτιώνετε ένα σχήμα, ετοιμάζετε έγγραφα κατασκευής, είτε απλώς θέλετε πιο καθαρά **dwg text annotation**, αυτά τα βήματα θα σας βοηθήσουν να πετύχετε ένα επαγγελματικό αποτέλεσμα γρήγορα και αξιόπιστα.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος σκοπός της προσαρμογής γραμματοσειρών σε DWG;** Να βελτιώσει την αναγνωσιμότητα και να ταιριάζει με το branding ή τα πρότυπα του έργου.  
- **Ποια βιβλιοθήκη διαχειρίζεται τις αλλαγές;** Aspose.CAD for Java.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να επεξεργαστώ μεγάλα αρχεία DWG;** Ναι – το API μεταδίδει δεδομένα αποδοτικά, κατάλληλο για μεγάλα σχέδια.  
- **Απαιτείται κάποιο επιπλέον λογισμικό;** Μόνο ένα Java runtime (JDK 8 ή νεότερο) και το Aspose.CAD JAR.

## Τι σημαίνει “προσαρμογή γραμματοσειρών” στο CAD;
Η προσαρμογή γραμματοσειρών σημαίνει την αντικατάσταση του προεπιλεγμένου στυλ κειμένου σε ένα αρχείο DWG με μια γραμματοσειρά της επιλογής σας, ρυθμίζοντας το μέγεθος, το βάρος ή εφαρμόζοντας διαφορετικό στυλ σε συγκεκριμένα στρώματα ή αντικείμενα. Αυτό εξασφαλίζει ότι το σχέδιο φαίνεται ακριβώς όπως το επιθυμείτε σε όλους τους προβολείς.

## Γιατί να χρησιμοποιήσετε το Aspose.CAD for Java για την προσαρμογή γραμματοσειρών;
- **Πλήρης έλεγχος** στα αντικείμενα κειμένου χωρίς το άνοιγμα του σχεδίου σε επεξεργαστή GUI.  
- **Επεξεργασία παρτίδας** – διαχειριστείτε δεκάδες αρχεία σε ένα μόνο script.  
- **Διαπλατφορμική** – λειτουργεί σε Windows, Linux και macOS.  
- **Χωρίς εξωτερικές εξαρτήσεις** – το API διαχειρίζεται την ανάλυση DWG εσωτερικά.

## Προαπαιτούμενα
- Εγκατεστημένο Java Development Kit 8 ή νεότερο.  
- Προστέθηκε το Aspose.CAD for Java JAR στο classpath του έργου σας.  
- Ένα αρχείο DWG που θέλετε να επεξεργαστείτε.

## Πώς να Προσθέσετε Κείμενο σε DWG
Η προσθήκη νέας κειμενικής πληροφορίας είναι συχνή ανάγκη όταν θέλετε να επισημάνετε μέρη, να προσθέσετε σημειώσεις ή να δημιουργήσετε **dwg text annotation**. Ακολουθήστε τα παρακάτω βήματα:

1. **Φορτώστε το αρχείο DWG** χρησιμοποιώντας το `CadImage`.  
2. **Δημιουργήστε μια οντότητα `CadText`** με το επιθυμητό κείμενο, θέση και γραμματοσειρά.  
3. **Προσθέστε την οντότητα** στη συλλογή οντοτήτων του σχεδίου.  
4. **Αποθηκεύστε** το τροποποιημένο αρχείο.

> *Σημείωση: Το πραγματικό απόσπασμα κώδικα παραμένει αμετάβλητο από το αρχικό tutorial και περιλαμβάνεται στο συνδεδεμένο sub‑tutorial.*

## Πώς να Αντικαταστήσετε τη Γραμματοσειρά σε DWG
Η αντικατάσταση μιας υπάρχουσας γραμματοσειράς σε όλο το σχέδιο βοηθά στη διατήρηση της συνέπειας, ειδικά όταν η αρχική γραμματοσειρά δεν είναι διαθέσιμη στο σύστημα-στόχο.

1. **Ανοίξτε το DWG** με το `CadImage`.  
2. **Επανάληψη** σε όλα τα αντικείμενα `CadText`.  
3. **Ορίστε την ιδιότητα `FontName`** στη γραμματοσειρά της επιλογής σας (π.χ., “Arial”).  
4. **Αποθηκεύστε** το ενημερωμένο σχέδιο.

Αυτή η μέθοδος καλύπτεται λεπτομερώς στο sub‑tutorial “Substitute Font in DWG”.

## Πώς να Αλλάξετε το Στυλ Γραμματοσειράς DWG για Συγκεκριμένο Στυλ
Μερικές φορές μόνο ένα συγκεκριμένο στυλ κειμένου (π.χ., “Title”) χρειάζεται νέα γραμματοσειρά ενώ τα άλλα παραμένουν αμετάβλητα.

1. **Εντοπίστε το όνομα του στυλ** που θέλετε να τροποποιήσετε.  
2. **Βρείτε το αντίστοιχο αντικείμενο `CadTextStyle`**.  
3. **Ενημερώ το `FontName`** του και τυχόν πρόσθετα χαρακτηριστικά στυλ.  
4. **Διατηρήστε τις αλλαγές** αποθηκεύοντας το αρχείο.

Ο οδηγός βήμα‑βήμα είναι διαθέσιμος στο sub‑tutorial “Substitute Font of a Particular Style in DWG”.

## Κοινές Περιπτώσεις Χρήσης
- **Σχέδια κατασκευής** όπου οι εταιρικές τυποποιημένες γραμματοσειρές είναι υποχρεωτικές.  
- **Αρχιτεκτονικά σχέδια** που χρειάζονται ευανάγνωστα σχόλια για πελάτες.  
- **Μαζική μετατροπή** παλαιών αρχείων DWG σε νέο σύνολο εταιρικών γραμματοσειρών.

## Συχνές Ερωτήσεις

**Ε: Μπορώ να χρησιμοποιήσω αυτές τις μεθόδους με αρχεία DWG που δημιουργήθηκαν σε παλαιότερες εκδόσεις του AutoCAD;**  
**Α: Ναι. Το Aspose.CAD υποστηρίζει εκδόσεις DWG από R12 έως τις πιο πρόσφατες εκδόσεις.**

**Ε: Τι συμβαίνει αν η στοχευόμενη γραμματοσειρά δεν είναι εγκατεστημένη στον υπολογιστή του θεατή;**  
**Α: Το σχέδιο θα επιστρέψει σε προεπιλεγμένη γραμματοσειρά, κάτι που μπορεί να επηρεάσει τη διάταξη. Συνιστάται η ενσωμάτωση της γραμματοσειράς ή η εγκατάστασή της σε όλους τους υπολογιστές.**

**Ε: Είναι δυνατόν να αντικαταστήσετε γραμματοσειρές μόνο σε συγκεκριμένο στρώμα;**  
**Α: Απόλυτα. Φιλτράρετε τα αντικείμενα `CadText` με βάση το `LayerName` τους πριν αλλάξετε το `FontName`.**

**Ε: Πρέπει να ξαναχτίσω το σχέδιο μετά τις αλλαγές γραμματοσειράς;**  
**Α: Δεν απαιτείται χειροκίνητη επανακατασκευή· η αποθήκευση του `CadImage` γράφει όλες τις ενημερώσεις στο αρχείο.**

**Ε: Πώς μπορώ να επαληθεύσω ότι η αλλαγή γραμματοσειράς εφαρμόστηκε σωστά;**  
**Α: Ανοίξτε το DWG σε οποιονδήποτε προβολέα που υποστηρίζει την επιλεγμένη γραμματοσειρά, ή προγραμματιστικά απαριθμήστε τα αντικείμενα `CadText` για να διαβάσετε το `FontName`.**
