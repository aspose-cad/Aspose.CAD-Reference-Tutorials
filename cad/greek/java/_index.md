## Πώς να Εξάγετε CAD σε SVG
Η εξαγωγή CAD σε SVG είναι απλή με το Aspose.CAD. Φορτώστε το πηγαίο αρχείο, δημιουργήστε μια παρουσία `SvgOptions` και καλέστε `save`. Το SVG διατηρεί τις διανυσματικές πληροφορίες, καθιστώντας το ιδανικό για προβολή στο web ή περαιτέρω επεξεργασία σε επεξεργαστές διανυσματικών γραφικών.

## Πώς να Εξάγετε CAD σε STL
Για ροές εργασίας 3D εκτύπωσης, μπορείτε να εξάγετε μοντέλα CAD σε STL. Χρησιμοποιήστε την κλάση `StlOptions`, καθορίστε μορφή binary ή ASCII και αποθηκεύστε το αρχείο. Η διαδικασία διατηρεί την τοπολογία του mesh που απαιτείται από τους περισσότερους slicers.

## Πώς να Μετατρέψετε DWFX σε PDF
Τα αρχεία DWFX, που συχνά παράγονται από το Autodesk Design Review, μπορούν να μετατραπούν σε PDF χρησιμοποιώντας την ίδια ροή εργασίας `PdfOptions` όπως για άλλα μορφότυπα CAD. Απλώς φορτώστε το αρχείο DWFX και καλέστε `save` με επιλογές PDF.

## Πώς να Rendering DWG σε Εικόνα
Το rendering ενός DWG σε raster εικόνα (PNG, JPEG, BMP) περιλαμβάνει τη δημιουργία ενός αντικειμένου `RasterizationOptions`, τον καθορισμό της επιθυμητής ανάλυσης και την αποθήκευση του αποτελέσματος. Χρήσιμο για δημιουργία προεπισκοπήσεων ή ενσωμάτωση σχεδίων σε αναφορές.

## Πώς να Εξάγετε Layout CAD σε PDF
Αν χρειάζεται να **εξάγετε PDF layout CAD** για ένα συγκεκριμένο layout μέσα σε ένα σχέδιο, ορίστε την ιδιότητα `LayoutName` στο `PdfOptions` πριν την αποθήκευση. Αυτό σας επιτρέπει να απομονώσετε ένα μόνο φύλλο ή προβολή model space για διανομή.
