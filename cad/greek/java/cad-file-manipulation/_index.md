## Πρόσβαση στις Σημαίες Υποβάθρου (Underlay) του DWG

Η κατανόηση των σημαίων υποβάθρου σας βοηθά να ελέγξετε πώς εμφανίζονται οι εξωτερικές αναφορές (Xrefs) σε ένα αρχείο DWG. Με την Aspose.CAD μπορείτε να ερωτήσετε αυτές τις σημαίες προγραμματιστικά, επιτρέποντάς σας να κρύψετε ή να εμφανίσετε υποβάθρα βάσει της λογικής της εφαρμογής σας.

## Λίστα Διατάξεων (Layouts) σε DWG

Τα αρχεία DWG μπορούν να περιέχουν πολλαπλές διατάξεις (paper spaces). Η λίστα τους σας επιτρέπει να δώσετε στους χρήστες τη δυνατότητα επιλογής της διάταξης που θα αποδοθεί ή θα εξαχθεί. Η Aspose.CAD παρέχει εύκολη απαρίθμηση των ονομάτων διατάξεων, καθιστώντας την ενσωμάτωση σε UI στοιχεία απλή.

## Αυτόματη Προσαρμογή Μεγέθους Σχεδίου CAD

Όταν χρειάζεται να προσαρμόσετε ένα σχέδιο σε συγκεκριμένο μέγεθος χαρτιού ή συντελεστή κλιμάκωσης, η λειτουργία αυτόματης προσαρμογής υπολογίζει τις βέλτιστες διαστάσεις αυτόματα. Αυτό είναι ιδιαίτερα χρήσιμο για επεξεργασία παρτίδας μεγάλων συνόλων σχεδίων όπου η χειροκίνητη κλιμάκωση θα ήταν μη πρακτική.

## Προσαρμογή Μονάδας Μεγέθους CAD

Αν το έργο σας απαιτεί ακριβή έλεγχο των διαστάσεων του σχεδίου — όπως η μετατροπή από χιλιοστά σε ίντσες — θα χρειαστείτε **adjust cad size unit**. Η Aspose.CAD σας επιτρέπει να καθορίσετε τον τύπο της στοχευόμενης μονάδας και αυτόματα να επανακλιμακώσετε τη γεωμετρία, διασφαλίζοντας ότι το αποτέλεσμα ταιριάζει με τα απαιτούμενα πρότυπα.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί Συμβαίνει | Γρήγορη Διόρθωση |
|----------|----------------|-------------------|
| **Out‑of‑memory errors on large DWFX files** | Το πλήρες αρχείο φορτώνεται στη μνήμη από προεπιλογή. | Αυξήστε το heap της JVM (`-Xmx`) ή επεξεργαστείτε το αρχείο σε τμήματα χρησιμοποιώντας `CadImage.load` με επιλογές ροής. |
| **Incorrect page orientation** | Η προεπιλογή `PdfOptions` χρησιμοποιεί κατακόρυφη προσανατολισμό. | Ορίστε `PdfOptions.setPageSize(PageSize.A4.rotate())` πριν την αποθήκευση. |
| **Missing raster images in the PDF** | Οι ραστερ εικόνες ενσωματώνονται ως εξωτερικές αναφορές. | Ενεργοποιήστε την ενσωμάτωση ραστερ εικόνων μέσω `PdfOptions.setRasterizeImages(true)`. |

**Q: Μπορώ να μετατρέψω αρχεία DWFX που περιέχουν ραστερ εικόνες;**

**Q: Πώς αλλάζω τον προσανατολισμό της σελίδας PDF;**

**Q: Τι γίνεται αν χρειαστεί να μετατρέψω DWFX σε άλλες μορφές όπως SVG;**  
A: Η Aspose.CAD υποστηρίζει πολλαπλές μορφές εξόδου· απλώς αλλάξτε την παράμετρο μορφής της μεθόδου `save`.

**Q: Η βιβλιοθήκη διαχειρίζεται μεγάλα αρχεία DWFX χωρίς υψηλή κατανάλωση μνήμης;**
