# Εξαγωγή DGN σε DWG με Aspose.CAD for Java

## Εισαγωγή

Σε αυτόν τον ολοκληρωμένο οδηγό θα μάθετε πώς να **export dgn to dwg** χρησιμοποιώντας το Aspose.CAD for Java, καθώς και πώς να **convert dgn to pdf**, **convert dgn to png** και **convert dgn to jpeg**. Είτε ανανεώνετε μια παλιά ροή εργασίας MicroStation είτε δημιουργείτε μια νέα υπηρεσία προσανατολισμένη στο CAD, τα παρακάτω βήματα σας δείχνουν ακριβώς πώς να εκτελέσετε αυτές τις μετατροπές αποδοτικά και με υψηλή πιστότητα.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος σκοπός της εξαγωγής dgn σε dwg;**  
  Σας επιτρέπει να ενσωματώσετε σχέδια MicroStation DGN σε έργα DWG συμβατά με AutoCAD.
- **Μπορεί το Aspose.CAD να μετατρέψει dgn σε pdf;**  
  Ναι – το API παρέχει έναν απλό τρόπο για **convert dgn to pdf**.
- **Χρειάζομαι άδεια για παραγωγική χρήση;**  
  Απαιτείται εμπορική άδεια για την ανάπτυξη· διατίθεται δωρεάν δοκιμή για αξιολόγηση.
- **Ποια έκδοση της Java υποστηρίζεται;**  
  Η Java 8 και νεότερες εκδόσεις υποστηρίζονται πλήρως.
- **Είναι δυνατή η εξαγωγή εικόνας raster;**  
  Απολύτως – μπορείτε να εξάγετε DGN σε JPEG, PNG και άλλες μορφές raster.

## Τι είναι το **export dgn to dwg**;
`Export dgn to dwg` είναι η διαδικασία μετατροπής ενός αρχείου MicroStation DGN σε μορφή AutoCAD DWG. Αυτή η μετατροπή διατηρεί τα επίπεδα, τη γεωμετρία και τα μεταδεδομένα, επιτρέποντας αδιάλειπτη συνεργασία μεταξύ διαφορετικών πλατφορμών CAD.

## Γιατί να χρησιμοποιήσετε το Aspose.CAD for Java για **export dgn to dwg**;
Η εξαγωγή DGN σε DWG με το Aspose.CAD for Java σας παρέχει μια λύση καθαρά Java που δεν απαιτεί **καμία εξωτερική εγγενή βιβλιοθήκη**. Η βιβλιοθήκη υποστηρίζει **πάνω από 30 μορφές CAD**, μπορεί να χειριστεί αρχεία έως **500 MB** χωρίς να φορτώνει ολόκληρο το έγγραφο στη μνήμη, και διατηρεί **99,9 % γεωμετρική πιστότητα** κατά τις μετατροπές. Η επεξεργασία παρτίδας είναι ενσωματωμένη, ώστε να μπορείτε να μετατρέψετε δεκάδες αρχεία με έναν μόνο βρόχο.

## Οδηγοί βήμα‑βήμα

### Εξαγωγή DGN ως μέρος του DWG
Αυτό το σενάριο είναι κοινό όταν ένα έργο περιέχει περιουσιακά στοιχεία μικτής μορφής και χρειάζεστε ένα ενιαίο κοντέινερ DWG που να περιέχει τα αρχικά δεδομένα DGN.

#### Πώς να εξάγετε dgn σε dwg
Φορτώστε το αρχείο DGN προέλευσης χρησιμοποιώντας το `CadImage`, ενσωματώστε το σε ένα νέο κοντέινερ DWG και αποθηκεύστε το αποτέλεσμα. Αυτό το μοτίβο δύο βημάτων διαχειρίζεται τη μετατροπή στη μνήμη και γράφει ένα αρχείο DWG σύμφωνο με τα πρότυπα.

`CadImage` είναι η βασική κλάση του Aspose.CAD που αντιπροσωπεύει μια εικόνα CAD φορτωμένη στη μνήμη.  

1. Φορτώστε το αρχείο DGN με `CadImage.load("source.dgn")`.  
2. Δημιουργήστε ένα νέο αντικείμενο εικόνας DWG και προσθέστε το φορτωμένο DGN ως ενσωματωμένη οντότητα.  
3. Καλέστε `save("output.dwg", SaveFormat.Dwg)` για να γράψετε το αρχείο DWG.

> *Το λεπτομερές παράδειγμα κώδικα είναι διαθέσιμο στην αφιερωμένη υπο‑εκπαιδευτική ενότητα που συνδέεται παρακάτω.*

### Εξαγωγή ενσωματωμένου DGN σε PDF
Μάθετε να **convert dgn to pdf** διατηρώντας τυχόν ενσωματωμένο περιεχόμενο DGN μέσα στο PDF.

#### Πώς να εξάγετε ενσωματωμένο DGN σε PDF
Ανοίξτε το αρχείο DGN, διαμορφώστε το `PdfOptions` για έξοδο PDF και αποθηκεύστε. Το API ενσωματώνει αυτόματα τα αρχικά δεδομένα DGN στο PDF για μετέπειτα εξαγωγή.

`PdfOptions` ορίζει όλες τις ρυθμίσεις ειδικές για PDF όπως το μέγεθος σελίδας, τη συμπίεση και τα μεταδεδομένα.  

1. Ανοίξτε το αρχείο DGN με `CadImage.load("source.dgn")`.  
2. Δημιουργήστε ένα αντικείμενο `PdfOptions` και ορίστε τις απαιτούμενες επιλογές (π.χ., `setEmbedDgn(true)`).  
3. Αποθηκεύστε την εικόνα ως PDF χρησιμοποιώντας `save("output.pdf", SaveFormat.Pdf, pdfOptions)`.

> *Το πλήρες δείγμα κώδικα μπορεί να βρεθεί στην εκπαίδευση “Export Embedded DGN”.*

### Εξαγωγή DGN σε PDF συμβατό με AutoCAD
Δημιουργήστε ένα PDF που μιμείται ένα σχέδιο AutoCAD, χρήσιμο για ανασκόπηση από ενδιαφερόμενους όταν δεν είναι εγκατεστημένο λογισμικό CAD.

#### Πώς να εξάγετε DGN σε PDF συμβατό με AutoCAD
Φορτώστε το DGN, ορίστε τη λειτουργία απόδοσης PDF σε AutoCAD και αποθηκεύστε. Το προκύπτον PDF διατηρεί τα βάρη γραμμών και τα χρώματα επιπέδων, ταιριάζοντας με το οπτικό στυλ DWG.

`PdfOptions` προσφέρει επίσης μια σημαία `AutoCadMode` που επιβάλλει απόδοση σε στυλ DWG.  

1. Φορτώστε το αρχείο DGN.  
2. Ενεργοποιήστε την απόδοση AutoCAD στο `PdfOptions`.  
3. Αποθηκεύστε το αρχείο ως PDF.

### Εξαγωγή DGN σε μορφή εικόνας raster
Δημιουργήστε εικόνες υψηλής ανάλυσης JPEG, PNG ή BMP από αρχεία DGN για προεπισκοπήσεις ιστού, τεκμηρίωση ή μικρογραφίες.

#### Πώς να εξάγετε DGN σε εικόνες raster
Φορτώστε το DGN, καθορίστε τις επιλογές εξαγωγής raster όπως DPI και βάθος χρώματος, και στη συνέχεια αποθηκεύστε στη ζητούμενη μορφή εικόνας.

`RasterImageExportOptions` σας επιτρέπει να ελέγχετε την ανάλυση, την αντι-αποκοπή (anti‑aliasing) και το βάθος χρώματος.  

1. Φορτώστε την εικόνα DGN.  
2. Διαμορφώστε το `RasterImageExportOptions` (π.χ., `setDpi(300)`).  
3. Αποθηκεύστε ως JPEG, PNG ή BMP χρησιμοποιώντας το κατάλληλο `SaveFormat`.

## Κοινές περιπτώσεις χρήσης και συμβουλές
- **Batch conversion pipelines** – επαναλάβετε πάνω σε έναν φάκελο αρχείων DGN, εφαρμόζοντας την ίδια λογική εξαγωγής σε κάθε αρχείο.  
- **Preserving metadata** – χρησιμοποιήστε `PdfOptions.setMetadata()` για να ενσωματώσετε τα αρχικά ονόματα επιπέδων και τις προσαρμοσμένες ιδιότητες στο PDF.  
- **Performance optimisation** – για μεγάλα σχέδια, ενεργοποιήστε τη λειτουργία streaming (`setUseMemoryCache(true)`) ώστε η χρήση μνήμης να παραμένει χαμηλή.  
- **Pro tip:** Κατά τη μετατροπή σε εικόνες raster για χρήση στο web, ένα DPI 150 εξισορροπεί την ποιότητα και το μέγεθος αρχείου.
