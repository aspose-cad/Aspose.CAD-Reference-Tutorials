- **Was ist der Hauptzweck von Aspose.CAD für .NET?** Dateien im CAD- und BIM-Format (einschließlich DWG, DWFX, DWF) programmgesteuert zu lesen, zu bearbeiten und zu konvertieren.

- **Welche .NET-Versionen werden unterstützt?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Häufig gestellte Fragen

**Q: Wie öffne ich eine DWFX-Datei in C#?**  
A: Verwenden Sie `Aspose.CAD.Image.Load("file.dwfx")`, um die Datei zu laden, und greifen Sie anschließend nach Bedarf auf deren Seiten oder Entitäten zu.

**Q: Kann ich eine DWG-Datei in C# lesen, ohne sie vorher zu konvertieren?**  
A: Ja – Aspose.CAD kann DWG-Dateien direkt laden, sodass Sie Ebenen, Entitäten und Metadaten abfragen können.

**Q: Was ist der einfachste Weg, Text zu einer DWG-Zeichnung hinzuzufügen?**  
A: Erstellen Sie eine `CadText`‑Entität, setzen Sie deren `Value` und Position und fügen Sie sie der Model‑Space‑Sammlung der Zeichnung hinzu.

**Q: Gibt es Größenbeschränkungen beim Arbeiten mit großen DWFX‑ oder DWG‑Dateien?**  
A: Die Bibliothek verarbeitet große Dateien effizient, jedoch sollten Sie ausreichend Speicher bereitstellen und das Streaming großer Rasterbilder in Betracht ziehen.

**Q: Benötige ich für jedes CAD‑Format eine separate Lizenz?**  
A: Nein – eine einzige Aspose.CAD‑Lizenz deckt alle unterstützten Formate ab, einschließlich DWG, DWFX, DWF, DXF und mehr.
