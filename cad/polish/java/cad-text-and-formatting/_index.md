## Co to jest wyszukiwanie tekstu w DWG?
Wyszukiwanie tekstu w plikach DWG oznacza lokalizowanie wartości ciągów znaków wewnątrz jednostek rysunku, takich jak MText, Text i definicje atrybutów. Ta funkcja jest niezbędna do zadań takich jak generowanie raportów, automatyzacja rewizji czy budowanie przeszukiwalnych repozytoriów CAD.

## Dlaczego warto używać Aspose.CAD for Java do tekstu i formatowania DWG?
- **Szybkość:** API przetwarza duże rysunki szybko, bez konieczności instalacji AutoCAD.  
- **Dokładność:** Obsługuje wszystkie jednostki tekstowe DWG, zachowując formatowanie i styl.  
- **Elastyczność:** Działa na różnych platformach (Windows, Linux, macOS) i łatwo integruje się z istniejącymi aplikacjami Java.  
- **Rozszerzalność:** Obsługuje ukryte linie, jednostki MLeader oraz manipulację atrybutami od razu po instalacji.
