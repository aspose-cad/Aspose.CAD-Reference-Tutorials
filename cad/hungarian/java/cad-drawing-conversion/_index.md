## Gyakran Ismételt Kérdések

**Q: Hogyan exportálhatok DWG fájlt PDF‑be anélkül, hogy elveszíteném a réteginformációkat?**  
A: Használja a `PdfOptions`‑t, és állítsa be a `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)`‑t a vektoradatok megtartásához, majd hívja meg a `image.save("output.pdf", options);` metódust.

**Q: Feldolgozhatok több DWG fájlt egyszerre PDF‑be egy futtatás során?**  
A: Igen—iteráljon a fájlgazdáján, töltse be mindegyiket a `Image.load`‑val, és mentse ugyanazokkal a PDF beállításokkal.

**Q: Van mód a DWT sablont közvetlenül DXF‑be konvertálni?**  
A: Természetesen. Töltse be a DWT‑t a `Image.load`‑val, és mentse DXF‑ként a `image.save("output.dxf");` metódussal.

**Q: Mely Java verziókat támogatja az Aspose.CAD?**  
A: A könyvtár Java 8‑tól kezdve, valamint a későbbi verziókkal, beleértve a Java 11‑et, 17‑et és a későbbi LTS kiadásokat működik.

**Q: Szükségem van külön licencre a PDF exporthoz?**  
A: Nem, az Aspose.CAD licenc lefedi az összes kimeneti formátumot, beleértve a PDF‑t, raszteres képeket és a DXF‑t.
