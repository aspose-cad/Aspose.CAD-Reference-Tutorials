# ส่งออก CAD เป็น SVG – การส่งออกภาพวาด CAD ไปเป็นรูปแบบ SVG

## บทนำ

การส่งออกภาพวาด CAD ไปเป็น SVG เป็นความต้องการทั่วไปเมื่อคุณต้องการกราฟิกที่ไม่ขึ้นกับความละเอียดสำหรับหน้าเว็บ รายงาน หรือการแสดงผลแบบโต้ตอบ ในบทเรียนนี้คุณจะได้เรียนรู้ **วิธีส่งออก CAD เป็น SVG** ด้วย Aspose.CAD for .NET สำรวจตัวเลือกเพื่อ **ปรับแต่งสี SVG** และดูวิธี **แปลง DWG เป็น SVG** (หรือ DXF เป็น SVG) เพียงไม่กี่บรรทัดของโค้ด ขั้นตอนรวดเร็ว API ใช้งานง่าย และไฟล์ SVG ที่ได้จะสเกลได้อย่างสมบูรณ์บนอุปกรณ์ใด ๆ

## คำตอบสั้น
- **ไลบรารีที่จัดการการแปลงคืออะไร?** Aspose.CAD for .NET.  
- **ฉันสามารถเปลี่ยนโหมดสีได้หรือไม่?** ได้ – ใช้ `SvgColorMode` เพื่อตั้งค่าเป็นระดับสีเทา, ขาว‑ดำ หรือสีเต็ม.  
- **รูปแบบ CAD ใดบ้างที่รองรับ?** DWG, DXF และรูปแบบอื่น ๆ อีกหลายประเภท (ดูเอกสาร Aspose.CAD).  
- **ฉันต้องการใบอนุญาตสำหรับการพัฒนาหรือไม่?** มีใบอนุญาตชั่วคราวสำหรับการทดสอบ.  
- **การแปลงใช้เวลานานเท่าไหร่?** ปกติใช้เวลาน้อยกว่าวินาทีสำหรับภาพวาดมาตรฐาน.

## การ “ส่งออก CAD เป็น SVG” คืออะไร?
การส่งออก CAD เป็น SVG หมายถึงการนำไฟล์ CAD ดั้งเดิม (เช่น DWG หรือ DXF) มารูปแบบเป็น Scalable Vector Graphics SVG ซึ่งเป็นรูปแบบ XML‑based ที่มีน้ำหนักเบาและสามารถกำหนดสไตล์ด้วย CSS ทำให้เหมาะกับแอปพลิเคชันเว็บและมือถือสมัยใหม่

## ทำไมต้องใช้ Aspose.CAD สำหรับการส่งออก CAD เป็น SVG?
- **ไม่มีการพึ่งพาภายนอก** – API .NET แท้, ไม่ต้องติดตั้ง AutoCAD.  
- **การควบคุมละเอียด** – คุณสามารถ **ปรับแต่งสี SVG**, ตัดสินใจว่าข้อความจะเก็บเป็นรูปทรงหรือไม่, และเลือกตัวเลือกการเรสเตอร์ไลซ์.  
- **รองรับรูปแบบกว้าง** – โค้ดเดียวกันทำงานสำหรับ **แปลง DWG เป็น SVG**, **แปลง DXF เป็น SVG**, และรูปแบบอื่น ๆ, ทำให้ฐานโค้ดของคุณง่ายขึ้น.  
- **ประสิทธิภาพสูง** – ปรับให้เร็วและใช้หน่วยความจำน้อย, เหมาะสำหรับการประมวลผลเป็นชุด.

## นำเข้า Namespaces

ก่อนอื่น ให้นำ Namespaces ที่จำเป็นเข้ามาในโปรเจกต์ของคุณเพื่อให้คลาสต่าง ๆ พร้อมใช้งาน

```csharp
using Aspose.CAD.ImageOptions;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## คู่มือขั้นตอนต่อขั้นตอน

### ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร  
กำหนดโฟลเดอร์ที่ไฟล์ CAD ต้นฉบับของคุณอยู่และที่ไฟล์ SVG จะถูกบันทึก

```csharp
// The path to the documents directory.
string MyDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: โหลดภาพวาด CAD  
ใช้ `Image.Load` เพื่ออ่านไฟล์ DWG (หรือ DXF) ซึ่งทำงานได้สำหรับสถานการณ์ **load DWG .NET**  

```csharp
using (Image image = Image.Load(MyDir + "sample.dwg"))
{
```

### ขั้นตอนที่ 3: กำหนดค่าตัวเลือกการส่งออก SVG  
ปรับการตั้งค่าการส่งออกให้ตรงกับความต้องการของคุณ ที่นี่เราตั้งค่าโหมดสีเป็นระดับสีเทาและบังคับให้ข้อความทั้งหมดเรนเดอร์เป็นรูปทรงเวกเตอร์  

```csharp
    var options = new SvgOptions();
    options.ColorType = SvgColorMode.Grayscale;   // customize SVG color mode
    options.TextAsShapes = true;                  // ensures text appears as shapes
```

### ขั้นตอนที่ 4: บันทึกไฟล์ SVG  
สุดท้ายให้เขียนไฟล์ SVG ลงดิสก์ ขั้นตอนนี้ **บันทึก CAD เป็น SVG** และเสร็จสิ้นการแปลง  

```csharp
    image.Save(MyDir + "sample.svg", options);
}
```

โดยทำตามสี่ขั้นตอนสั้น ๆ นี้ คุณสามารถ **แปลง DWG เป็น SVG** (หรือ **แปลง DXF เป็น SVG**) พร้อมควบคุมลักษณะการแสดงผลของผลลัพธ์ได้อย่างเต็มที่

## ปัญหาทั่วไปและวิธีแก้

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|-------|--------|
| **Blank SVG output** | เส้นทางไฟล์ต้นทางไม่ถูกต้องหรือไฟล์เสียหาย | ตรวจสอบ `MyDir` และให้แน่ใจว่าไฟล์ CAD เปิดได้โดยไม่มีข้อผิดพลาด |
| **Colors look wrong** | ตั้งค่า `SvgColorMode` เป็น Grayscale โดยไม่ได้ตั้งใจ | เปลี่ยน `options.ColorType` เป็น `SvgColorMode.FullColor` เพื่อรักษาสีเดิม |
| **Text missing** | `TextAsShapes` ตั้งเป็น `false` และตัวดูไม่รองรับฟอนต์ที่ฝังอยู่ | ตั้งค่า `TextAsShapes = true` หรือฝังฟอนต์ที่จำเป็นใน SVG |

## Frequently Asked Questions

**Q: สามารถใช้โค้ดนี้กับ .NET Core หรือ .NET 6 ได้หรือไม่?**  
A: แน่นอน. Aspose.CAD for .NET รองรับ .NET Framework, .NET Core, และ .NET 5/6+

**Q: สามารถส่งออกเฉพาะเลย์เอาต์หรือเลเยอร์ที่ต้องการได้หรือไม่?**  
A: ได้. ใช้ `SvgOptions` เพื่อตั้งค่า `PageIndex` หรือกรองเลเยอร์ก่อนบันทึก

**Q: จะฝังสไตล์ CSS แบบกำหนดเองลงใน SVG ที่สร้างขึ้นอย่างไร?**  
A: หลังบันทึก คุณสามารถทำการประมวลผลต่อ XML ของ SVG เพื่อแทรก `<style>` หรือใช้ `options.CustomCss` หากเวอร์ชันใหม่มีให้

**Q: มีขีดจำกัดขนาดของไฟล์ CAD ต้นทางหรือไม่?**  
A: ไลบรารีรองรับไฟล์ขนาดใหญ่ แต่การใช้หน่วยความจำจะเพิ่มตามความซับซ้อน สำหรับภาพวาดขนาดใหญ่มาก ควรพิจารณาประมวลผลหน้าแยกกัน

**Q: การแปลงจะรักษาน้ำหนักเส้นและประเภทเส้นไว้หรือไม่?**  
A: โดยค่าเริ่มต้น น้ำหนักเส้นจะถูกเก็บไว้ คุณสามารถปรับตัวเลือกการเรนเดอร์ใน `SvgOptions` เพื่อควบคุมละเอียดเพิ่มเติมได้
