# แปลง DWG เป็น PDF พร้อมการสนับสนุน Mesh ใน Aspose.CAD สำหรับ .NET

## บทนำ

ยินดีต้อนรับสู่บทแนะนำเชิงลึกของเราว่า **วิธีการแปลง DWG เป็น PDF** ด้วย Aspose.CAD สำหรับ .NET! Aspose.CAD เป็นไลบรารีที่ทรงพลังซึ่งให้ฟังก์ชันการทำงานที่แข็งแกร่งสำหรับการทำงานกับไฟล์ Computer‑Aided Design (CAD) ในแอปพลิเคชัน .NET ในคู่มือนี้เราจะเน้นที่ฟีเจอร์การสนับสนุน mesh ซึ่งทำให้ **cad mesh conversion** เป็นไปอย่างราบรื่นและให้คุณ **save CAD as PDF** ด้วยความแม่นยำสูง

## คำตอบอย่างรวดเร็ว
- **ฟังก์ชันของ mesh support คืออะไร?** มันรักษาเรขาคณิต mesh แบบ 3‑D ไว้เมื่อแปลงไฟล์ CAD เป็นรูปแบบ raster หรือ vector  
- **ไลบรารีที่ทำการแปลงคืออะไร?** Aspose.CAD for .NET  
- **ฉันสามารถแปลง DWG เป็น PDF ด้วย C# ได้หรือไม่?** ใช่ – ตัวอย่างด้านล่างแสดงการทำงานของ C# อย่างครบถ้วน  
- **ฉันต้องมีลิขสิทธิ์หรือไม่?** ต้องมีลิขสิทธิ์ Aspose.CAD ที่ถูกต้องสำหรับการใช้งานในสภาพแวดล้อมการผลิต; ลิขสิทธิ์ชั่วคราวสามารถใช้ได้สำหรับการประเมินผล  
- **ขนาดผลลัพธ์ที่คาดว่าจะเป็นเท่าใด?** ในตัวอย่างเราทำ raster ที่ 1600 × 1600 px, แต่คุณสามารถปรับขนาดตามต้องการ  

## การแปลง DWG เป็น PDF พร้อมการสนับสนุน mesh คืออะไร?

การแปลงไฟล์ DWG เป็น PDF พร้อมการรักษาข้อมูล mesh ทำให้พื้นผิว 3‑D ที่ซับซ้อนปรากฏอย่างถูกต้องในเอกสารสุดท้าย ซึ่งมีประโยชน์อย่างยิ่งสำหรับการตรวจสอบด้านวิศวกรรม การนำเสนอให้ลูกค้า และการเก็บข้อมูล BIM

## ทำไมต้องใช้ mesh support ของ Aspose.CAD?

- **การเรนเดอร์ที่แม่นยำ** ของวัตถุ 3‑D โดยไม่สูญเสียรายละเอียด  
- **ไม่มีการพึ่งพาภายนอก** – ไลบรารีจัดการทุกอย่างภายใน .NET  
- **ประสิทธิภาพที่เร็ว** สำหรับภาพวาดขนาดใหญ่ด้วยตัวเลือกการ rasterize ที่ปรับแต่งไว้  
- **ความเข้ากันได้แบบข้ามแพลตฟอร์ม** กับ .NET Framework, .NET Core, และ .NET 5/6

## นำเข้า Namespaces

ในโครงการ .NET ของคุณ, นำเข้า namespaces ที่จำเป็นเพื่อเข้าถึงฟังก์ชันของ Aspose.CAD. เพิ่มบรรทัดต่อไปนี้ในโค้ดของคุณ:

```csharp
using Aspose.CAD.FileFormats.Cad;
using Aspose.CAD.ImageOptions;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีเอกสารของคุณ

```csharp
string MyDir = "Your Document Directory";
```

## ขั้นตอนที่ 2: ระบุเส้นทางต้นทางและผลลัพธ์

```csharp
string sourceFilePath = MyDir + "meshes.dwg";
string outPath = MyDir + "meshes.pdf";
```

## ขั้นตอนที่ 3: โหลดภาพ CAD และกำหนดตัวเลือกการ Rasterization

```csharp
using (CadImage cadImage = (CadImage)Image.Load(sourceFilePath))
{
    CadRasterizationOptions rasterizationOptions = new CadRasterizationOptions();
    rasterizationOptions.PageWidth = 1600;
    rasterizationOptions.PageHeight = 1600;
    rasterizationOptions.Layouts = new string[] { "Model" };

    PdfOptions pdfOptions = new PdfOptions
    {
        VectorRasterizationOptions = rasterizationOptions
    };
```

## ขั้นตอนที่ 4: บันทึกภาพที่ประมวลผลแล้ว

```csharp
    cadImage.Save(outPath, pdfOptions);
}
```

ขอแสดงความยินดี! คุณได้ใช้ mesh support ใน Aspose.CAD for .NET เพื่อ **แปลง DWG เป็น PDF** และ **save CAD as PDF** อย่างสำเร็จแล้ว คุณสามารถสำรวจฟีเจอร์เพิ่มเติมและปรับแต่งโค้ดตามความต้องการของโครงการของคุณได้

## ปัญหาทั่วไปและวิธีแก้ไข

| ปัญหา | วิธีแก้ไข |
|-------|----------|
| **Meshes ปรากฏว่างเปล่า** | ตรวจสอบให้แน่ใจว่า `Layouts` มี `"Model"` และไฟล์ DWG ต้นทางจริง ๆ มีเอนทิตี้ mesh อยู่ |
| **Output PDF มีขนาดใหญ่เกินไป** | ลดค่า `PageWidth`/`PageHeight` หรือเปิดการบีบอัดโดยใช้ `PdfOptions.CompressionLevel` |
| **License ไม่ได้ถูกนำไปใช้** | เรียก `Aspose.CAD.License license = new Aspose.CAD.License(); license.SetLicense("Aspose.CAD.lic");` ก่อนโหลดภาพ |
