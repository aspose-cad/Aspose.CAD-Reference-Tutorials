## คู่มือทีละขั้นตอน

### ส่งออก DGN เป็นส่วนหนึ่งของ DWG
สถานการณ์นี้เป็นเรื่องทั่วไปเมื่อโครงการมีสินทรัพย์หลายรูปแบบและคุณต้องการคอนเทนเนอร์ DWG เดียวที่บรรจุข้อมูล DGN ดั้งเดิม

#### วิธีการ export dgn to dwg
โหลดไฟล์ DGN ต้นฉบับของคุณโดยใช้ `CadImage` ฝังลงในคอนเทนเนอร์ DWG ใหม่ และบันทึกผลลัพธ์ รูปแบบสองขั้นตอนนี้จัดการการแปลงในหน่วยความจำและเขียนไฟล์ DWG ที่สอดคล้องกับมาตรฐาน

`CadImage` คือคลาสหลักของ Aspose.CAD ที่แสดงภาพ CAD ที่โหลดเข้าสู่หน่วยความจำ  

1. โหลดไฟล์ DGN ด้วย `CadImage.load("source.dgn")`.  
2. สร้างอ็อบเจ็กต์ภาพ DWG ใหม่และเพิ่ม DGN ที่โหลดเป็นเอนทิตีฝัง.  
3. เรียก `save("output.dwg", SaveFormat.Dwg)` เพื่อเขียนไฟล์ DWG.  

> *ตัวอย่างโค้ดโดยละเอียดมีให้ในซับ‑ทูทอเรียลที่อุทิศไว้ด้านล่าง.*

### ส่งออก DGN ฝังใน PDF
เรียนรู้วิธี **convert dgn to pdf** พร้อมคงเนื้อหา DGN ที่ฝังอยู่ใน PDF

#### วิธีการ export embedded DGN to PDF
เปิดไฟล์ DGN, กำหนดค่า `PdfOptions` สำหรับการส่งออกเป็น PDF, และบันทึก API จะฝังข้อมูล DGN ดั้งเดิมลงใน PDF โดยอัตโนมัติเพื่อการสกัดภายหลัง  

`PdfOptions` กำหนดการตั้งค่าเฉพาะ PDF ทั้งหมด เช่น ขนาดหน้า, การบีบอัด, และเมตาดาต้า.  

1. เปิดไฟล์ DGN ด้วย `CadImage.load("source.dgn")`.  
2. สร้างอินสแตนซ์ `PdfOptions` และตั้งค่าตัวเลือกที่ต้องการ (เช่น `setEmbedDgn(true)`).  
3. บันทึกภาพเป็น PDF ด้วย `save("output.pdf", SaveFormat.Pdf, pdfOptions)`.  

> *ตัวอย่างโค้ดเต็มสามารถพบได้ในบทเรียน “Export Embedded DGN”*

### ส่งออก DGN เป็น PDF ที่เข้ากันได้กับ AutoCAD
สร้าง PDF ที่เลียนแบบการวาดของ AutoCAD ซึ่งเป็นประโยชน์สำหรับการตรวจสอบของผู้มีส่วนได้ส่วนเสียเมื่อไม่มีซอฟต์แวร์ CAD ติดตั้ง  

#### วิธีการ export DGN to AutoCAD‑compatible PDF
โหลด DGN, ตั้งโหมดการเรนเดอร์ PDF เป็น AutoCAD, และบันทึก PDF ที่ได้จะคงน้ำหนักเส้นและสีของชั้น, ตรงกับสไตล์การแสดงผลของ DWG  

`PdfOptions` ยังมีแฟล็ก `AutoCadMode` ที่บังคับการเรนเดอร์แบบสไตล์ DWG.  

1. โหลดไฟล์ DGN.  
2. เปิดการเรนเดอร์ AutoCAD ใน `PdfOptions`.  
3. บันทึกไฟล์เป็น PDF.  

### ส่งออก DGN เป็นรูปแบบภาพเรสเตอร์
สร้างภาพ JPEG, PNG หรือ BMP ความละเอียดสูงจากไฟล์ DGN สำหรับการแสดงตัวอย่างบนเว็บ, เอกสาร, หรือรูปย่อ  

#### วิธีการ export DGN to raster images
โหลด DGN, ระบุตัวเลือกการส่งออกเรสเตอร์ เช่น DPI และความลึกสี, จากนั้นบันทึกในรูปแบบภาพที่ต้องการ  

`RasterImageExportOptions` ให้คุณควบคุมความละเอียด, การตัดขอบ, และความลึกสี.  

1. โหลดภาพ DGN.  
2. กำหนดค่า `RasterImageExportOptions` (เช่น `setDpi(300)`).  
3. บันทึกเป็น JPEG, PNG หรือ BMP ด้วย `SaveFormat` ที่เหมาะสม.  

## กรณีการใช้งานทั่วไปและเคล็ดลับ
- **Batch conversion pipelines** – ทำการวนซ้ำในไดเรกทอรีของไฟล์ DGN, ใช้ตรรกะการส่งออกเดียวกันกับแต่ละไฟล์.  
- **Preserving metadata** – ใช้ `PdfOptions.setMetadata()` เพื่อฝังชื่อชั้นต้นฉบับและคุณสมบัติกำหนดเองลงใน PDF.  
- **Performance optimisation** – สำหรับภาพวาดขนาดใหญ่, เปิดโหมดสตรีมมิ่ง (`setUseMemoryCache(true)`) เพื่อลดการใช้หน่วยความจำ.  
- **Pro tip:** เมื่อแปลงเป็นภาพเรสเตอร์สำหรับการใช้บนเว็บ, DPI ที่ 150 จะสมดุลระหว่างคุณภาพและขนาดไฟล์.

**Q:** *How do I know if my DGN file is compatible with export dgn to dwg?*  
A: Aspose.CAD supports most DGN versions (MicroStation V8, V9, V10). Use the `CadImage` loader to verify successful loading before exporting.  
**Q:** *ฉันจะรู้ได้อย่างไรว่าไฟล์ DGN ของฉันเข้ากันได้กับ export dgn to dwg?*  
A: Aspose.CAD รองรับส่วนใหญ่ของเวอร์ชัน DGN (MicroStation V8, V9, V10). ใช้ตัวโหลด `CadImage` เพื่อตรวจสอบการโหลดสำเร็จก่อนทำการส่งออก.  

**Q:** *Can I batch convert multiple DGN files to DWG in one run?*  
A: Yes – iterate over a file collection, applying the same export logic to each file.  
**Q:** *ฉันสามารถแปลงหลายไฟล์ DGN เป็น DWG เป็นชุดในรอบเดียวได้หรือไม่?*  
A: ใช่ – ทำการวนซ้ำในคอลเลกชันไฟล์, ใช้ตรรกะการส่งออกเดียวกันกับแต่ละไฟล์.  

**Q:** *What quality settings affect the raster image export?*  
A: You can control DPI, color depth, and anti‑aliasing via `RasterImageExportOptions`.  
**Q:** *การตั้งค่าคุณภาพใดที่ส่งผลต่อการส่งออกภาพเรสเตอร์?*  
A: คุณสามารถควบคุม DPI, ความลึกสี, และการตัดขอบผ่าน `RasterImageExportOptions`.  

**Q:** *Is there a way to preserve custom properties when converting to PDF?*  
A: Use `PdfOptions` to embed metadata and retain layer information.  
**Q:** *มีวิธีใดในการคงคุณสมบัติกำหนดเองเมื่อแปลงเป็น PDF หรือไม่?*  
A: ใช้ `PdfOptions` เพื่อฝังเมตาดาต้าและคงข้อมูลชั้น.  

**Q:** *Do I need a separate license for each output format?*  
A: A single Aspose.CAD license covers all supported export formats (DWG, PDF, raster).  
**Q:** *ฉันต้องการใบอนุญาตแยกต่างหากสำหรับแต่ละรูปแบบผลลัพธ์หรือไม่?*  
A: ใบอนุญาต Aspose.CAD หนึ่งใบครอบคลุมรูปแบบการส่งออกที่รองรับทั้งหมด (DWG, PDF, เรสเตอร์).
