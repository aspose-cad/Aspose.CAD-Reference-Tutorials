## นำเข้าแพ็กเกจ

ในโครงการ Java ของคุณ ให้นำเข้าคลาส Aspose.CAD ที่จำเป็น:

```java
import java.io.FileNotFoundException;
import java.io.FileOutputStream;
import java.io.OutputStream;
import com.aspose.cad.Image;
import com.aspose.cad.ImageOptionsBase;
import com.aspose.cad.fileformats.dgn.DgnImage;
import com.aspose.cad.imageoptions.CadRasterizationOptions;
import com.aspose.cad.imageoptions.JpegOptions;
```

## วิธีการส่งออก dgn เป็น JPEG ด้วย Aspose.CAD ใน Java?

คลาส `CadImage` แสดงเอกสาร CAD ที่โหลดเข้าสู่หน่วยความจำ ให้เมธอดสำหรับการเรนเดอร์และบันทึก

โหลดไฟล์ DGN ของคุณด้วย `CadImage.load("source.dgn")` ตั้งค่า `JpegOptions` (รวมคุณภาพและความละเอียด) กำหนด `RasterizationOptions` เช่น ขนาดหน้าและสีพื้นหลัง แล้วเรียก `image.save("output.jpg", jpegOptions)`. กระบวนการสามขั้นตอนนี้จัดการการแปลงจากเวกเตอร์เป็นแรสเตอร์ รักษาน้ำหนักเส้น และเขียนไฟล์ JPEG คุณภาพสูงภายในไม่กี่วินาทีสำหรับภาพวาดทั่วไป

## ขั้นตอนที่ 1: โหลดไฟล์ DGN

คลาส `CadImage` แสดงเอกสาร CAD ที่โหลดเข้าสู่หน่วยความจำ.

```java
String dataDir = "Your Document Directory" + "ExportingDGN/";
DgnImage dgnImage = (DgnImage) Image.load(dataDir + "Nikon_D90_Camera.dgn");
```

## ขั้นตอนที่ 2: สร้างอ็อบเจ็กต์ JpegOptions

`JpegOptions` กำหนดการตั้งค่าสำหรับการส่งออก JPEG เช่น คุณภาพการบีบอัดและความละเอียด.

```java
ImageOptionsBase options = new JpegOptions();
```

## ขั้นตอนที่ 3: กำหนด Rasterization Options

`RasterizationOptions` กำหนดวิธีการแรสเตอร์กราฟิกเวกเตอร์ รวมถึงขนาดหน้า, DPI, สีพื้นหลัง, และการทำ anti‑aliasing.

```java
CadRasterizationOptions vectorOptions = new CadRasterizationOptions();
vectorOptions.setPageWidth(600);
vectorOptions.setPageHeight(400);
vectorOptions.setNoScaling(true);
vectorOptions.setAutomaticLayoutsScaling(false);
options.setVectorRasterizationOptions(vectorOptions);
```

## ขั้นตอนที่ 4: บันทึกภาพที่แปลงแล้ว

การเรียก `save` จะเขียนภาพแรสเตอร์ลงดิสก์โดยใช้ตัวเลือกที่คุณกำหนด.

```java
OutputStream outStream = new FileOutputStream(dataDir + "ExportDGNToRasterImage_Out.jpg");
dgnImage.save(outStream, options);
```

### กรณีการใช้งานทั่วไป

- **การสร้างตัวอย่างเว็บ** – แปลงภาพวิศวกรรมเป็นภาพย่อ JPEG ที่มีน้ำหนักเบาสำหรับแสดงผลอย่างรวดเร็วในเบราว์เซอร์.  
- **การจัดเก็บแบบชุด** – ทำการแปลงไฟล์ DGN จำนวนพันไฟล์เป็น JPEG อัตโนมัติสำหรับการเก็บระยะยาวโดยไม่ต้องใช้ MicroStation.  
- **การทำรายงาน** – ฝังภาพสแนปช็อต CAD ลงในรายงาน PDF หรือ HTML โดยตรงจากโค้ด Java.

### ปัญหาทั่วไปและวิธีแก้

| ปัญหา | วิธีแก้ |
|-------|----------|
| **ภาพปรากฏเป็นสีขาว** | ตรวจสอบให้แน่ใจว่า `RasterizationOptions.setPageWidth/Height` ตรงกับขอบเขตของภาพวาด และตั้งค่าพื้นหลังที่ไม่โปร่งใส. |
| **ผลลัพธ์คุณภาพต่ำ** | เพิ่มค่า `JpegOptions.setQuality(100)` และตั้งค่า DPI ที่สูงขึ้น (เช่น `RasterizationOptions.setResolution(300)`). |
| **ข้อผิดพลาดหน่วยความจำไม่พอ** | ใช้ `CadImage.load` พร้อม `loadOptions.setLoadMode(LoadMode.Memory)` เพื่อสตรีมไฟล์ขนาดใหญ่. |
