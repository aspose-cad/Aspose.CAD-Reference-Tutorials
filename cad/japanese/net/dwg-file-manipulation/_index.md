# DWG ファイル操作

## はじめに

C# と Aspose.CAD for .NET を使用した DWG ファイル操作に関する包括的なガイドへようこそ。経験豊富な開発者でも、これから始める方でも、これらのチュートリアルは Aspose.CAD の可能性を最大限に活用できるよう支援します。このシリーズでは、**DWFX ファイルの開き方**、**DWG ファイル C# の読み方**、そして **DWG にテキストを追加** 方法も簡単に学べます。自信を持って DWG ファイルの取り扱いの詳細に入りましょう。

## クイック回答
- **Aspose.CAD for .NET の主な目的は何ですか？** CAD および BIM ファイル（DWG、DWFX、DWF を含む）をプログラムで読み取り、編集、変換することです。  
- **C# で DWFX ファイルを直接開くことはできますか？** はい – Aspose.CAD は DWFX コンテンツをロードしてアクセスするためのシンプルな API を提供します。  
- **開発にライセンスは必要ですか？** 無料の評価ライセンスはテストに使用できますが、本番環境では商用ライセンスが必要です。  
- **サポートされている .NET バージョンはどれですか？** .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6/7。  
- **DWG ファイルにテキストを追加することはサポートされていますか？** もちろんです – CAD API を使用してテキストエンティティを挿入できます。

## よくある質問

**Q: DWFX ファイルを C# で開くにはどうすればよいですか？**  
A: `Aspose.CAD.Image.Load("file.dwfx")` を使用してファイルをロードし、必要に応じてページやエンティティにアクセスします。

**Q: DWG ファイルを事前に変換せずに C# で読み取れますか？**  
A: はい – Aspose.CAD は DWG ファイルを直接ロードでき、レイヤー、エンティティ、メタデータを照会できます。

**Q: DWG 図面にテキストを追加する最も簡単な方法は何ですか？**  
A: `CadText` エンティティを作成し、`Value` と位置を設定してから、図面のモデルスペースコレクションに追加します。

**Q: 大きな DWFX または DWG ファイルを扱う際にサイズ制限はありますか？**  
A: ライブラリは大きなファイルを効率的に処理しますが、十分なメモリを確保し、大きなラスタ画像はストリーミングすることを検討してください。

**Q: 各 CAD フォーマットごとに別々のライセンスが必要ですか？**  
A: いいえ – 1 つの Aspose.CAD ライセンスで、DWG、DWFX、DWF、DXF など、すべてのサポート対象フォーマットをカバーします。
