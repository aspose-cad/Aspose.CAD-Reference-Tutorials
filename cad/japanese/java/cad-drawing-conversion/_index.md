## Frequently Asked Questions

**Q: DWG ファイルをレイヤ情報を失わずに PDF にエクスポートするにはどうすればよいですか？**  
A: `PdfOptions` を使用し、`PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)` を設定してベクトルデータを保持し、`image.save("output.pdf", options);` を呼び出します。

**Q: 複数の DWG ファイルを一度にバッチ処理して PDF に変換できますか？**  
A: はい。ファイルコレクションをループし、各ファイルを `Image.load` で読み込み、同じ PDF オプションで保存します。

**Q: DWT テンプレートを直接 DXF に変換する方法はありますか？**  
A: もちろんです。`Image.load` で DWT を読み込み、`image.save("output.dxf");` で DXF として保存します。

**Q: Aspose.CAD がサポートしている Java バージョンは何ですか？**  
A: このライブラリは Java 8 以降、Java 11、17、その他の LTS リリースで動作します。

**Q: PDF エクスポートに別途ライセンスが必要ですか？**  
A: いいえ、Aspose.CAD のライセンスは PDF、ラスタ画像、DXF などすべての出力フォーマットをカバーしています。

**Q: 大きな DWG ファイルを変換する際のレンダリング速度を向上させるにはどうすればよいですか？**  
A: `PdfOptions` インスタンスを再利用し、各レイアウトを別スレッドで処理してマルチスレッド化を有効にし、オーバーヘッドを削減します。

**Q: エクスポートした PDF のページサイズをカスタムに設定できますか？**  
A: はい、保存前に `PdfOptions.setPageSize(PageSize.A4)`（または他のサイズ）で `PdfOptions` を設定します。
