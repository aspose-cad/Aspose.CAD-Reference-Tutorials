## よくある質問

**Q:** *大きな DWG ファイルを PDF/A に変換してもメモリ不足になりませんか？*  
A: はい。`CadImage` クラスをストリーミングオプションと共に使用し、画像を速やかに破棄してメモリ使用量を低く保ちます。

**Q:** *CAD ファイルから特定のレイアウトだけを SVG にエクスポートすることは可能ですか？*  
A: もちろんです。名前で目的のレイアウトをロードし、SVG オプションで `Save` メソッドを呼び出します。

**Q:** *各出力フォーマットごとに別々のライセンスが必要ですか？*  
A: いいえ。単一の Aspose.CAD for Java ライセンスで、サポートされているすべてのエクスポートフォーマットをカバーします。

**Q:** *1 回の実行で複数の CAD ファイルをバッチ処理できますか？*  
A: はい。ファイルコレクションを反復処理し、各ファイルを `CadImage.load` で読み込んでターゲットフォーマットに保存します。

**Q:** *CAD ファイルにラスタ画像が含まれている場合、PDF にエクスポートするときに保持されますか？*  
A: PDF にエクスポートすると、ラスタ画像は自動的に埋め込まれ、視覚的な忠実度が保たれます。
