## Často kladené otázky

**Q: Jak exportovat soubor DWG do PDF bez ztráty informací o vrstvách?**  
A: Použijte `PdfOptions` a nastavte `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)`, aby se zachovala vektorová data, pak zavolejte `image.save("output.pdf", options);`.

**Q: Mohu dávkově zpracovat více souborů DWG do PDF v jednom běhu?**  
A: Ano — procházejte kolekci souborů, načtěte každý pomocí `Image.load` a uložte pomocí stejných PDF možností.

**Q: Existuje způsob, jak převést šablonu DWT přímo na DXF?**  
A: Rozhodně. Načtěte DWT pomocí `Image.load` a uložte jej jako DXF pomocí `image.save("output.dxf");`.

**Q: Jaké verze Javy jsou podporovány Aspose.CAD?**  
A: Knihovna funguje s Javou 8 a novějšími, včetně Java 11, 17 a dalších LTS verzí.

**Q: Potřebuji samostatnou licenci pro export do PDF?**  
A: Ne, licence Aspose.CAD pokrývá všechny výstupní formáty, včetně PDF, rastrových obrázků a DXF.
