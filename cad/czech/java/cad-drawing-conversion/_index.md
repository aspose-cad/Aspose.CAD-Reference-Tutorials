## Často kladené otázky

**Q: Jak exportovat DWG soubor do PDF bez ztráty informací o vrstvách?**  
A: Použijte `PdfOptions` a nastavte `PdfOptions.setVectorRasterizationMode(VectorRasterizationMode.Vector)`, aby se zachovala vektorová data, poté zavolejte `image.save("output.pdf", options);`.

**Q: Můžu hromadně zpracovat více DWG souborů do PDF v jednom běhu?**  
A: Ano – projděte kolekci souborů, načtěte každý pomocí `Image.load` a uložte pomocí stejných PDF možností.

**Q: Existuje způsob, jak přímo převést DWT šablonu na DXF?**  
A: Rozhodně. Načtěte DWT pomocí `Image.load` a uložte jej jako DXF pomocí `image.save("output.dxf");`.

**Q: Jaké verze Javy jsou podporovány Aspose.CAD?**  
A: Knihovna funguje s Java 8 a novějšími, včetně Java 11, 17 a dalších LTS verzí.

**Q: Potřebuji samostatnou licenci pro export do PDF?**  
A: Ne, licence Aspose.CAD pokrývá všechny výstupní formáty, včetně PDF, rastrových obrázků a DXF.

**Q: Jak mohu zlepšit rychlost renderování při převodu velkých DWG souborů?**  
A: Povolením multi‑threadingu zpracujte každý layout v samostatném vlákně a znovu použijte jedinou instanci `PdfOptions`, čímž snížíte režii.

**Q: Můžu nastavit vlastní velikost stránky pro exportovaný PDF?**  
A: Ano, nakonfigurujte `PdfOptions` pomocí `PdfOptions.setPageSize(PageSize.A4)` (nebo jiné velikosti) před uložením.
