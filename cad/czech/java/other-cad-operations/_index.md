## Často kladené otázky

**Q: Mohu přidat vodoznak do DWG souboru, který je již chráněn heslem?**  
A: Ano. Načtěte DWG s příslušným heslem, aplikujte vodoznak pomocí API a poté soubor uložte.

**Q: Ovlivňuje přidání vodoznaku velikost souboru výrazně?**  
A: Nárůst je minimální, protože vodoznak je uložen jako lehký překryv.

**Q: Je možné hromadně zpracovat více CAD souborů a přidat stejný vodoznak?**  
A: Rozhodně. Projděte svou kolekci souborů a pro každý výkres zavolejte stejnou rutinu vodoznaku.

**Q: Mohu řídit průhlednost a rotaci vodoznaku?**  
A: API poskytuje vlastnosti pro nastavení průhlednosti, úhlu rotace a umístění, což vám dává plnou vizuální kontrolu.

**Q: Které verze Javy jsou pro tyto tutoriály podporovány?**  
A: Java 8, 11 a novější LTS verze jsou plně podporovány.
