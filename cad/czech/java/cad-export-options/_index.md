# Možnosti exportu CAD

## Úvod

Pokud jste vývojář Java a hledáte **export AutoCAD do PDF** rychle a spolehlivě, jste na správném místě. V tomto průvodci projdeme nejčastější scénáře exportu v Aspose.CAD pro Java – včetně obrázků AutoCAD, rozvržení CAD, BMP, PDF, IFC a STL souborů. Zjistíte, proč jsou tyto funkce důležité, jak zapadají do reálných projektů, a získáte krok‑za‑krokem instrukce, které můžete zkopírovat do svého kódu.

## Rychlé odpovědi
- **Jaký je nejjednodušší způsob, jak exportovat AutoCAD do PDF?** Použijte `Image.save("output.pdf", new PdfOptions())` z Aspose.CAD.  
- **Do jakých formátů mohu převést soubory IFC?** PNG je plně podporováno; stačí načíst soubor IFC a uložit jako PNG.  
- **Mohu exportovat soubory STL přímo do PNG?** Ano – načtěte model STL a zavolejte `save("image.png")`.  
- **Potřebuji licenci pro produkční použití?** Pro ne‑zkušební nasazení je vyžadována komerční licence.  
- **Jaká verze Javy je požadována?** Doporučuje se Java 8 nebo vyšší.

## Co je **export autocad to pdf**?

Exportování výkresů AutoCAD do PDF znamená převod vektorových DWG/DXF souborů do široce akceptovaného, stránkového formátu. PDF zachovává vrstvy, tloušťky čar a barvy, což je ideální pro sdílení návrhů s klienty, recenzenty nebo downstream systémy, které nativní CAD soubory nečtou.

## Proč používat Aspose.CAD pro Java?

- **Zero‑install, pure Java** – Žádné nativní závislosti ani COM interop.  
- **Vysoká věrnost** – Geometrie, text a rastrová data jsou reprodukována přesně.  
- **Dávkové zpracování** – Exportujte desítky souborů v jedné smyčce s minimální paměťovou stopou.  
- **Široká podpora formátů** – Od klasických DWG/DXF po moderní IFC a STL, vše s jednotným API.

## Předpoklady
- Java 8 nebo novější nainstalovaná.  
- Knihovna Aspose.CAD pro Java přidaná do projektu (Maven/Gradle nebo JAR).  
- Platná licence Aspose pro produkční použití (k dispozici bezplatná zkušební verze).

### Tipy a triky
- **Pro tip:** Nastavte `PdfOptions.setVectorRasterizationOptions()` pro kontrolu DPI u rastrových exportů.  
- **Vyhněte se špičkám paměti:** Používejte `Image.dispose()` po každém uložení při zpracování mnoha souborů.  
- **Řešení problémů:** Pokud zmizí vrstvy, ujistěte se, že zdrojový DWG obsahuje viditelné vrstvy a že je povoleno `PdfOptions.setCompress(true)`.

## Často kladené otázky

**Q: Jak převést IFC na PNG pomocí Aspose.CAD?**  
A: Načtěte soubor IFC pomocí `Image.load("model.ifc")` a zavolejte `save("model.png", new PngOptions())`.

**Q: Mohu exportovat CAD rozvržení přímo do PDF bez předchozí konverze na obrázek?**  
A: Ano – Aspose.CAD zachází s rozvrženími jako s obrázky interně, takže `save("layout.pdf", new PdfOptions())` to provede automaticky.

**Q: Jaký je rozdíl mezi exportem AutoCAD do PDF a exportem CAD rozvržení do PDF?**  
A: Export AutoCAD výkresu převádí celý výkres, zatímco export rozvržení cílí na konkrétní pohled v papírovém prostoru a zachovává jeho nastavení stránky.

**Q: Existuje limit na počet stránek při exportu více‑listého DWG do PDF?**  
A: Ne, neexistuje žádný inherentní limit; každé rozvržení se stane samostatnou stránkou ve výsledném PDF.

**Q: Potřebuji samostatnou licenci pro každý exportní formát?**  
A: Jedna licence Aspose.CAD pokrývá všechny podporované formáty (PDF, PNG, BMP, atd.).
